Copernicus Open Access Hub

Download data at: https://scihub.copernicus.eu/dhus/#/home
  • Log in (or create an account) on the upper right
  • Pick region on map, using bottom button on the top right ("switch to area mode").  If you don't pick an area you will have millions of results.
  • Open search with the hamburger menu on the top left
  • Pick sensor
    • Sentinel-1
      • Modes at https://sentinels.copernicus.eu/web/sentinel/user-guides/sentinel-1-sar/resolutions/level-1-single-look-complex
      • Pick GRD product type, IW sensor mode
      • There may be very long file names, so put the files in a very short path location before unzipping
    • Sentinel-2
      • For cloud cover, it is case sensitive and you need the brackets:  [0 TO 5]  It will be black when acceptable
      • Pick the level with "Product Type". You want S2MSI1C (the 1C is the key part)
        • The Level 2A data has folders with 10 m, 20 m, and 60 m resolution.  The lower resolution contains all the bands with a downsampled version of the higher resolution band. 
      • You can leave the other two fields blank (you probably don't care which satellite you get, or the orbit which is sequentially numbered over time)
    • Sentinel-3, mode, dates with the filter
      • Not currently used by MICRODEM
  • Pick the search (black magnifying glass)
  • Results will appear.  You have a thumbnail, which will show the coverage; for Sentinel-2 it shows if the edge of the orbit creates missing data along one edge.
  • Limit of 4 concurrent downloads
  • Older tiles may be offline (indicated in the upper right of the information about the tile in the search results), and you must request them to be added to your cart.  You will not get an email, but they showed in under a day (very fast at quiet times, like weekends).  Your cart is at the top left of the screeen.  The cart files might not show up immediately, wait a short time to see if they are there.  After downloading, delete them from your cart.
  • Files will be zipped; you can  unzip the file in MICRODEM.
    • Use the "Unzip to this directory" option, which makes the SAFE folder the top.
    • Put the files in as short a path as possible, since the full paths approach a limit in Windows for file names.
