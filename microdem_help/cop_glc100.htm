Copernicus Global Land Cover (GLC100)

MICRODEM recognizes these with "LC100_global" in the file name.  Do not rename unless you keep this in the file name

https://land.copernicus.eu/en/map-viewer?dataset=a8d945f0edd143a0a5240c28bafa23da

  • Can get entire world, or use map to pick rectangle, country, or upload file
  • Whole world file must go into a X:\monster_dems' with the likes of EDTM or GEDTM
  • Version 3, 2019
  • 3.6" spacing (~100 m)
  • MICRODEM will use GDAL to open
    • Use a map open to the area you want because otherwise the file will probably be too large, will be very slow, and might fail.
    • Full global coverage (1.7 GB in a very compressed state; this will not realistically open without subsetting), with additional layers at https://zenodo.org/record/3939050#.X67ebWhKj8A.
    • After import you can save an a MICRODEM DEM, and the program will remember what it is and open faster

Land Cover overview

  • Old version
    • 20x20 degree tiles named for the NW corner, but you can pick on the map to get the correct one.
    • Leave all settings
    • annual versions 2015-2019; click on the year to start download
    • You can skip sending your opinion
