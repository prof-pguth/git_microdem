Raster GIS Operations

Menu available with a raster grid as the active map window.  Those options requiring multiple open grids will not appear until you have opened the grids.  A DEM is just a specialized grid.

  • DEM/Grid Histogram--create simple histogram of  current grid.  Settings on options form, (Analysis tab for using sea level for statistics and clipping the tails) will be used for histogram. Other histogram options
    • Enitre grid
    • Current map area
    • Select multiple DEMs/grids

    • Aspect rose diagram
    • Geomorphometry by categories
    • SSO diagram
    • Elevation/grid value moment
    • Elevation and slope moments
    • Slope moments
    • Elevation slope plot
  • Geomorphometry analysis  (requires geomorphometry menu option set)
  • Create derivative grid--create new grids from current grid.
  • Multiple grid arithmec (map algebra)
  • Filter DEM/grid: creates a new grid, leaving the original unchanged.
  • Downsampling (creating a lower resolution grid)
  • SSIM
  • Elevation percentiles
  • Fixed palette stats
  • Heat Map

  • Create bitmap mask: pick area shape file, and then the location and file name for it to be saved.  You should probably use the GIF format for most efficient storage on disk.  This will create a bitmap with the current screen resolution and coverage area, with the shape file as a solid color.
  • Export values all open grids
  • Random sampling points: pick the number of points.  If there is a DEM on the map, the points will only occur at locations with valid points on the grid.  The points will be in the current map area.
