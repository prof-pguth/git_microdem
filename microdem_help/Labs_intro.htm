Setting up MICRODEM

 

Set Program version on Program tab of options form.  Pick the appropriate option.  This does two things:

  1. Simplifies menus, removing choices you will not need.  The "regular" option is the most comprehensive.
  2. Opens up labs menu choices, which automates downloading and opening data sets.

General Directions for the program

  • Basics of running MICRODEM.
  • Legends/marginalia on maps. All large scale maps should have a scalebar.  Small scale maps, where the scale changes significantly from top to bottom, will not allow a single scalebar.
  • Key buttons on map toolbar (which is different from the main program toolbar)
    • Copy to clipboard You can copy any window's contents to the clipboard and then to Word or PowerPoint.  If this is not on the toolbar, use the right click popup menu, or the File menu.  For a map you have the choice of a quick export, or a cartographically pretty map with the coordinates labeled.  This gets all of graphic, and none of the border, both of which can be challanges if you use the Windows "Snip" tool. Copy to clipboard also works with all graphs and other figures created by the program.
    •   Map zooming.  For large maps, use subset and zoom.  These data sets are larger than will fit on screen at full resolution.
    • Zoom out to show entire data set.
  • Key options with database.  The database table will be displayed in its own form, which allows you a number of options with the data.
  • Quick map enhancement options
  • Map query on database table display (especially Rectangular region and Current Map Area to find just the database records that are on a smaller region of the map)
  • For a DEM, the default display is a combination reflectance map with color elevation tinting.  If you want to display other data on the map, you might go with a grayscale reflectance map.  Right click on the map window, and change the display parameter, or adjust the "reflectance options".
