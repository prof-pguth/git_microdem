Detailed directions

The DEMIX database contains a rich resource for looking at how the test DEMs, different criteria, and tile characteristics interact.

MICRODEM can provide multiple graphics to explore these.  It will automatically combine graphics into large combined images.

Mixed filters have the following format.  They can combine or mix multiple fields in the database, and can be created by the user.  The FILTER field must be a valid dBase filter; the LABEL can be more human readable and will be used to label graphics.

FILTER LABEL
avg_slope > 50 Mean slope > 50%
avg_slope > 30 and avg_slope < 50 Mean slope 30-50%
avg_slope > 5 and avg_slope < 30 Mean slope 5-30%
avg_slope < 5 Mean slope < 5%

Symbol options

Landscape filters

  • Mixed filters point to dbf files in the DEMIX settings directory with the name in drop down list.  Most options use the first, but some use both.
  •  Standard landscape filters make a series for the particular feature.  They are created in code

Pick button

  • Criteria FUV, standard landscape: each DEM will be in a separate row, each standard landscape filter in a row of graphs with a legend on the bottom.
  • Criteria FUV, 1 mixed filter: each graph will have one filter form the top mixed filter, with all selected DEMs on that graph
  • Criteria FUV, 2 mixed filter grid: each graph will have all the DEMs with color symbols the FUV on the vertical axis, and the criteria on the horizontal axis.  The top mixed filter will be in the rows of the diagram, and the second in the columns.
  • Criteria FUV, just DB filter: allows creation of a single graph, mean/median FUV on the vertical axis, and the criteria on the horizontal axis, with whatever the active filter on the data base is.  If you want more details work with that filter, save the filtered database and run the analysis on it.
