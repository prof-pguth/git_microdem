Global scores for the 1" DEM at https://visioterra.fr/telechargement/P317_DEMIX/global_scores/  Each folder has the grids, and a Word document with a description.

  • A01 is Product fractional cover, percentage of the tile with an elevation
  • A02 is Completeness, percent of tile with coverage by valid data
  • A03 is Primary data fraction which provides an indication of how much of the tile was filled with other DEMs: 
  • A04 is the valid land fraction, how much of the land is covered by the primary instrument
  • Ao5 if Completeness, percentage of the cell that has a valid elevation from the primary sensor, is not masked as water, and is not extrapolated, masked, or filled in

Key database fields in the -HYDRO and -SSIM versions.  These correspond to REF_TYPE="DTM" and LAND_TYPE="ALL" for the classic version

  • CRITERION: what was evaluated.  You should filter to a single criterion, unless you are sure you want more (say to get the average score using multiple criteria)
  • COP, ALOS, etc.: the evaluations, low score wins.
  • COP_SCR, ALOS_SCR, etc.: the scores (opinions)
  • AVG_SLOPE, AVG_RUFF, RELIEF: tile average characteristicss, helpful for sorting to visualize trends.   If not included in some versions of the database, they can be added using a JOIN.
