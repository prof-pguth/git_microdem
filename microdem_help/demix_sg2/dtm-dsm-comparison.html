DTM-DSM Comparison

Create database

  1. Insure the MICRODEM code can identify the DTM/DSM pair.  Currently only Spain and Canada work.
  2. Put DEMs in area directories under the main data directory.
  3. Make one of the databases (Mixed FUV, Tile statistics), merging the CSV files, and put it in the main directory
  4. Open DB
  5. Add "DEM_PAIR" and "REF_DEM" fields with DEMIX button, Modify this table, Identify DSM/DTM pair
  6. Add tile statistics to the database
  7. DEMIX button, "Create new tables", "Create DB for DTM/DSM comparison".  This will make a CSV file for each test areas, which will be in subdirectory "aa_dsm_dtm" under the project directory
    • Currently elevation, slope, and RRI
  8. Merge the CSV files, and place in the folder with the area

 

Analyze database

  1. Stats, DSM/DTM comparison, DSM < DTM. 
    • Adds field DSM_LT_DTM, currently hard wired for average tile slope where (DSM_SLOPE < DTM_SLOPE - 1)
  2. Filter DB if desired
  3. Stat menu, Compare DSM/DTM
