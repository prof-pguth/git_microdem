Create/Index Menu

Menu on data manipulation form with options to:

  • Map library
  • TIGER index update: cycle through all the data in the TIGER directory and its subdirectories and create the data base.
  • Vertical datums: select all the TIFF files with a vertical datum transformation grid (for instance from PROJ downloaded via OSGEO4W) and create a database with the bounding box for each of the grids.
