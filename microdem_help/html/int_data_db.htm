Map Library

Start
  • Map library-- main program toolbar  or
  • Right click on the map, Load, Map library. 

Set up and manage map library data base

Map with library coverag from the database..

Use Key buttons on the toolbar on  map toolbar to adjust map size and coverage area.  

  Use "Load button" to outline the region you want.  This will merge DEMs on the fly.

 "LIB" button to change the series to show.
  Right click on the map, and pick "Map library setup" to select the series to show.  

Double click to move series from the enabled (shown on the map and opened) to disabled.

The combo box at the top allows you to restrict the view to a single kind of data.

You will see all enabled series on the map, and they will all be loaded if you select a region.
