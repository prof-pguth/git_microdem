ASCII XYZ Import Options

Values in the XYZ file can be separated by spaces, commas, or tabs.

This form gives you a view of the first 10 records in the XYZ file, and lets you set some options for reading them:

  • Import to a DB table, a grid, or both.
  • The XY format, for how the data will be interpreted (UTM, lat/long, or generic rectangular, or dd-mm-ss.sssN)
  • A multiple for the Z values.
  • No data value: lines with this will be ignored.
  • Elevations in feet
  • Multiply longitude by -1, since west longitude must be negative.
  • Multiply the z values by -1, which might be necessary for depths to behave properly in hydrographic data.
  • Option to plot the raw XYZ data so you can evaluate its coverage.
  • Hemispshere and UTM zone for UTM data, so LAT/LONG fields can be added.
