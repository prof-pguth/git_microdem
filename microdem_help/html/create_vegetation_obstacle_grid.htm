Create Vegetation Obstacle Grid

 

Removed October 2021

 

 

Option on Raster GIS, Create derivative grid menu from map window menu.

Options:

  • From NLCD 2001.
  • From a shape file
  • From LandFire data
  • From NBCD--you need the "NBCD_MZ12_BAW_height.tif" file.

From NLCD 2001.

  • Get NLCD 2001 from the USGS Seamless server in Geotiff format.  This is a conic projection, and must be converted to a compatible projection to use the your DEM or other data set.
  • You must have the HEIGHT field correctly filled out in the Land cover database. 
  • The data has 30 m spacing, and you should use a DEM with a similar scale.
  • Create vegetation obstacle grid.  This will be a slow process.
