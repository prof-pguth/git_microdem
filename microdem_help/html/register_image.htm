Register Image Caveats

Affine image registration directions.

Caveats on image registration, which might work if the simple World file options will not:

  • The MICRODEM image registration provides a "quick and dirty" result.  It should only be used carefully, and as a last resort.
  • If your image has geometric distortions, or relief displacement, the simple affine transformation used cannot work well.
  • This should only be used for a large scale imagery of a small area.
  • If your image has an inappropriate projection, the simple affine transformation used cannot work well.  The smaller the scale of the image (the larger the area covered), the greater the chance that this will not work.
  • If you scan an image, you almost certainly have to use this method (or rotate the image), because other alternatives (like world files) assume that the map's grid is oriented north-south and it is impossible to get that in a scan.
  • The following will limit the effectiveness of this technique:
    • Camera lens distortion (poor optics).
    • Off-nadir viewing geometry.
    • Terrain displacement in steep areas.
    • Scanner imperfections and distortions.
  • If at all possible, let a professional handle this.  In addition to errors that you might make, your data may not be possible to register using this simple technique.
