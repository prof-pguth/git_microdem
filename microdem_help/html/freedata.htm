Free Data Sources

Caveats on these lists.

Data portals US state sites  Country sites Key data Key Global Regional DEMs

MICRODEM supports many types of data and  data formats.   Insure the files you download are in an acceptable format. Avoid ESRI e00 format and SPCS data if possible, and never get ERDAS Imagine data.

Some data sets consist of multiple files, and you must pick the correct file to open the data set.
