Raster GIS External tools

Submenu on the Raster GIS menu. 

Some of these might not work with arc second DEMs.

  • Whitebox Tools
    • Whitebox slope map (Horn)
    • Whitebox Aspect map
    • Whitebox curvature
      • Plan
      • Profile
      • Tangential
      • Minimal
      • Maximal
      • Mean
      • Gaussian
      • Casiorati
    • Whitebox fill holes
    • Whitebox multiscale roughness
    • Whitebox Pennock Landform Classification (Pennock,D.J., Zebarth,B.J., and De Jong,E., 1987. Landform Classification and Soil Distribution in Hummocky Terrain, Saskatchewan, Canada. Geoderma, 40: 297-315.) on Calculate menu, Ridges/valleys
    • Whitebox Geomorphons
    • Whitebox average normal vector angular deviation
    • Whitebox circular variance of aspect
  • GRASS. 
    • Grass Slope (Horn)
    • Grass aspect (Horn)
    • GRASS curvature
      • Grass Profile curvature
      • Grass Tangential curvature
    • Add-ons you have to add to GRASS.  There is an option to get these on the File menu with a DEM map open, only with the GIS full version.  Alternatively load them directly in the GRASS GUI. 
      • TPI
      • TRI
      • GRASS  r.roughness.vector; relatively slow
  • GDAL
    • GDAL slope and aspect maps (2 methods available each; problems with geographic DEMs)
    • GDAL hillshade (problems with geographic DEMs)
    • GDAL TRI (Wilson, bathymetric)
    • GDAL TRI (Riley, terrestrial)
    • GDAL TPI
    • GDAL roughness  (not the same as the MICRODEM roughness; Roughness is the largest inter-cell difference of a central pixel and its surrounding cell, as defined in Wilson et al (2007, Marine Geodesy 30:3-35).)
    • Contours shapefile:  this will be in the temp directory; save elsewhere before closing the program if you want. 
  • SAGA
    • SAGA aspect map
    • SAGA TRI map
    • SAGA TPI map
    • SAGA VRM map (vector ruggedness)
    • SAGA curvature
      • SAGA profile curvature
      • SAG plan curvature
  • LSP Calculator
    • All LSPs
    • All partial derivatives
    • kns
    • kncc
    • kncs
    • knss
  • Line detection (Hough transform)
  • Raster to vector
  • RVT grid creation

There are also image analysis external tools
