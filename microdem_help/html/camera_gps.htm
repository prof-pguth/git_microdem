Camera GPS Setup

  • Nikon D5300 (and some other Nikon models)
    • Download A-GPS file from https://downloadcenter.nikonimglib.com/en/download/fw/111.html
    • You may have to use a card reader to transfer the file to the camera memory card (I cannot do it on two Windows 10 machines, on both of which I can copy and delete JPEGs)
    • File is good for 2 weeks.
    • Without the current file, camera never seems to lock onto GPS (even with file, it is very hit or miss)
  • Olympus TG-7
    • Turn on GPS in settings, G (field sensor)
    • On PC, get OM Workspace at https://download.omsystem.com/pages/owdownload/en/
      • Camera, Update GPS
      • Insure camera is connected with cable, pick the MTP option and then OK
  • Setting up smart phone.
