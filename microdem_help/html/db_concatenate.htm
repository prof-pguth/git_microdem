Concatenate Fields

On Insert new fields submenu of Edit popup menu of database table form..

 
This form lets you combine two or three fields (integer or character)  into a new string field.
  • Name the new field in the left edit box, keeping in mind the rules for field names.
  • The two fields to concatenate are selected from the two drop down boxes.
  • Edit boxes allow you to put text in front of, between, and after the two field names.  The text can include blank spaces.

If you need to combine more than 3 fields, do the process in stages.
