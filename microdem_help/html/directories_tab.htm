Directories Tab

Tab on options form, brought up from Options program menu selection.

 
  • Main data path: only the drive letter you select is important. You must have write permission for this location. Should be at  root level of drive.
  • Global topography DEM:
  • GDAL/FW tools directory-- read a variety of formats.   This setting is read only; if you do not want the default installation, you will have to edit the MICRODEM INI file to change this.
  • Vector map projection: default vector map.
  • Viewshed Dir: location for storing viewsheds.
