Edit a Database

This is deliberately not easy to start, because any edits are permanent, and most users should not be editing databases. Sve a copy of the database if you are concerned about editing mistakes.

To a edit a dBase database (DBF file):

Enable edits by checking the "EDIT" box on the right of the row of buttons on the database table form, since all changes are permanent (save a copy of the database if you are concerned about editing mistakes).  There is a (dangerous) option to automatically open databases for opening, on the Database tab of the options form

 

 

  • On  options form, Database options tab, enable "Edit GIS database" so it is not never.
  • Open a map.
  •   Database: open a database.
  • You can edit the DBF file in MICRODEM.
    • Edit button on the database table form
    • Double clicking on a record in the database table form and pick the edit record option.

You can also edit a DBF file in other software:

  • You can edit a DBF file and save it as a CSV file (perhaps in Excel)  CSV file can be imported into MICRODEM or opened as a  Data base with on the fly conversion to DBF

You can perform any edits on a point shape file.

For an area or line shape file, you cannot delete records or reorder them, because that will destroy the links with the shp and shx files.
