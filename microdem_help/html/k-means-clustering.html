K-means clustering

Option on Stats menu with a databae

  • Create cluster.  Set the options
  • Cluster frequency: select a field like AREA, and see how many tiles are in each cluster from that area.  This shows which areas have tiles in a cluster, and the number of tiles.  This shows what is on the cluster map, but unless you zoom the map in, that can only show that tiles in that area
  • Cluster maps: separate map for each cluster.  Set the map area and symbol size before picking the option.
  • Cluster statistics: create a table with 5 statistics for each of characteristics defining the clusters.  You can filter it to get just a single statistics.
