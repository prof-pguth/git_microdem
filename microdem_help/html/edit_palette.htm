Edit Palettes

This applies to Fixed color palette from table, which uses pre-defined categories, usually good for only one thing.

 

  • Open the file c:\microdem\hard_limit_color_palettes.dbf.  In the future this might have a version number.  Use Excel or your favorite editor.
  • Fill in the table.  The smallest category must come first
    • PALETTE: all the colors must have the same palette name
    • NAME: used for legends
    • MAX: starting with the first entry in the palette, the program checks if the value to be colored is less than MAX.  If so, that color is applied.
    • RED,GREEN,BLUE: set the colors to use, range 0-255.  You can use the color picker in Paint ("Edit colors" on the right of the toolbar), or any of a number on the web such as https://www.w3schools.com/colors/colors_picker.asp .
    • USE: set to Y (or the program will not show you this palette)
    • SOURCE: take credit for your work
    • REC_ID: the program can set this.
  • Save As in the CSV format, in c:\microdem.
  • Convert to a DBF file, overwriting the existing version, with either
    • Open database without map
    • Open database
  • To use, open a grid, pick elevation, Fixed color palette from table, and pick the palette name you selected.
