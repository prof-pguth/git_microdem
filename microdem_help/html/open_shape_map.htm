Open shapefile map

  On the main program toolbar, or the File, Open submenu choice.

  • You can pick either the SHP or the DBF file for a shapefile. 
  • If you pick a CSV or TXT file, the program will try to convert it to a dBase table and them display it.

 Problems with CSV import

This creates a Quick vector map covering just the area required to display the shapefile.  It will use the default projection, which is set on the on the Directories tab of the Options form.

  • Database on the map toolbar can also overlay the shapefile on any other map.
