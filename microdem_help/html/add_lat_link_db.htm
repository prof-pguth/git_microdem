Add lat/long from Linked database

Option on the Edit option (which must be enabled) of the database table form.

You must have Linked data base  from the Stats option of the database table form, and the linked data base must have the fields LAT and LONG.  When you link the databases, you must select the fields in the two data bases that will be used to find the lat/long locations.

If there are no LAT/LONG fields, they will be added and populated.

If there are LAT/LONG fields, LAT2 and LONG2 fields will be added and populated.

If you need to add two sets of LAT/LONG coordinates, you will have to do the operation once, then pick a second link table with different fields for the linkage operation, and then repeat this step.
