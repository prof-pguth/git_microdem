Export Contour Lines

Option when contouring.

  • This will be a line shape file in the database directory (c:\mapdata\database).  Select when you draw the contours; the choice will not be remembered.
  • This is a direct translation of the grid contouring, with the contour lines done in one grid box at a time. The contour lines will not be continuous and were never intended to be; they will draw correctly.

Calculate, Contours shapefile (GDAL, Geotiff input grid) provides another option
