HTML Report

Option on Report menu of dabatabase table form.

HTML:  opens in the default browser; if you want to save it, use the File, Save as option in the browser.  The HTML files will be in the temporary map directory.  You should have a reasonably small number of records for this option, perhaps by Filtering the table.  You can also Hide fields you do not want.  Alternatively you can clean up the HTML after importing into your word processor or spreadsheet, either of will will import the HTML via cut and paste.
  • Single table--HTML will look like the table display.  In addition to opening in a browser, the table will be on the clipboard and you can paste it into Excel.
  • Table per record--each record will be in a two column table with the field name and its value. For a field with many columns, this can be more effective.
