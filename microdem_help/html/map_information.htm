Map Information

Main menu ("Info") when a map window is active.The results will vary with the kind of data on the map.

  • Map (also from a button on the map window's toolbar)
  • GDAL info for a number of data sets.  This is a very readable format, but does not include some information like vertical datum.
  • MICRODEM Geotiff info
  • Landsat metadata
  • Sentinel-2 metadata
  • Data header
  • Compare DEM headers: select the DEMs, which will each be a column.  The rows will be all of the information stored by MICRODEM about the DEM/grid; for other formats, you will see the conversions made before using the data.
  • Summary all open DEMs/grids: parameters in columns, DEMs in rows

 

Map information shows key information used by MICRODEM to display the data set includes:

  • The map projection, both for the map, and the data set
  • The screen pixel size.  This should be very accurate for large scale maps of a small area, but could be very approximate for small scale maps of the entire world unless you have specifically selected an equal area projection.
  • For raster data
    • Whether or not every data point can fit on screen.
    • The size of the data pixels
