15" https://www.gebco.net/data_and_products/gridded_bathymetry_data/

https://download.gebco.net/ to download a region
  • Current release is 2023.  Geotiff is 4 GB, uncompresses to 8 GB.  Will require subset with GDAL before use
  • Format: Geottff (best bet), netCDF, or ESRI ASCII
  • data set based on SRTM15+
  • Entire data set is huge, but you can get just the region you want.
  • Can get polar projection, arc second at 15"
  • The TID grids shows the source of the data
  • Can download user defined regions
  • Try Global Multi-Resolution Topography Data Portal first, which will probably be higher resolution
