Pick Map Center and Scale

  • Pick the coordinates of the map center point.
  • Pick the scale.  It will be the last scale you selected.
  • DPI: the screen resolution used to compute the scale for the map.
  • Map size: the width and height of the map, in pixels.

 

  • If you change any parameters, the size covered by the map will be updated. 

This form can be used to recenter a map using the recenter choice after a right click on the map window.
