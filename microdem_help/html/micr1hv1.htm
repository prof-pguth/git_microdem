Semivariogram

From Geostatistical analysis option.

This option computes semivariograms in four directions (NW-SE, N-S, E-W, and NE-SW).  It uses the average spacing in those directions, so is not appropriate for a geographic grid covering a large area.

  Log-log plot.  The fractal dimension is taken from the slope of the linear portion of the graph.
