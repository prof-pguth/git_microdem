Insert Point at Distance/Bearing

On popup menu when you click on a record in table display window of database table form.

Enter

  • Distance from the record clicked on
  • Azimuth from the record clicked on
  • Name
  • Notes

If the NAME and NOTES fields are not in the database, they will be added

 

Distance and azimuth will be used the compute LAT/LONG.  If you them added to the database, put in the NOTES.
