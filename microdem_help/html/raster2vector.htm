Raster to Vector

Raster GIS menu option

Suppress speckles: under this size will be removed.  Larger values will be less noisy.

Black level: under this level, pixels converted to black.  Value range is 0-1.

Invert image outlines white regions.  Otherwise black regions will be outlined.

Border details. If the border option selected is white or black, it will draw a box around the image, with either a white or black line.  This might close off regions on the edge which are truncated.

Convert:

  • Blows map to 100%
  • Takes the bitmap, and turns all non-white to black.

Notes

  • Uses: https://potrace.sourceforge.net/  .  Place it in x:\gis_sfotware\potrace\
  • Command created: potrace test.bmp -b geojson -L 270795 -B 3289770 -x 2.5
  • Convert back to shapefile: 0gr2ogr from GDAL
