Size of Data Sets

The size of a digital data set depends on several things:

  • The scale of the data set. All other things being equal, the larger the scale the larger the data set. (Large scale data covers a small area with detailed data.)
  • The area covered. All other things being equal, the larger the area covered the larger the data set.
  • The data included for a vector data set. A data set that include boundaries and roads will be bigger than one that includes only one of the two.
  • A raster data set includes something for every defined point (even if nothing is there, a code for nothing must be included), but vector data sets only have to show what is present.
  • A raster data set might have a number of layers, which would affect the size of the data set.

Larger data sets require more storage and more computer processing to display.
