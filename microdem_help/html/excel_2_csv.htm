Excel --> CSV --> MICRODEM

See the rules for what you must do in Excel to insure that your file will work.  The computer will be very picky.

Excel can still read DBF files, but will no longer save as DBF.

  • Save your file as an Excel spreadsheet.  If you did not follow the rules, you will have to come back to this file to make corrections.  Until you are sure your file is correct, it is better to have too many versions than to have only a single trashed version.
  • Save as a CSV file:
    • Select "File, Save As" in Excel.
    • Pick CSV as the "Save as type".
    • Save the file.

Import CSV into MICRODEM.

  •   Database on Map window toolbar
  •   Open database without map on Main Program Toolbar

Once the file is in MICRODEM, you should use the DBF file that MICRODEM creates, as edits will only be made to this file.
