Magnetic Anomaly Picks

Geology options

This has the Seton and others (2014) data set, with the ages from the Gee and Kent (2007) time scale in the GEEK2007 fields and the anomaly name in the CHRON field.  The pick is where a geologist has interpreted a magnetic profile, and selected the location of a particular anomaly on the profile and extracted its latitude and longitude. The pick can be for the old end of the anomaly ("o" in the ANOMALYEDN field), young end ("y"), or center.

Seton and others (2014) magnetic anomaly picks along the Juan de Fuca ridge. 

 

  • GEEK2007 field (numeric) above
  • CHRON field (string) below which works if there are a limited number of anomalies on the map (filter the database)

 

Prep the data.

Plot, Color code by DB field, with the GEEK2007 field (numeric) or the CHRON field (string).  You will want to Filter the database to just have the records on the map, or restrict the color range.  If you zoom in enough, you can Label the picks.
