Database Record Click Popup Menu

These options will be on a popup menu when you click on a record in table display window. Some options will be available only for certain types of database files.

  • Record display.    In addition to the text for each records, graphics will also be displayed.
    • Earthquake focal mechanism: requires Harvard CMT project database file.
    • Image: IMAGE field present. You can bring up an image (BMP, PNG, TIF, GIF, or JPEG) stored in the same directory as the database. If the file is a GIF, it will appear in a window that can play animated GIFs. The program will attempt to load other files (like MPEGs, WAVs, etc.) using the program set as the default in Windows.
    • Climograph--Koppen climate classification
  • Edit record options, requires edit enabled: You can also Color multiple records from the edit button on the toolbar.
    • Record edit:  You will get a pop up window with the record, and can edit it.
    • Edit point location (graphical): requires point database, and edit enabled. If available, you can then double click a location on the map. If the point verification (Verify Graphical Selections, on the coordinates table of the options form) is set to verify all or reasonable, you will have to verify the selected points, either graphically in a small blowup window or via the keyboard. After selecting the point, and verifying it if necessary, you can double click on records in the data base to change their locations. This option currently only works for data bases which include LAT and LONG fields, and not for Shape files.
