Grid Tab

Select with Options choice on the program menu, and then the grid tab on the Options form.

  Affects Multiple Grid statistics option on the Raster GIS.

Pick the grids you want created.

Checkbox determines in a single NoData value in one grid makes the result NoData, or if the program can compute statistics from the other data

Pick tolerance if creating a grid showing the highest and lowest grids.

"Filter grids to edge", set on Grids tab of the Options form, allows filtering to the edge of data set, even if the filter should require points outside the grid limits.
