Create Points File

On Report option of  database table form.  

Creates a new, points database with the requested number of records for each record in the shapefile.

Along lines, by number or separation


Lines shape file

Database must have "NAME" field, which will be used in the new database.
Two options

  • Along lines (by number): must have "NPTS" field for each record which will control the number of evenly spaced point records added along each line segment.
  • Along lines (by separation): select the separation in meters, which will be the same for all records.

 Options:

  • Include first point
  • Include last point (partial distance): the last point might be a different distance apart from the others
  • Include turn waypoints (partial distance): if you force the waypoints to be included, not all points will be the same distance apart

 

In polygons

Area shape file

Database

  • must have "NAME" field, which will be used in the new database.
  • must have "NPTS" field for each record

The points will be randomly distributed in the polygon.  Graphically drag them (Edit, Graphically move points) if you want a more even distribution, or to match the imagery.
