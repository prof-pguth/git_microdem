Arbitrary Shape File Coordinate Shift

To perform an arbitrary shift in shape file coordinates:

  • Insure the shape file is in UTM coordinates;
  • Find the amount of the shift to apply:
    • Plot the map with the shape file; insure that your primary map datum is the one you want to use.
    • Use the Calculate distance tool
    • The dialog will give you the shift in both x and y directions.
    • Take several well defined points spread over the map, and average the results.  If the numbers for the shift are not consistent, this is not the primary problem.
  • Shift the coordinate with the "Resample, Shape File, Specify Shift" menu choice on the data manipulation form.
  • Set the shift in meters in the x and y direction.
