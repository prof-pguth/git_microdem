Edit DEM Grid

  View the actual grid data stored in a DEM, available with the "Edit  DEM grid" option on the Edit Menu.

After picking the option, outline the region you want to edit.

The most common convention for DEMs has the 0,0 point in the SW corner, with rows increasing northward and columns eastward. This convention differs from the common convention for imagery, but MICRODEM will handle all details for any data its supports.  This will show you with the SW corner for the origin.

The coordinates at the bottom show the location of highlighted grid value.

This is not a very efficient way to edit data, but it can be done.   You must resave the DEM or you will lose the edits.
