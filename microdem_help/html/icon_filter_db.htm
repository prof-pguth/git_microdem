Icon Filter DB

Option on  Table view of a database, Edit, Icons.

 

Use this file to set icons for records in database, using Filter applied to each record.

The ICON and ICON_SCALE fields will be added to the database, and  an  icon will be created in c:\mapdata\icons for each symbol, which can be used for KML database export to Google Earth. 

Database of earthquakes, after creating icons based on magnitude with both color and size set, and exported to Google Earth.

You can make a legend, with a screen capture of the NAME column in the database, and place legend in lower or upper left corner of screen for KML export.
