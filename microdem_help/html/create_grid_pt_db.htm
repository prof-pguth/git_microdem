Create Grid from Point Database

Option on Report menu of database table form.

Makes grid with the same extent as the open map, and a projection and pixel spacing selected.

  • Values from DB: the grid point closest to the record will get the value.
  • Radius from DB: all points within the radius specified by the RADIUS_M field will get the value.
  • Box: specify a radius in m, and all points will have the value of the point.
  • Specify code: all points in the database will have the specified code attached.
  • Point density: specify a grid size in meters, and the program will create a grid with the number of points in each grid cell.

For the first three options, you will pick the fields from which to create the grids, and the value of the field will be placed in the grid.

This uses a drop in the bucket algorithm.  If multiple points would plot at the same location, the value of the last will be the one that appears in the grid.
