Force Redraw

The program should generally be good about knowing when to redraw the map when you change something about the data shown, the coverage area, or the display options.

The exception will be to clear temporary features, such as the trace for a distance computation.  For this you can use:

Force redraw

  • on the map toolbar
  • on right click on the map popup menu
  • on Modify menu

If you do need to force a redraw because you have confused the program, use the Modify menu

  • Force redraw base map layer
  • Force redraw legends and scalebars.  This will just erase any map data that hides part of the legends or scalebar.
  • Force complete redraw--deletes all stored layers.  Should not be needed, but in case you do something and a layer should be redrawn but is not, this option is available.  It will be slower than the other two.
