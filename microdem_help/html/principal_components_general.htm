Principal Components 

Because many data sets have variables which are highly correlated, there is actually a lot of redundancy in the data. It might be possible to create fewer variables that contain all the information in the data set, and requires less storage space and processing power. The resulting images can be "better" than any of the original images.

  • For satellite imagery, Selected on Imagery analysis menu,
  • For a database

This method does the following:

  • Extracts the eigenvalues from a covariance matrix.
  • Orders the eigenvalues, and computes the percentage of the variance explained by each eigenvector.
  • Computes the loading of each band on each of the principal components. The sum of the squares of the loadings on the principal components equals 1.
