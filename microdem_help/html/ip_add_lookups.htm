IP Address Lookups and Mapping

Services exist to compute a location (lat/long) for an IP address.  This can be used for targeted advertising when you log onto a site (it gets your location before returning a page to you, and thus can tailor the ads you see), or to track the locations of people interested in your web site.

To get your IP address:

  • Open a command prompt window, and type "IPCONFIG".  The resulting address might actually be internal, if it starts with 10.xxx.xxx.xxx or 192.168.xxx.sss.
  • To find your "real" IP address:
  • http://www.whatsmyip.org/ This should give you the external IP address that the web sites you visit will see, which might be something like 131.121.xxx.xxx.
  • https://www.wizcase.com/tools/whats-my-ip/ does the same thing
  • http://www.whatsmyip.org/whois-dns-lookup/ and enter your external IP into the Whois lookup.  This will tell you who owns your IP address

Directions for IP-Mapper (http://www.zonums.com/iptools/ipmapper.php) for a number of addresses:

  • Get your IP addresses into a list, one per line or cell, in Notepad or Excel.  If it has not already been done, you must remove everything but the IP address itself.
  • Cut up to 50 addresses, and paste into the memo box.  Excel will make it easy to get the addresses in blocks of 50.
  • Insure that you do not have a blank last line ("There is an error at line #22" will indicate that you neglected this check).
  • Hit the geocode button.
  • Pick a file format.
    • KML format.
    • If you pick CSV, you might have to edit the file because some countries have a comma in the name (Korea, Republic of) and will cause problems.
    • Tab version might be better than CSV.
  • Cut the text out of the memo box, and past it into Notepad.  (There might be problems trying to use the download button.)  You want the line with the headers, but not the lines above that.
  • To use KML
    • If you want to open directly in Google Earth, save each export separately.
    • You can combine all the exports into a single file in Google Earth, save it with a KML extension.
  • To use CSV or tabs:
    • If you need more than 50 IP addresses, repeat until you have all the points you need.  For these subsequent groups you do not need the headers.
    • Save the file as a CSV.
  • Open World Base map
  •   Open Database and pick the CSV or KML file you saved.  This will automatically create a dBase DBF file.
  • Report,  KML: export for Google earth.
