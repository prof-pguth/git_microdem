Triangulation Contouring

Steps in triangulation:

  • Group data into triangles
  • Find where contour lines cross join between two data points
  • Must exit on one of other two sides of triangle, and that elevation can occur on only one of the other sides.
  • Early problem in finding right optimal set of triangles (want triangles closest to equilateral; with no long skinny triangles)

Delaunay Triangulation: first pass find optimum triangles

Thiessen, Voronoi, or Dirichlet polygons

Many applications: town locations in middle of regions (peasants go to nearest town for market), crystal cooling

  • find points that are closest to each data value
  • each data value has neighbors
  • connect point to neighbors, and get triangles
  • draw only one triangle at at time (do not know where contour goes, it will eventually be completed)

Honors every data point--no data point will be on the wrong side of the contour lines.

Strictly speaking triangulation does not extrapolate beyond data points and has an irregular edge at data points, but the edges may be "beautified" by extrapolation.

Can smooth the curves later for more pleasing look to avoid the straight line segments.
