GIS Color from DB Numeric Field

Picked from the "Plot" button menu on the database table display window.

 
 Pick color palette and range of values to use for assigning colors
Pick field to use; it can be from a joined table, indicated with "LINK-".
  • Field lists the range of values in the field in the data base, as filtered.
  • Color range lists the color range picked with the Pick color palette  option.
Scaling: linear, quantile, or log.  You must use a chloropleth color scheme for quantiles.

Legend caption:

Bottom panel controls:
  • Color from string field
  • Color from numeric field
