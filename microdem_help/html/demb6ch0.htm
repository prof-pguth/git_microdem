Crooked UTM Grid Lines

For lat-long data sets, true north is to the top of the screen, while for the UTM based data set the grid north is to the top. This provides the fastest operation with each type of data, and avoids a rotation if the other north were placed at the top of the map. UTM grid lines for arc second data break because true north lies to the top of the screen, and grid and true north only coincide at the center of a 6° UTM zone. Near the edges of UTM zones the grid lines definitely do not run true north-south.
