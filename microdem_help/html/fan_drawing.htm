Fan Drawing Options

 
  • LOS and fan algorithm.  Unless you are investigating the effects of the different choices in the LOS algorithm, these parameters should probably not be changed.
  • Show: you can show either the masked or the visible terrain, or both. If you want to overlay fans, you should only use the visible option.
  • New IHS fans: this options shows all points, using IHS colors so the underlying map will show through.
  • Visible color: pick color for terrain that can be seen.
