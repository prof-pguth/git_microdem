Map Overlay Manager

Invoked:

  • Icon on map toolbar.
  • Option on popup menu when right clicking on a map.
  • Option on overlay menu.

Use checkboxes at the top to enable/disable layers.  Some checkboxes requires that the corresponding Menu choices be selected.

  • Database display--these have the DB table open.
  • TIGER options--the data must be in the correct location, and indexed.
  • Contours
  • Carto DBs--drawn only, and not opened for GIS database operations (may not be working); use options on Database menu
  • Grid
  • Gazetteer
  • PLSS
  • SRTM water bodies
  • World outlines
  • US outlines
  • Tissot

Use SHP button to add Database (shape file) overlays.

For panels in the middle of the form:

  • Use buttons on the left to modify drawing options.
  • Use the arrows to move layers up/down in drawing order.Layers at the top will be drawn last, and will be on top of the map. They may obscure lower layers.
Viewshed/Sensors Layer: Trackbar sets opacity of  viewsheds layer.
