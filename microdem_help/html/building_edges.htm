Building Edges

  • Option on Calculate menu
  Min/Max building height: specifying both lets you try to pick out different height classes.

Edges:  there must be between the min and max number of immediate neighbors lower than the min building height.

Roof: if there is at least one point the min building height or more below the point, within the search radius, the point will be noted as a potential roof.  The roof height and radius will be in the database.

 

This will create a database.  Turn it into a grid with Report, Create grid option.
