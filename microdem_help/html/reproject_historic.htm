Delaunay Triangulation Reprojection of Historic Maps

Create Delaunay triangulation which warps the map to fit control points.

Use "Export registered image" to open in MICRODEM.

Applies to:

  • Register with Map Analyst
  • Register Quad Map
  • Register Quad Map with gazetteer supplemental points
  • Register historic MrSid Map scans
  • Tutorial: Register Image and Digitize
  • Extrapolate to the edges of the map after you do the Delaunay triangulation, and then retriangulate.
