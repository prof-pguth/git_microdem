Image Overlays

Option Imagery (KML, KMZ, world files) on  Overlay menu

  • Loads:
    • Image with a world file
    • KML file with a single Ground Overlay.  The overlay cannot have a rotation, which means that the edges of the map must be parallels and meridians.  You can create this in Google Earth with the "Add image overlay" option.  If there are multiple overlays in a file, this will work best if they do not overlay each other.
    • The opacity for the overlay can be set on the Map Overlays form, Vector tab
  • Clear with the Overlay, Clear Overlay menu choice.
