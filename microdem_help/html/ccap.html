NOAA Coastal Change Land Cover

NOAA Coastal Change Analysis Program Land Cover (CCAP or C-CAP)
  • Download at https://coast.noaa.gov/dataviewer/?#/landcover/search/
    • Pick UTM projection
  • https://coast.noaa.gov/ccapftp/#/  Download, no choice on the projection, and you will probably want to reproject to UTM
  • for very limited, targeted areas, NOAA has 1-5 m resolution.  https://coast.noaa.gov/htdata/raster1/landcover/bulkdownload/hires/
  • Online atlas: https://coast.noaa.gov/digitalcoast/tools/lca.html
  • Categories used: https://coast.noaa.gov/digitalcoast/training/ccap-land-cover-classifications.html

Using CCAP in MICRODEM:

  • Open the file as a DEM/grid
  • Edit, Set z units to see the category names
