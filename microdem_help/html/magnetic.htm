Magnetic Model

Microdem uses the MAGVAR code, an NGA product. The code, data files, and documentation are available from NGDC. The code has been converted to Object Pascal, but was also converted to a C DLL. Both reproduce the test results.

The World Magnetic Model coefficients are produced jointly by the British Geological Survey (BGS) in Edinburgh, Scotland, and the U.S. Geological Survey, Golden, Colorado USA, on behalf of the British Hydrographic Office in Taunton, England, and the National Geospatial-Intelligence Agency, Washington, D.C. The model, associated software, and documentation are distributed by NGDC on behalf of NGA. These models are produced at 5-year intervals.

The program requires the data file wmm_xxxX.cof . This is supplied with the program and will be updated automatically, or can be obtained from NGDC at the link above.  If you get the file from NGDC, you will have to rename it.  The program currently uses the WMM2025 (2025-2029) coefficients file.

Current code is at https://www.ncei.noaa.gov/products/world-magnetic-model

 MAGVAR IS COVERED UNDER DMA MILITARY SPECIFICATION: MIL-W-89500 (1993).
