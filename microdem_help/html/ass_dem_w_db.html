Associate DEM with Database

Many operations on a database require that it be associated with a DEM.

The easiest way to do this:

  • Open Grid/DEM
  • Database (shapefile)

Other ways

  • Open another map
    • Open image
    • Open scanned map 
    • Open Landcover map
    • Open Vector Map
    • Open quick vector map
  • Database (shapefile)
  • Make the map window active, then Modify associate DEM with map
