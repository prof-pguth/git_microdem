Color Database Field

  • Insure that the database contains a COLOR field. If needed, add it to a regular dBase file (10 character integer field) or to a shape file.
  • Insure option to edit a database is enabled.
  •   Open database:
  • Check edit checkbox.
  • Double click on a single record to color it.
  • Pick edit button for options on coloring multiple records.

Line shape files: 

  • Insure  database contains LINE_WIDTH and  LINE_COLOR fields. If needed, add  to shape file.
  • Insure  option to edit a database is enabled.
  •   Open database:
  • Check the edit checkbox.
  • Double click on a single record to color it, and pick the option from the pop up menu.
  • Pick the edit button for options on coloring multiple records.  You can either color all active records, or only those not already colored.
  • This coloring will be used by default when the shape file plots.
