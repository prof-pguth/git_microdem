Legend Placement Selection

  Selected from Map Marginalia form after a right click on map.
  • Checkbox if item displayed automatically on map.
  • Location of item on the map.
  • Orientation (will apply to all legends; use vertical very sparingly)
  • Size: affects the length of the legend (will apply to all legends)
  • Font--the same font will be used for all legends.
  • Bar size: height in pixels
  • Tick size: in pixels
  • x location: if other than 0, this will be the pixel coordinate of the left edge of the legend.
