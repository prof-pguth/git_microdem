Quick DB Filtering

Picked from Quick filter on "Plot" button menu on database table display window.

  • Pick one field or two fields, which must be integer or string/character.

 

  • Options to show the filter, and the number of records, on the upper left corner of the map
  • Change the font
  • Use "AND" or "OR" to combine the two terms of the filter.
  • Cycle through all the records matching the filter with the "<" and ">" buttons, and highlights them on the map.
  • Use filter for all the databases
  • If there is a graph up with the database, it will be redrawn with the new filter
