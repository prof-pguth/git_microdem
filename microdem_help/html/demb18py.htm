Problems with Text Files

The compiler used for this program assumes the DOS/Windows standard that lines in a text file will be terminated with a carriage return and a line feed. Some other operating systems do not have this convention. Files prepared on such other operating systems may have problems.

Data Manipulation form   Edit menu there is an option to insure that every line ends with a carriage return and a line feed. This will replace the original file.
