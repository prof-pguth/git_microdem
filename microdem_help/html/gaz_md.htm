Gazetteers in MICRODEM

Enabler Gazetteer operations on menu tab of options form.

You must have a gazetteer symbols file  in the program root directory.

Select GAZ button on main program toolbar, pick DBF file desired, to open a vector map.

Alternatively, right click on any existing map, and select Gazetteer option from Load submenu choices.

Search on NAME/FEATURE field, using edit box on gazetteer toolbar.  As you type entries are automatically filtered.

You can merge NGA or USGS gazetteer files, but cannot mix the two types because they have different fields and formats.

You can also use gazetteers to label peaks on panoramas, and a variety of features on maps.
