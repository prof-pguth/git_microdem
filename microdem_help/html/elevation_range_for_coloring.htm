Elevation Range for Coloring

Access with z range button from

  • Elevation Map Options
  • Reflectance options
  • Edit,  Mark as missing, Interactive adjustment
    • Turn on clip colors
    • Set contrasting colors for the highs and lows
    • Specified for the z Range
    • Mode the Max and Min
    • Force screen redraw to see the effect
    • Mask to Z range when range is right
  You can restrict colors to get better detail, or to match multiple maps.

Percentiles allows you to ignore a few extreme elevations.  If you pick the 1% and 99%, the top and bottom 1% will plot with the same color.  This works when you have a skewed distribution with long tails.

Clip colors

  •  Checked, all those above/below the limits will be show in a particular color, selected with the buttons above.
  • Unchecked, all colors above or below the color range will use the max or min colors in the color scale

Mask  to z range: all values above or below the current color scale will be marked missing.  This will only be in memory; you must save the DEM/grid if you want the changes to be permanent. 

 

Stretching

  • None
  • Percentiles--equivalent to a histogram equalization.  All colors should appear with the same frequency.
  • Standard deviation--no difference from no stretch, but the legend will be in terms of the mean elevation at 0 and standard deviations away from it

Force screen redraw to see the effects of the changes without leaving the form.
