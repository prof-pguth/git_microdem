Export (Right map click menu)

  • Quick map to Google Earth: export just the map, as a raster, to KML and then open in Google Earth or other default viewer.
  • Deliberate map to Google Earth: KML export map image, which provides options to set parameters.  The choice is set on the Web tab of the the options form.  KML database export can be done from the Report button's popup menu of  database table form.
  • Grid/Graticule to Google Earth (using settings from Grid, where you can also export it)
  • Location to Google Maps: opens Google Maps with the default browser, and puts a push pin at the location you clicked.
  • Point cloud to GoogleEarth
  • Point cloud to database
  • USGS data, TNM viewer; gets USGS color imagery
  • Point cloud to OpenGL
  • Drape current map to OpenGL
