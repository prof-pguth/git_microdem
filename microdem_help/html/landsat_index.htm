Landsat Full Scene Index

Put Landsat full scenes in a directory, one scene per sub directory.

On the main menu, Analyze, Create, Landsat full scene index. This will create a database in the directory with the files, which will look in any subdirectories of the main directory.

You can filter the database by data, sensor, cloud cover, sun elevation.

You can load the image, by highlighting a record, Double clicking on record in data base, and selected the Load, TM Scene
