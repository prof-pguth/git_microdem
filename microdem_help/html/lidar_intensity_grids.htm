Lidar Intensity Grid

Create on "TOOLS" tab, Grid based options of point clouds options form.  This makes most sense with a very dense point cloud.

Generation will be much faster than plotting on screen of the point cloud, and you can manipulate the resulting grids.

The intensity grids can be dominated by a very small number of very large returns, which leads to very dark images which should probably have the display parameters adjusted.

The all intensity option also creates first and last return intensity grids.  These will be drop in the bucket, and the last return processed will be the one in the grid

Maximum intensity.  Scaled for the display to stretch the contrast (1% to 99%).  Probably the best choice for display
Minimum intensity.  Scaled for the display to stretch the contrast (1% to 99%) 
Intensity difference (Max - Min).  Scaled for the display to stretch the contrast (1% to 99%)
