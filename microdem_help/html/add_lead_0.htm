Add Leading Zeros

Option on String field operations, Edit button on Table view,

Pick a field, and a desired length, and leading zeros will be appended.  This can repair a field like a Zip code (New England) or FIPS (A and C states) code that has been treated as an integer in a program like EXCEL, and thus cannot be used for join/link databases.  Excel is a likely cause for the loss of leading zeros.
