Database Symbol Selection

This menu appears

  • when the leftmost button is pressed on the database table form
  • multiple database legend.  
  • From the "Display" button on form that selects display options

If the database has one of the special color schemes (Koppen climate classification, geology dip and strike, earthquake focal mechanisms) it will be symbolized for that type of data.

 This will take you to the selection options for the current display mode:

  • Color code by DB numeric field:
  • Color code by DB string field:
  • Color field in DB
  • Proportional symbols/Icons:  (True Type fonts)
  • Default symbols
  • Icon field
  • Single icon for entire database
  • Zipatone Field
  • Single Zipatone for the entire database
  • Vector data: use two fields (magnitude and direction) to plot vectors on the map.
  • Two or three field RGB
  • Positive/negative
  • Earthquake mechanisms
  • Point symbols in DB
  • Color/line weight in DB
