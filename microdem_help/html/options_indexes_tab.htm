Indexes Tab

Tab on options form,  from Options menu selection.  Local map databases, for Map library can display the location of your data on a base map for graphical map selection.

 

  • Automatically close index maps: or leave them open to immediately open data covering a different region.
  • Map library picks the drive with the data.  It has not been tested for a location other than the root of the drive, but it can be on an external drive
  • Library settings: pick which types of data can be opened, and the series in use.
