Experimental options on the menu of the data manipulation form:

  • Under active development
  • File names, directories, and DB fields are hard coded, and in some cases it would be difficult to generalize the option.
  • May have very little applicability beyond very specific conditions.
  • If they turn out to be useful, they will be generalized and moved elsewhere in the program

The current options include:

  • Merge CSV files. Operates on two directories. The files must have matching names, and the same number of lines, and no header line. The contents of the two files will be combined in a single file with a comma separator.
  • Cloud compare. Adds Roughness and Gaussian curvature. The column labels will have to be repaired when the operation is done.
  • Batch CSV to DBF: batch conversion, works with many (hundreds) of files. If the files do not have a header line, you will be asked to enter it for the first file, and then all the other lines will use the same header.
  • Histograms by class: create histograms of several fields, for each class in the database. BMPs stored in the directory.
  • Add colors: add color field to the databases, using a join table (e.g. with the LAS color codes)
  • Color maps: makes a BMP with a map view of each table, colored by the COLOR field.
  • Stats by class: for each file, computes basic statistics for each CLASS.
  • Add ground classified points: adds to the database the ground classified points, where they only have X,Y,Z ASCII lines in a file (for instance from CSFDemo_v2)

The following options have not been updated recently:

  • Photogrammetry from video
  • Lidar map displays
  • Multiple band imagery
  • ETM bathy
  • Point clouds from imagery
  • Batch TM options
