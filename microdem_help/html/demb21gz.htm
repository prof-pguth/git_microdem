Frequently Asked Questions

How do I register a satellite image?

  1. The best results will be obtained with imagery that is already registered.
  2. Open the DEM that goes with the image, and then the image.
  3. Go to registration, add point. It will now be checked.
  4. Double click on three registration points, and then enter the coordinates. You will enter MGRS, UTM, or lat/long depending on your present choice. You should spread these throughout the image for best results. You can add more than three points, but currently only the first three will be used.
  5. Uncheck registration, add point. When you move the cursor over the image, the coordinates should now be displayed.
  6. If you make a mistake, you can either edit the XY registration file in any ASCII editor or delete it and restart.
  7. This assumes that the image has been geometrically registered, or that you are willing to assume that you can ignore geometric irregularities.
  8. No error checking will be done beyond verification that you enter at least three points.

How do I get my local datum?

  1. Go to the options menu choice, and pick the maps page.
  2. Pick your local map datum. The program will use this for all calculations and computations, performing the datum shift from the digitizing datum. The datum code will be displayed with coordinates when you roam on the map.

What is a CDO file?

  1. This is an ASCII DEM purchased from USGS. You can rename it as a DEM file, or TerraBase II/MICRODEM will now provide an import filter for it.
