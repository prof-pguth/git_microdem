Box Plots

Option on Stats menu of table display window.

Database must have values for the distribution, in the following fields: 'NAME','MIN','MAX,'PC99','PC98','PC5','PC95','Q1','Q3','MEAN;

On the left:

  • Point symbol at the data set minimum

In the center:

  • Wiskers from the 5th to the 95 percentiles
  • Box from Q1 (25th percentile) to Q3 (75th percentile)
  • Vertical mark at the mean

On the right:

  • Point symbol at the 98th and 99th percentiles and data set maximum
  • If these are extreme outliers, they may be missing

On the left:

  • Point symbol at the data set minimum
