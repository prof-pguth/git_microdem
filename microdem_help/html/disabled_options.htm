Disabled Options. 

These could be re-enabled if anyone actually uses them. Most were related to TerraBase II.  We are working to eliminate duplicate procedures and code.

  • MGT--use newer options in the geological oceanography course
  • SPLAT
  • GMT
  • UAV scenario  
  • Target approach
  • Import ERDAS LAN/GIS and IMG formats.  Might work the GDAL.
  • Import CSIL index.
  • Gazetteer indexing (too many points were outside state or national border to make this very useful)
  • Read old format USGS DOQQ;
  • Import USGS DLG
  • Button to open Geotrans
  • Data sets formerly in the USGS database and NGA data base and DIN Indexes must now use Map library 
  • Scanner operations.
  • DXF digitizing, display, and contour line export.  Contour line export uses shape files.
  • Display PNT vector files.
  • Alberta ASCII DEM import
  • NLAPS TM BSQ: the USGS/EROS format for Landsat TM, in band sequential format.
  • Read/write Targa (TGA) graphics format.   If you need need, save as a BMP and find a program to convert.
  • Modis Rapid Response  removed 12/31/2012
  • Blue Marble   removed 12/31/2012
  • GPS NMEA logging not working and special waypoint.  You probably want to do this on a tablet or cell phone anyway, and standard GIS database operations.
  • Geowall
  • NGA data format options, and other military operations
    • Direct plot VPF.
    • Do anything with RPF TOC files.  Use Map library
    • Import CDs from NGA with DTED and RPF. Use Windows commands to copy them to the correct directions, and then index them.  
    • Use FalconView data in FalconView directory structure.
    • DTSSOP 
    • IPADS base map generation
    • Geosym Evaluation and CGM demonstration, for Geosym display of VPF 
    • PAPA (Pipeline Automated Planning Aid)

  • -PAPA: Pipeline Analysis mode
  • -MEA: MEA mode. 
  • -DTSS:  display DTSSOP products.
  • -IPADS
  • -POCKETPC
  • -MODIS
  • -QUICKVIEW
