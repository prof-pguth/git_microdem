CSV File

A CSV file (Comma separated variables or values) is a least common denominator for exchanging data among programs.  It is an ASCII file which used commas, tabs, or spaces to separate the entries.  Other separators are less common (such as "|").

CSV files can be viewed or edited in a text editor like Notepad or WordPad, or in Excel.

The handling of "exotic" values in the file, such as actual commas in a text field like an address, can be problematical.  If there are problems, you can edit the file to replace those values with another character, and then change back once you have the data imported.
