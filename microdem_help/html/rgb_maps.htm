RGB Three Parameter Maps

Available from Raster GIS menu.

  Select 3 grids.  They do not have to have the same resolution, and may not need to have the exact same coverage area.

Set the max and min values for coloring for each grid.

The initial ranges for coloring can be either the extremes in the grids, or a percentile.  If the distributions are extreme, the percentiles can provide a better distribution of colors.

Plot redraws the maps.

Color range sets the color levels for the max and min values.  The best selection is probably percentiles, to skip the tails of the distrubtion, between which there will be a linear stretch.
Map showing three geomorphometric parameters.

Stoker (2010) showed examples of this with LIDAR grids and derivative surfaces.
