Points in polygon (areas)

This uses the Point in Polygon operation, a standard GIS operation.

Option on "EDIT" button on database table form for an open shapefile.   If your original shapefile is an area or line shape file, the centroid will be used, and must have been computed previously (Field geometry).

You  then pick an open area shapefile, and the field in that database to insert into the first database.

For those records which lie within a record on the area shapefile, the desired field value will be put in the first database.

This could find the County outline which contains a record.  You want to be careful in picking the field to insert: for counties, the FIPS are unique, but half the states have a "Washington County".  The FIPS will also enable you to find the counties.
