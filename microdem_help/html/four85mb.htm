Get Fourier Options

Select the following parameters for the FFT:

  • Window: this attempts to control power leakage into adjacent frequencies. The Parzen or Welch should give the best results.
  • Overlap segments: if the series is longer than the segment size, you will combine a number of segments. If you overlap them, each segment will share half its points with the segment before and half with the segment after. Except for the initial points and the end points, each point will be considered twice.
  • Zero pad end series: to get the series to an even power of two, zeros can be appended to the end of the series.
  • The segment size for processing must be a power of two. The segment size determines the speed of the operation and the frequencies that can be resolved.
  • Whether the power density spectrum will be plotted as periods or frequencies (they are reciprocals).
  • The bins and Units are used for labeling the axes of the graphs.
