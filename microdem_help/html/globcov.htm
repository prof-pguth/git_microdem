More Recent 300 m C3S Global Land Cover and Land Cover Maps - v2.0.7

GLOBCOVER uses the 300m MERIS sensor on the ENVISAT satellite mission.

  • GlobCover 2009 and 2006  (10", about 300 m)  http://due.esrin.esa.int/page_globcover.php is a 372 MB download, which includes a compressed TIF file.  In an uncompressed format, the file is 7 GB in size, and is probably painfully slow to display.   Users should open in QGIS, and immediately subset the file to cover just the region needed.  Keep "GLOBCOVER" in the file name so it will be recognized.
  • GlobCorine 2005 and 2009 for Pan-European region (300 m) http://dup.esrin.esa.int/page_project114.php   Europe, North Africa, Middle East, Turkey, and much of Russia.  Costs money now(?)
