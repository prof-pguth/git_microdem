Create Mask From Grid

Option on the Raster GIS menu.

Change the Z values for the mask.  The program shows the values, and the percentiles that they represent within the grid.

Plot shows the effect.

Export grid creates a new grid, with the value you want.  All other points will be 0.

Masking by category.  This allows edits of the DEM/grid based on category membership. This will be in memory; to save, you must save the DEM/grid.  Reload the DEM/grid to undo the mask.
  • Mask cat in: in the DEM/grid, all points that meet the single category will be retained, and all others removed. 
  • Mask cat out: in the DEM/grid, all points that meet the single category will be removed.
