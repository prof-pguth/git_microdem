Point Selection Along LOS

Point selection along the LOS can use:

  • Scaled radial spacing:  you can set the spacing as a fraction of either the DEM spacing (a constant for a DEM), or the map pixel size (which varies as you blow up the map).  Whichever number is smaller will be chosen for the LOS.  The Nyquist sampling theorem suggests the default of 1/2.  This is the only option recommended.  The only recommended changes, which only slightly degrade results and speed up computations by a factor of two, would be to set the two spacings to 1.  
  • Constant radial spacing: select the spacing in meters.  If you used DEMs of varying resolution, this can lead to poor choices (either overkill or insufficient sampling).

Yoeli (only selection points between two grid nodes) and Bresenham (only computing for grid nodes) options have been removed

This is selected on the LOS/Weapons Fan Algorithms form.
