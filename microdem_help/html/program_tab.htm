Program Tab

Tab on options form, brought up from Options menu selection.

 
  • Can save screen real estate (particularly useful with a netbook, laptop, or tablet),  since most choice are available via menus.
    • Show Toolbars:
    • Show Menus
  • Save defaults on exit: turn off if you want to save defaults only manually.
  • Auto open: you can open a data set automatically when the program opens
    • Last project
    • Last DEM
    • Vector map
  • Menus: flavors of the program, with simpler menus.
  • Menu settings
  • Status bar: change widths of the panels in the status bar at the bottom of the screen
