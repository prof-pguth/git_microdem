Fire Data

Global

  • Global fire, interactive online map: https://www.globalfiredata.org/fireatlas.html
  • NASA FIRMS
  • ESA Sentinel-3 World Fire Atlas

US:

  • https://inciweb.nwcg.gov/   GeoMAC was replaced in 2020 by NIFC (National Interagency Fire Center)
  • Monitoring Trends in Burn Severity, MTBS: https://www.mtbs.gov/
  • Historic
    • https://data.fs.usda.gov/geodata/edw/datasets.php?xmlKeyword=S_USA.FinalFirePerimeter: 186 MB shapefile of fire perimeters.  Consider subsetting before use ( File, Import, Shapefile after subset to match map extent).
    • https://data-nifc.opendata.arcgis.com/datasets/nifc::wfigs-wildland-fire-locations-full-history/explore?location=-0.000000%2C0.000000%2C0.00  can download a shapefile, but it only contains points, and does not have the fire name
  • Active fire mapping program
    • https://fsapps.nwcg.gov/
    • Current perimeters: https://data-nifc.opendata.arcgis.com/datasets/nifc::wfigs-current-wildland-fire-perimeters/about
  • NASA Fire Information for Resource Management System (FIRMS):
  • Landfire:  https://landfire.gov/viewer/   GIS data, especially vegetation type
  • Drought relationship: https://www.drought.gov/drought/data-maps-tools/fire
  • Fire model (may be outdated)
    • https://www.firelab.org/project/flammap
    • https://www.firelab.org/project/farsite (older, no longer supported)
  • Short, Karen C. 2022. Spatial wildfire occurrence data for the United States, 1992-2020 [FPA_FOD_20221014]. 6th Edition. Fort Collins, CO: Forest Service Research Data Archive. https://doi.org/10.2737/RDS-2013-0009.6
