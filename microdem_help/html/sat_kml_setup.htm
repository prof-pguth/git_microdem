KML Satellite Export Parameters

Requires Delauney triangulation registration of the map, and called with two options on the File menu of the satellite image analysis window.

  • Tile for Google Earth
  • Export registered image.  Selection of this option will close the satellite image analysis window and open a regular map window.  Pick the name and location for the new image.

Set the following, some of which might not be available depending on the option you selected:

  • Image blowup.  25% will show 1/4 of the rows and 1/4 of the columns (1/16 of the entire data).  This trades off legibility versus size.
  • Max tile size: Google Earth once had problems with sizes over 2000 pixels. The tiles will be the smallest number of equally sized images to cover the area without exceeding your specified size.  Tiles size no longer appears to be an issue.
  • Whether you want to zip the KML file into a KMZ file.
  • You can put the out in the Windows Temp Directory (which will be deleted when you close MICRODEM; the alternative is a new directory in c:\mapdata\kml, where you can recover it if you do not save it from Google Earth.

As you change the parameters, the preview to the right will change, as will the report with the number of tiles and their sizes.

If you want the output to be a single file, increase the Max tile size.
