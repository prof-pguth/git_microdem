DB Menu Options

Menu Option when a map window is active, to open vector database.

  •   Load: opens database with its table for Map query or filtering.
  • Standard: load states, counties, rivers, or highways.  This will open the table for use in operations like masking. The files to use are defined on Directories tab of options form.  Map Overlays also opens these.
  • Load cartographic overlay: opens a database with a list of layers solely for use as a map overlay.
  • Load OSM overlay: pick the directories, which must all be subdirectories of the same main directory, with the OSM data.  This allows, for example, selecting a number of cities, and as you zoom the correct one will be selected.
  • Shape file grouping
    • Load DB grouping
    • Add files
    • Update grouping
    • Create grouping
  • ID 
  • Merge multiple CSV/TXT files: alll files must have the same structure, and they will be loaded into a single DBF, in alphabetical order.  This is intended to work with GPS log files; there will be no intermediate files left on the hard disk.
