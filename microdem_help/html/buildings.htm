Building Outlines

 

Microsoft project, vector outlines.  This will be a huge file with challenges.

  • https://github.com/Microsoft/USBuildingFootprints/,  geojson format by state,
    • convert to shapefile with ogr2ogr (which will be slow, but shows progress in the command window) with is part of GDAL.
    • Subset to just the region you want as soon as possible
      • File, Import Shapefile after subset to match map extent, which required GDAL
      • Report, Shapefile export, Clip to map extent
  • Descriptions of the data and a project using them
    • https://www.nytimes.com/2018/10/13/insider/maps-data-visualization-america-buildings.html
    • https://www.nytimes.com/interactive/2018/10/12/us/map-of-every-building-in-the-united-states.html?action=click&module=RelatedLinks&pgtype=Article
  • State files are huge, so you should subset the shapfile to the region you need for best performance.

Alternatives

  • OpenStreetMap, vector outlines of buildings, but coverage varies from very good to not so good
  • Overture Maps
  • USGS from NationalMapViewer, but does not have house outlines, and only points

Use of buildings

  • Building density along coastline
  • Number of buildings within in area
    • Create area shapefile
    • Use Point in polygon
