Reclassify

This is a GIS operation that works on database fields or grids that use category coding.  Examples would be the MFCC in Tiger edges, or the NLCD categories.  It replaces on code with another.  Examples would be:

  • Upgrade a road category after construction.
  • Merge categories to simplify the map, such as making different forest categories into a single category.
