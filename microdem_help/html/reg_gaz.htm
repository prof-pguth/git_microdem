Register with Gazetteer

The first time you select the "GAZ" button, or if you have closed the gazetteer, you will be prompted to enter a Gazetteer, which will open.

The edit box on the lower right has a lat/long tolerance.  When you select an Image Point, if the map/image already has a rough registration, the gazetteer will be filtered to show only those points within the selected limits of the point's rough registration.  This will likely only have a single screen worth of points, and you can select the one you want.  If there are no matches for the location, select a different Image Point without moving to the Next Point.

If you then hit the "GAZ" button on the Registration window, the program will transfer the coordinates to the registration window from the gazetteer.  In addition, if the POINT name has not been filled it, it will be copied from the NAME (NGA) or FEATURE (USGS) field in the gazetteer.

You can use the "Find" edit box to locate the record you want in the Gazetteer table.  If there are multiple points, select the one you want so that the pointer on the left of the display points to it.

You can use these locations to register an image.  If you do this, you will have to enter the screen coordinates for the last point, and then move each control point to the bottom of the list and add its image coordinates.
