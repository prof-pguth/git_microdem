DEM Resampling

 

There are limited valid reasons to resample a DEM.  Resampling gets new data points, and can remove original data points.

Valid reasons

  • Resample to a larger resolution

The following are NOT valid reasons to resample:

  • Compute slope or similar derived properties.  Suggestions that the data must be in a UTM or similar projection are simply wrong.  Properly designed software can work with lat/long data.  Slope, aspect, hillshade algorithms for arc second DEMs
  • Deal with arc second data with the lat and long spacings are different in an attempt to keep pixels more square, which is done at high latitude for some global 1" and 3" DEMs.

Resampling options in MICRODEM

  • Save DEM/grid on the File menu
  • Downsampling on the Raster GIS menu (creating a lower resolution grid)
