DEM Metadata Grids

The following metadata grids for DEMs are supported, for automatic labels for the codes and creationn of legends.  The z unit code will be stored in saved files.

  • euCOPEDM = 65;
  • euCOPFLM = 66;
  • euTANEDM = 67;

The definitions are stored in the LAND_COVER_n.dbf file.

When you open the grid, it is recognized from the file name, and a VAT file is created to go with it.
