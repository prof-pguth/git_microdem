Multiple coverage summary

Option on viewshed toolbar.  You can select a target area (an open area shapefile) or compute the coverage of the entire map area.

This currently save the output for analysis by another program, and is not recommended.  Let us know if you would like to guinea pig improving it.

 

Map Colored by the number of sensors seeing a point.
