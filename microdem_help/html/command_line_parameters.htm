Command Line Parameters

MICRODEM accepts the following command line parameters which affect program operations.  They are not case sensitive.

  • -EXPERT: forces the program to come up in the standard mode
  • -GEOLOGY or -BRUNTON: option tailored for structural geology.
  • -MINIMIZE: open minimized
  • -NOAUTOOPEN: ignore automatically opening a data set; useful if some combination of map options causes a crash.
  • -RESET: reset the program defaults
  • -DEBUG: turn on the debug log
  • -NODEBUG: turn off the debug log
  • -RESET: reset all program default settings
