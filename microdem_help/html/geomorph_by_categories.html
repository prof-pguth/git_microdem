Geomorphometry by Categories

This is still experimental, and the top row of graphs might not be created

With the grid with categories as the active map, On  Raster GIS operation, pick this option

Works with two categories of grid

  • Numerical grid with a Fixed color palette from table
    • Display numerical grid with a color scale like dNBR and NDVI.
    • "Numeric grid with VAT to byte grid with codes" option on Edit, Reclassify menu.  This makesa new grid byte codes, and links a VAT table, with the color scale.
  • Grid with integer codes and a VAT such as vegetation types

Get histograms of elevation, slope, and roughness for each classification code in the VAT.  You must selecte the VAT grid as the active map, and then select a DEM that covers the area.

There will be 8 graphs created

  • On the top row, the percentage of the area in each bin of the histogram that falls within each category.  This can blow up on the tails of the distribution (the left side of the graphs on the left side, and the right side of the next two graphs)
  • On the bottom row above, histograms for elevation, slope, roughness, and aspect for each of the categories in the VAT
