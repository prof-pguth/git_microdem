Data Base Animate Field

Accessed from Plot button on database table display window.

Select a field (string or integer), and shows all the unique values in the data base on the map

  • Pick field to use from the combo box.
  • Pick font to use to put title and number of records in the upper left of the display.
  • Pick line size on color for the "Connect sequence" option.
  • Make a movie. 
  • Connect sequence: requires a field "SEQUENCE", and that it have records in sequential order starting with 1.
  • n>= requires that there be at least this number of records.
  • > Picks next record and plots it.  This is the fastest way to cycle through records, especially in you set "Immediate redraws".
  • < Picks  previous record and plots it.  This is the fastest way to cycle through records.
  • List box shows all records for selected field.  Clicking on one draws it on the map.
Symbol options. Will not appear if you are showing Icons.

Color options.  Will not appear if you are showing Icons.
