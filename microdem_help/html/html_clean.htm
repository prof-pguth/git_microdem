HTML Cleaning

MICROSOFT Office creates bloated HTML files.  An option allows you to do the following steps to clean up the code.

The following tags will have all options removed from within the tag:

  • <li>
  • <table>, <tr>, <td>
  • <body>
  • <div>
  • <p>
  • <span>

The following tags will be removed:

  • <p>, </p>
  • <span>, </span>

The following substitutions will be made:

  • &nbsp; will be replaced with a space.
  • <table> will be replaced with <table border="1">
