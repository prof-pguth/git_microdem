Projection Encoding

Data type Projection encoding
Raster, Geotiff Geotiff keys, in Geotiff file
Vector, Shapefile WKT in PRJ file
Point Cloud, LAS EPSG codes, Geotiff keys or WKT, in LAS file

If the projection is not encoded, or not correctly encoded, you may have to assign the projection.  GDAL can do this, or Lastools for lidar.  The Resample menu has some options to correct or assign the datum and projection.

Using EPSG codes may be the most efficient way to encode the projection; you can look them up online.
