Buffers in MICRODEM

Buffering theory

This option can to used to show buffers around features, or to use the buffers to mask a grid or database.

Options

  • Force screen redraw if not automatic
  • Show symbols in addition to buffers
  • Show buffers
  • Buffering/masking
    • Mark points in the buffer
    • Mark points outside the buffer
  • Pick the buffer size in meters
  • Set opacity, and color of the buffer/mask
