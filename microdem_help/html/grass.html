GRASS

Raster GIS, External tools menu for the currently available options.

These use a batch file that will be in c:\mapdata\temp.  If the operation fails, run the batch file in a command prompt and watch the failure messages.

Get installation from OSGeo4W

To get GRASS extensions, use the option on the File menu from a DEM map

Currently set to use GRASS83 which now comes with OSGEO4W.

https://grass.osgeo.org/learn/manuals/

Insure that you open QGIS with the option to include GRASS
