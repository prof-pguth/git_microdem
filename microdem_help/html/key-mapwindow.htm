Key Buttons--Map Window Toolbar

  • Set percentage data shown for raster data
  • Zoom out to show entire data set.
  • Subset and zoom data set
  • Zoom in (blow up).  Indiscriminate use of this will get you a huge map.  Subset and zoom is generally a better choice. 1:1 view (best resolution view of the data, but may be large and unwieldy unless you subset and zoom first)
  • Force screen redraw--removes any extraneous annotations and markings
  • Information on map, pixel size and percentage of data displayed.
  • Identify data base records.  Double click on a feature, and see its attributes from the data base table.
