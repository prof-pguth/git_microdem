Color Palettes

The program has two palette files, stored in the directory with the program executable.

Choose from the elevation map options form.

  • Pick either fixed or stretched color palette
  • Select the palette from the name in the PALETTE field.

Scaled or stretched palettes

  • Select the palette from the name in the PALETTE field.
  • The scheme uses even divisions across the elevation range.
  • All colors will be used, stretched to color the min and max values in the data, or other values you can select manually.
  • N is the number of colors in the palette

Fixed palettes

  • NAME field will be used for legends.
  • Currently for
    • Geologic time
    • NDVI
    • dNBR
    • Your own palettes

Each color in a palette must have the same unique PALETTE name, and the colors must progress in sequence from low to high.

Online tools to create and explore palettes

  • https://sciviscolor.org/
  • https://sciviscolor.org/colormaps/discrete/
  • http://vrl.cs.brown.edu/color (up to 20 colors in palette)
  • https://medialab.github.io/iwanthue/

To add a new palette:

  • Edit the palette file (say in Excel), located in c:\microdem
  • Match the rules and structure of the existing entries
  • Save the file as CSV
  • Open database without map for the CSV file in MICRODEM
