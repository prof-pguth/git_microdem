Geotiff Images and Grids

GIS programs often distinguish between grids (DEMs) and imagery.  By saving as a Geotiff, it is possible to operate on a data set with both sets of tools (Image analysis and Raster GIS).

32 bit floating point grid, displayed with hillshading.
8 bit (byte) grid
  • Elevations stretched from 1 to 255 (range was -3 to 264) (Single grid arithmetic. Expand given z range).
  • Missing data set to 0
  • Geotiff save to 8 bit
  • 8 bit grid opened
  • Note stair stepping from quantization of elevations
  • As a grid, there are better options to edit and reclassify perform Raster GIS
8 bit (byte) image
  • if displayed with no stretch, would look like grid displayed as elevations
  • Can do Image analysis on the image, which has options not possible for a grid
