This shows the difference between using the Vincenty algorithm (UTM would be effectively identical) and using lat/long as if it were a cartesian grid. 

Generate these graphs, and adjust the scaling, with the "Analyze, Algorithms, Horizontal earth curvature" menu choice.  This requires an open DEM, and the computations will be for the center of the DEM.

Note that the graph is almost symmetrical about the 90° Azimuth, but is not quite because of the convergence of the meridians.
 
 For 10 km range, the horizontal earth curvature is almost 2 m.
