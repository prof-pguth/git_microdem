Digitize Contours

Choice from popup menu at the Annotate Map icon on map toolbar.  This requires that you use a registered map.  If you have a scanned map that you need to register, use the image registration option from the satellite image analysis window.

First open or create a new file for the digitized contours.  This will be a dBase table with three fields (LAT, LONG, and Z).  If you are appending to an existing file, the points already digitized will show on the map.

You have three controls:

  • Z value: this will be same until you change it, since you are digitizing contour lines.  The initial value is -9999, often chosen for missing data.
  • Mode:
    • Stream: as long as the left mouse button is down, points will be automatically digitized once the tolerance value is exceeded.
    • Point: points will be recorded only when you double click.
  • Tolerance: in stream mode, this is the distance a point must be from the last digitized point in order to be recorded.  This is measured either horizontally or vertically, and not along the diagonall.

As you digitize points, they will be added to the data table and marked on the map.

When your file is created, you have the following options:

  • Ultimately you would like 1-5 elevations within each grid square that you will interpolate. There are bad consequences for both too many and too few digitized points. See Picking the Grid Size.

Once the map has been scanned:

  •   Data Manipulation  form's Import, ASCII XYZ triples option  to import the resulting XYZ file and create a DEM.
  • TINs (Triangulated irregular networks) , which includes an option to create a DEM
  • Graphically edit XYZ data:   Open dBase table using    Database option from a map window, and you can edit Z values,  graphically move points, or delete points.
  • Fill holes.

This option should be undertaken only as a last resort.
