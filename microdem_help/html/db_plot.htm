Data Base Table Plot Options

The plot options cover displaying the data on a map window. These options will be on a popup menu when you select the PLOT button on the table display window, and there will one of these windows for each database that is open. Some options will be available only for certain types of database files.

  • Color code by DB numeric field:
  • Color code by DB string field:
  • Color field in DB
  • Color field in joined DB
  • Proportional symbols/Icons:  (True Type fonts)
  • Default symbols
  • Icon field
  • Single icon for entire database
  • ZipATone field
  • Single ZipATone
  • Vector data: use two fields (magnitude and direction, or x and y components) to plot vectors on the map.
  • Two or three field RGB
  • Positive/negative
  • Earthquake mechanisms
  • Point symbols in DB
  • Color/line weight in DB
  • Terrain fabric
  • Dip and strikes: geologic orientations, if there are "DIP" and "STRIKE" fields in the data base.
  • Point separation
  • Connect sequential points: for a point data base, plots and connects points in sequential order on the map. Option to put arrows on the line in the direction of travel.
  • Plot fan locations
  • Label fans
  • Plot coverage circles
  • Edit Layer symbology
  • Edit gazetteer symbology
  • Selected symbology: use the default symbology, assigned with the leftmost button on the table display window. All selected records will be plotted. By selectively filtering, changing the symbology, and picking this option, you can plot features differently.
  • Change TIGER symboloby
  • z values: Very fast display for 3D point shapefiles.  If you want more drawing control, or a legend, use Color code by DB field.
  • Record centroids:  if the area records are small on a map covering a huge area, you can display them with a large symbol so you can locate them to zoom the map in. 
  • Buffers: Buffers on the current map with the currently selected records in the database. 
  • Restrict by map scale
  • Zoom to DB coverage (must be a shapefile, or a point DB with an SHX which you can create): sets the map limits to just cover the database.  You can use Annotate Map, Create shapefile, Create shapefile with the map outline and then use the file to rapidly over maps to the desired coverage area..
  • Color code by ratio two fields
  • Quick filtering
  • Label
    • Label all records: pick a field, and label all data with the value in the selected field.
    • Label every nth record:
    • Label selected data: Pick a field, and then click a record to label it with the value in the selected field.
  • Legend: for some types of database (e.g. Koppen, earthquake focal mechanisms) get a legend for the map.
  • Highlight color:  color used to highlight records
  • Font
  • TIN contour
  • Animate field
  • Time Sequence (experimental, may not be working)

For viewsheds:

  • Plot fan locations
  • Label fans
  • Plot coverage circles
