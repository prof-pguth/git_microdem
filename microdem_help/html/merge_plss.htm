Merge PLSS ShapeFiles

To merge PLSS shape files, you must:

  • Have version 14.38.18 or better.
  • First display each of the PLSS files you will want to use:
    • Set the PLSS directory on the Options form, Directories tab
    • Display the PLSS file on the Map, Overlay, PLSS menu choice
  • Select "Merge shapefile" from the merge menu on the Data Manipulation Form.
  • Manually merge three sets of files:
    • quarter_84.shp:
    • section_84.shp
    • twnshp_84.shp
  • Name the new file, placing in a new directory under \mapdata\plss
  • Select files to merge. You must use the same set of files for each of the three merges. Hit cancel for the last of the files.

These directions supplement normal merging directions.

This has not been tested recently.
