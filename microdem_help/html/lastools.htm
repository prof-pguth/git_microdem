lastools

lastools is a suite of command line tools for lidar point clouds.  Since the the untimely end of Martin Isenburg, the program has seen massive changes to rationalize some of its idiosyncracies.  The progam has an open source part, which includes LASZIP and some other useful utilities, and a licensed part.

  • New rapidlasso site: https://rapidlasso.de/, with the download at  https://rapidlasso.de/product-overview/  and then https://rapidlasso.de/downloads/
  • The open source site: https://github.com/LAStools/LAStools
  • Old sites that might not survive the untimely end of Martin Isenburg
    • At UNC some have license for free use, restricted to personal or educational purposes with very limited exceptions.  LASZIP is there for a very easy download.  The site uses http, so it might be blocked by your IT department or your browser settings.
    • At RapidLasso which has a direct link to lastools download

You should unzip or install into X:\gis_software\LAStools, insuring you do not get X:\gis_software\LAStools\LAStools.  MICRODEM will attempt to find the program in this folder on any drive.  If it cannot find the program, it will ask you to find it.

You can also get their free viewer, laslook.

Help for lastools (and lidar in general):

  • The forum at https://groups.google.com/forum/#!forum/lastools has a lot of helpful discussions about lidar in general, and lastools in particular.
  • https://rapidlasso.com has many helpful blog posts

MICRODEM encapsulates lastools for:

  • Use lastools LASZIP to convert LAZ files to LAS.  Access with Compress/Decompress.  LASZIP has no restrictions on use.
  • resampling with las2las and lasinfo which has some limitations (will not do a datum shift)
  • Lidar point classification--restricted license for these programs may not allow you to use them. 
You must use the lastools GUI or command line interface to:
  • Use lastools  TXT2LAS to convert text file to LAS (txt2las -parse xyz -i lidar.txt).  This can be accessed on the Export menu choice of the Data Manipulation form.  If your XYZ file is more complicated, you will have to create a command line.
  • Use lastools  LAS2TXT to convert LAS to text (las2txt -sep comma  lidar.las).  You will have to add the header row in something like Notepad, which works better than Word for files around 75 MB in size.
