Two File Menu Choices

There may be two file menu choices available on the main menu at times. If this is the case, the left option gives the main program choices, while the right option gives you specific choices for the currently active window. This is a minor idiosyncrasy of the program and a direct consequence of the high cost of the software.

This will occur with the following kinds of windows:

  • Graph
  • Text editor

These are standard child windows shared among a number of programs. The correct windows procedure would be to customize the windows, and merge their File menu into that of the host program. This requires a significant effort, best devoted to improvement of the overall program.
