Merge All Images

Option on the Edit menu, if you have more than one image open.

This option will be slow, and possibly not very forgiving of incorrect usage.

This will produce an image, in geographic coordinates with an approximate data spacing specified by you in meters.

It will cycle through all open images, in the order they were opened.  For each pixel in the final image, if it is white, it will look in the image and find the nearest value and put in the final image.

 

This was added to merge back pieces of a large map that had to be scanned in sections.  Each was registered, and they were then merged.  It may or may not be applicable to other operations, although it will allow reprojection of imagery.
