Imagery Options Tab

This tab is on the options form, brought up from the Options menu selection.

  • For Mr Sid imagery
    • Largest image size in pixels for automatic extraction.
    • Ask to verify the extraction level lets you pick the level for extraction
  • Delete JP2 after import to Geotiff.  This is for Sentinel-2, which will save disk space (on the order of 1 GB per scene).  Do not select if using ACOLITE.
  • Average zoomed out imagery
  • Ignore 0 in histograms: use if the image has large black areas that are missing, as these will bias the histogram and provide poor image contrast.
  • Band computations for Landsat TM data
  • Multibands to grids
  • ACOLITE options
