The text can print using any font, style, and size that is available to Windows. Time scale units, but not lithology file units, can display either vertically or horizontally. The program will attempt to use a smaller point size to keep the text within bounds. Currently if you select a size that it too large it may run beyond its correct limits. To print vertically you must select a scalable font like a True Type font. The only part of the font that will be saved between sessions will be the point size; the font will always default to Times New Roman.

The width for time scale units will be generated automatically based on the font used; you can set the width (in pixels) for the lithologic columns.
