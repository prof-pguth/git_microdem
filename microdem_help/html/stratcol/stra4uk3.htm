STRATCOL (not updated recently) Create Thickness Diagram

To create a thickness diagram, go through the following steps:

  1. On the options menu, select the choices you want.
  2. On Rock Correlation, select Label file for any time or lithology name files that you want plotted on the left side of the diagram. You will first also have to pick a thickness .
  3. On Rock correlation, pick Column file to plot the lithology symbols.
  4. On Rock correlation, pick Fossil ranges to select the files.

5. You can save the completed diagram with the File, Save Diagram option.
