STRATCOL (not updated recently) Creating New Time Label File

You create a new time label file from the top down. For each label you enter the top and bottom boundaries in age or thickness; it is up to you to keep these separated if you will be drawing both kinds of diagrams. The top of one unit is the bottom of the unit above, and after the first unit the top should already be correct (if you change the top, it will also change the bottom of the unit below). You continue to add units until you select the Cancel button; the current information in the edit window will be ignored when you select Cancel.
