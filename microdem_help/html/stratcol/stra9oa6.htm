STRATCOL (not updated recently) Creating a Time Correlation Diagram

1. Clear any existing diagram.

2. Set the size of the diagram

3. Insure that the Options have been set correctly.

4. Select an existing Time scale from file, or create a new time scale. A time scale is always the left-most element of the diagram, after the optional vertical time axis.

5. Select additional time scales or columns. These can be inserted in any order you desire. Before selecting the file, insure that the Options have been set correctly if you want to change any display parameters.

6. After a stratigraphic column is displayed you can edit any of the displayed stratigraphic data columns.

7. You can create a new column and add it to the diagram.

8. You can save the graphic image, print the graphic image, or save new or revised data files at any point from the File menu.

9. When you close the program or clear a diagram, the program will give you the chance to save any new columns that you have created or any columns that you have modified.
