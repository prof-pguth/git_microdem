STRATCOL File Menu Options

  • Save image
  • Save files:  if you have added a new column or modified one of the existing columns, you can save them to the same or a different file. You will automatically be prompted to save them before you can close the program or start another diagram, so you do not have to select this option manually.
  • Open diagram
  • Save diagram: You can save the diagram at any point in standard Windows BMP format. You can then import the BMP file into any Windows program. A saved diagram is just a representation of the screen; it does not contain any of the raw stratigraphic data used to create the image.
  • Close Diagram: You can clear the screen, remove any data files from memory, and start over on a new diagram.
  • Convert length units
    • Time file
    • Fossil ranges
  • Exit Stratcol
