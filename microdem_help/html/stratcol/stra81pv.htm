STRATCOL (not updated recently) Time Files

Time files are used to label stratigraphic columns. The units can be either distance (meters) or time (Ma). While the program should handle all aspects of these files, you can manually edit them in any editor or word processor (insure that you keep them in plain ASCII format).

Time files are stored with the extension *.TM*. The intention is that you will store those for a single project with the same name and differing extensions, such as CAMBRAIN.TM1, CAMBRIAN.TM2, and so on.

On each line the first number is the base of the first labelling unit, the second number is the top of the first labelling, and then comes the label. The youngest unit should be at the top.

Sample time file

  • 505.0000 490.0000 Croixan
  • 520.0000 505.0000 Albertan
  • 530.0000 520.0000 Waucoban
  • 545.0000 530.0000 Placentian
