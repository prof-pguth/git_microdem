STRATCOL (not updated recently) Setting Diagram Size

 

There is a good chance that this does not work.

 

 

The white portion of the screen indicates the diagram size. If it is larger than your screen, you will be able to scroll through the diagram using scroll bars on the right side and bottom.

Use the Resize diagram command in the File menu. You will set the vertical and horizontal resolution in pixels. You can use the Help, About menu choice to see the current resolution of your printer (although this may not be accurate). You should set the printer resolution at 150-180 dpi (dots per inch) before printing; anything with higher resolution will only work if you select the lithologic patterns very carefully.

The size of the diagram determines the thinnest units that will show, and that will have a name printed. If units do not show, you must either make the diagram larger or combine thin units.
