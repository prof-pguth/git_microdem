STRATCOL (not updated recently) Stratigraphic Files

Stratigraphic files contain the information to draw columns scaled either to time or thickness.

The fields are:
1.  Short Name --up to 10 characters
2.  Thickness  in m
3.  Duration in Ma (older files like the example below will have thickness in feet, and will not plot properly as time correlation files)
4.  Three character lithology symbol
5.  Resistance (1 to 5, the most resistant)
6.  Long Name

If the base is anything other than conformable, a base label and drawing parameters will occur on a separate line between units.

The first line is optional, and contains the latitude and longitude an an optional location which can plot as the second line above the column.
The second line contains a title for the column.
