STRATCOL (not updated recently) Fossil Ranges

  1. You will first select a fossil file

  2. The window above allows customization of the display.

    The top line shows the number of fossil ranges in the file, which you cannot change within the program.

    The first edit box allows you to select the number of fossils to place, running from left to right, If the number of ranges in the file exceeds this number, the addition files will move back to the starting position and run left to right. This will only work if the ranges are short compared to the overall column. This number is based on how much you have already placed on the diagram, the size of the diagram, and the font size you have selected.

    The starting location in the number of pixels from the left side of the diagram at which the first ranges will plot. You should change this only with great care.

    The ending location shows where the fossil ranges in end, again in pixels.

3. You should place fossil groups that you want to appear together in separate files. This includes long ranging taxa that cannot have another species above or below them, and groups that you want to plot in proximity if you will have the ranges plot in multiple groups.
