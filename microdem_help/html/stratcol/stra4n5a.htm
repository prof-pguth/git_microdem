STRATCOL (not updated recently) Edit Stratigraphic Column

You can currently make the following edits to a lithologic column:

1. Change a stratigraphic unit's thickness, duration (in terms of top and bottom ages), resistance, short and long names, top and bottom boundary types, and lithology. To edit a unit, move the mouse cursor onto the unit and double click. This will bring up the unit edit window. In standard Windows practice, hitting the Enter key will close the window.

2. Move a contact. Move the mouse to the desired contact, depress the left mouse button, and move the contact to the desired location. Note that the Window status bar gives your position along the vertical axis. If you want more precise placement than you can get with the graphical move, use the unit edit window.

3. Insert a new unit. Move the mouse to the desired boundary between two units, and press the Insert key to get to the unit edit window. Insure that you correctly set the top and bottom ages; the program currently will not check the consistency of what you enter.

4. Delete a unit. Move the mouse within the unit to eliminate, and hit the delete key. You must confirm the deletion. In a time correlation diagram, the time duration of the deleted unit will be added to the unit below. If necessary you must edit that unit or the unit above to adjust the boundary.

You can either save the modified columns manually from the File menu, or the program will prompt you to save it when you clear the diagram or leave the program.
