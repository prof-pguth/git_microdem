STRATCOL (not updated recently) Adding a New Column

 1. To add a column, you will first have to enter the following:

a. Latitude and longitude.

b. Name for the column, which will be printed above the column on the diagram (keep this short).

c. The first (top) unit, using the unit edit window.

d. Additional units until you hit the cancel button in the unit edit window. The unit on which you hit cancel will not be entered into the column.

2. Once the column has been entered, you modify the column or add additional units the same way that you edit an existing column entered from a file. You can edit the new column at any time.

3. You can either save the column manually from the File menu, or the program will prompt you to save it when you clear the diagram or leave the program.
