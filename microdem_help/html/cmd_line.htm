Notes

  • Start from a command line, type the program name, a space, and the command line parameters:
    • microdem  ?act=slopeuncertain+in=c:\temp\test.tif
    • c:\microdem\microdem  ?act=slopeuncertain+in=c:\temp\test.tif
  • You can also include in a batch file
  • The astute will notice that the command lines mirror the format for passing parameters to a CGI program running over the web
  • Parameter list starts with "?"
  • Parameters separated with "&" (or "+" if using a batch file, because that environment does not like "&")
  • Parameter order is not important
  • Case is not important; the program will convert all to upper case before processing
  • Supply full names of files, which should not have spaces; there is an in= and out=
  • Input can be any grid format supported by MICRODEM, but more esoteric ones may require GDAL to be correctly configured so MICRODEM can find it.
