INI File

The INI file which contains the Program Options is stored in the user's Windows\application_data folder (something like C:\Users\pguth\AppData\Roaming\microdem\microdem.ini) .   There are over 2000 settings.

  • The subfolder under roaming there is based on the program name, so if you rename the EXE, there will be multiple directories, which allows running multiple versions of the progam, each with its own configuration. 
  • To navigate to this folder, you will have to show the normally hidden system folders. 
  • You can also save a copy of the INI file to another location, and look at it in any ASCII editor, on the Options form.
  • If there are multiple users on the computer, each will have their own INI file, in a separate subfolder under c:\Users\.
  • Old settings remain unless you explictly delete there.  This allows you to change the version of the program without renaming the EXE, and it will have the correct settings.
    • Settings no longer needed by the current version of the program
    • Settings that have been renamed so that all users get an improved default operation.

You can reset the settings on the Options form, or delete the file via Windows.  Changing the program mode also changes a number of settings.

MICRODEM does not make any registry settings.

If you change settings, either deliberately or accidentally, your choices can lead to unfortunate consequences which the program does not anticipate.  If you are having unexplained problems you can reset the values to the defaults using the "Restore Defaults" button on the bottom of the Options form..
