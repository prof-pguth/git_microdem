Graph Modification

When a graph is open, you can change it:

Using menu choices

  • File (Graph)
    • Separate histograms
    • Print
    • Save image
    • Copy to clipboard
    • Show toolbar
    • Legend
    • Close
  • Rescale
  • Analyze graph
  • Graph options

Right click, and use popup menu

  • Rescale graph
  • Line and point markers
  • Legend
  • Save image
  • Copy to clipboard
  • Paste from clipboard (to add a legend, for instance)
  • Abort current operation
