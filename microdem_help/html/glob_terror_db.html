Global Terrorism Database

  •  https://www.start.umd.edu/gtd/.  May have problems with your institutional firewall with partially renders the page but won't allow it to do anything.  You will have to deal with the fortress firewall defenders, or download at home.
    • After register, can download data as Excel file
    • has lat/long
    • In Excel
      • Change all commas to semicolons
      • Change all Unknown to nothing
    • Save as CSV
    • Import into DB
    • Filter out with just the region, dates, terror organization you need to get a workable file.
  • https://www.start.umd.edu/gtd/search/ This page works fine. (10/24/2019)
    • No export option found (Wikipedia says it can be done, and you can get it as a Geodatabase, which might be importable via GDAL, and should have coordinates), but you can cut and paste from the results into Excel
    • Filter the results to just what you want (country, group, time period).  If you cannot find bulk download, make the filter as restictive as possible.
    • Change to 100 items per page
    • Cut and paste into excel. Repeat to get all the records.
    • Change the field names to <= 10 characters
    • Save as CSV
    • Import into DB
    • There are no coordinates; use a join from another database. Depending on the country and the challenges of names in different languages and alphabets, the success my vary.
      • Get the NGA gazetteer
        • Join the terror DB to the gazetteer, using the CITY and NAME fields
        • Add lat/long from the join to the terror database
      • Get OpenStreetMap for the area
        • Join the terror DB to the PLACES file from OSM, using the CITY and NAME fields
        • Add lat/long from the join to the terror database
