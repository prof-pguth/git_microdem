Radar Principles

 

Physics

  • Rough surface reflects more back to the sensor
  • Dielectric constant
    • Most materials 3-8
    • Water approximately 80, so moisture greatly increases reflectivity
  • C band has some attenuation during heavy rain events

Color scaling from https://gis.stackexchange.com/questions/400726/creating-composite-rgb-images-from-sentinel-1-channels

  • Single date imagery affected by ground and atmospheric conditions
  • composite RGB (colour) image created using the VV channel for red, VH channel for green and the ratio |VV| / |VH| for blue.
  • RGB combination to map if you work with linear data:
    • red=VV
    • green=VH
    • blue=VV/VH
  • If you work with log-scaled data (in dB) it is slightly different because the laws of logarithms which turn ratios into subtraction:
    • red=VV
    • green=VH
    • blue=VV - VH
