PDAL

 

PDAL offers 2 methods for ground classification (SMRF and PMF) and it is significantly faster than MCC-LIDAR, already has 64-bit binaries ready for use and also supports compressed LAZ files. You can download and install it via OS4GEO here: http://download.osgeo.org/osgeo4w/osgeo4w-setup-x86_64.exe.   Its command line can be challenging. 

  • https://www.pdal.io/tutorial/las.html PDAL tutorial
  • https://www.pdal.io/tutorial/pcl_block_tutorial.html
  • https://www.pdal.io/apps/index.html#apps
  • https://www.pdal.io/workshop/exercises/analysis/ground/ground.html

 

commands

  • pdal translate c:/temp/aa.las -o c:/temp/aa_smrf.las smrf -v 4
  • pdal translate c:/temp/aa.las -o c:/temp/aa_pmf.las pmf -v 4 -f filters.reprojection --filters.reprojection.out_srs="EPSG:26918" --filters.reprojection.in_srs="EPSG:26918"

 

Consider:

Point cloud distances: https://www.pdal.io/apps/hausdorff.html

Outlier removal: https://www.pdal.io/tutorial/pcl_ground/index.html#pcl-ground
