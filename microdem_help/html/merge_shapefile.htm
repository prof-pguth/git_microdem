Merge Shapefile

  Data Manipulation Form,  Resample,   GDAL/OGR vectors,   Merge shapefiles

  • First select the name of the new file. It must not be one of the existing files you will merge.
  • Select the files to be merged. They must be the same type (line or area) and have the same fields in the dBase table. The easiest way to do this is the click on the first file, depress the shift key, and the click on the last file.  You can use Control, Click to pick selected files.  All the selected files will be highlighted, and you can hit enter.
  • Hit cancel when all files have been selected.

This option is not idiot proof. If you do not select compatible shape files, it will crash.

If you need to check the contents of a shape file, use the Analyze, File Metadata choice on the main program menu.

This requires GDAL.
