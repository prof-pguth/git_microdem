GIS Plots Monthly Filter

Selected from the PLOT button on the table display window.

Pick months: (only available for annual data sets)

  • Pick particular month with the radio button.
  • Cycle through months with the < and > buttons.
  • If present, the clear button removes the month filter and plots all the data.
  • Label month puts the month, and the number of readings, in the upper left corner of the map.
