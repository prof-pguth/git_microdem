DEM Holes

Option on Edit menu

These work on the entire DEM.  To work on a designated region, use Outline regions options.
  • Missing data to sea level
    • Entire grid
    • Island perimeter--starts at each edge, and replaces any missing data with 0.
  • Missing data to specified value
  • Fill holes from reference DEM: for all voids/holes in the DEM, the value from the reference DEM will be interpolated and filled into the original DEM.
    • Grid in file--does not open a map, and closes the reference grid after the hole filling
    • Open grid
  • Replace values with reference DEM: for all points in the DEM, the value from the reference DEM will be interpolated and filled into the original DEM. This will work if the reference DEM covers only a part of the original DEM, or has missing data in the regions you do not want replaced.
    • Grid in file--does not open a map, and closes the reference grid after the hole filling
  • Drop in the bucket from ASCII XYZ (lat/long/z) or 3D shapefile or DBF file.   Will put the values from the file in the closest location, and will use only the first value found.
  • Merge grids, take lowest value: the two grids must have the same extent and spacing.
  • Interpolate across holes (smooth): fill holes in the DEM which have missing data.
  • Replace with mode must be a byte grid.  Uses mode (or majority) filter to replace with the most common value in a 3x3 or 5x5 region.
  • Fill holes with external program
    • Whitebox
    • GDAL
