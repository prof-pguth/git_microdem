Modify Menu--Map Window

Choices will depend on the type of map, and current options.

  • Map Area
  • Set DEM with map: for images and vector maps. This DEM will be used for showing elevations as you roam over the map, or for DB computations that need a DEM.
  • Set second DEM on map
  • Display Parameter: set the parameter used for coloring the map.
  • Grid
  • Map marginalia
  • Map shading options
  • Missing data color
  • Elevation
  • Slope Categories
  • Contour
  • Reflectance
  • Colors from grid categories
  • Duplicate map window
  • Weapons fan/viewshed (DEM must be loaded)
  • Interpolate satellite colors
  • Recolor DRG
  • Merge color scene
  • No changes to map
  • Edit shape file group
  • Force redraw
  • Force redraw base map layer
  • Force redraw legends and scalebars
  • Force complete redraw--deletes all stored layers.  Should not be needed, but in case you do something and a layer should be redrawn but is not, this option is available.
  • Clear WMS overlay.
