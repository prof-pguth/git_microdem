DB Layer Definitions

  • Currently only works for point files with a string or numeric field.
    • If you plot a string field, this will be created automatically.
    • If you plot a numeric field, this will be created with the Layer button.
  • Save the layer on the Color code by DB field form, with this button.
Key fields in this database:
  • NAME, used for legends and labels.
  • PLOT, toggle Y/N, determines if it plots.
  • FILTER, the filter used on the database.  Note that in this case there are  "Cone" and "Cones".  You could combine them with "CONE*", where * is the wildcard used in the database.

You can double click on a record, and pick

  • Edit point symbol
  • Delete Record: permanent; use the PLOT toggle to temporarily change.
Edit from "Edit Layer symbology" from the "Plot" button menu on the database table display window.

Radio buttons on the top determine what happens when you click on a record.
