Color DB Recs

Color Database records
  • Insert
    • Insert Color Field
    • Insert line color/width fields for line shapefiles
    • Insert area color/symbology for area shapefiles
  • Color all records: all records in the display (if you have a filter active, some records in the database will not be displayed) will be colored with a selected color, line width, or fill pattern.
    • records: for point shapefiles
    • line segments for line shapefiles
    • fill pattern for area shapefiles
  • Color uncolored records: all records in the display with an empty color field (if you have a filter active, some records will not be displayed) will be colored with a selected color.
    • records: for point shapefiles
    • line segments for line shapefiles
    • fill pattern for area shapefiles
  • Color from RGB float fields
  • Color from RGB integer fields
  • Color based on field: select a field, and then the program will cycle through each unique value in that field and have you select a color.  As each color is selected the records matching the criteria will be colored.
  • Color based on joined table
  • Color from Koppen category
  • Fill pattern all records
  • Fill pattern all uncolored records
  • Assign unique colors (based on the sequence within the file number)
  • Recolor all fans when edits being done for viewsheds

You can also color individual data records, after you enable edits:

  • Double click on a record in the table, and then pick "Edit record".  There will be a color button at the top of the edit form if the database has a color field
  • Pick Graphical pick and edit database: and then double click on the record.  There will be a color button at the top of the edit form.

You can also click on the column title, and then color that column from the popup menu.
