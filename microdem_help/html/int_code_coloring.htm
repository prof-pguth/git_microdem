Coloring by Code for Grids

When the grid has values that are codes, you might not want to use the spectrum color methods because it may be hard to differentiate adjacent areas.  This problem will depend on how the areas are arranged.  With this option, changing the number of colors can help to show the records.

4 colors.  Note the large magenta area in the center, which is actually three records.
5 colors
6 colors
8 colors.  Note that the large magenta area now has two records merged.
