Create Tables

A table is an incredibly effective way of presenting and organizing information.

MICRODEM can create tables from databases in several ways.  You might first want to limit to a reasonably small number of records for this option, by Filtering the table.  You can also Hide fields you do not want.

From  dabatabase table form, Report:

  1.  HTML.  Both Excel and Word can open HTML files.
  2.  DBF Table.  You can open this DBF file in Excel, and then edit, and copy into the word processor.  You can also save it from Excel as a CSV file, and then re-import into MICRODEM.
  3.  Text (CSV) which you can open in Excel.
