Point Cloud Database Project

Start with "Point cloud" choice on program menu.

You can open multiple projects.  The windows will be labeled with the project name, and you will have multiple control forms. 

 

Controls for slices:

"Pick files" button allows you to toggle display of files on and off.

Three different slices open at the same time.  Select a new slice with the right radio group, and pick its orientation with the left one.
