Match other maps

Option on File menu for a map, or the Right click on map.
  • This coverage area and Map pixel size
  • This coverage area: all other maps will have the same size and area as this one.  If you mix types of map (for example, UTM and lat/long, "same" will be approximate.
  • Same elevation colors; other elevation maps will have the same color scale.
  • All 1:1 scale--the best size to show all data.
  • All same pixel size as this map
  • All same map size as this map (in pixels, if you want to combine the maps into a single figure)
  • All same display (slope, elevation, hillshade)
  • Create new grid for this map area
  • Mask other grids to match this one: missing data on this map will be marked missing on all other open maps
  • Outline coverage area on other maps: show the coverage area of this map on the other maps.  This can help provide a location map for a detailed map.
  • Outline other map coverage areas on this map: show the coverage area of the other maps on this map.
  • Force all redraw
