EPSG Codes

The International Assocciation of Oil and Gas Produceres (IOGP) has created the EPSG Geodetic Parameter Dataset to define coordinate reference systems and coordinate transformationsfor mapping.  Their numerical codes are used for many common data sets, including LAS and Geotiff.  The website allows you to look up the 4 digit code, and get a WKT representation.

  • Geodetic database: Geodetic Database (epsg.org) might need an account and password for maximus usage
  • Best starting location: https://spatialreference.org/ref/epsg/
  • Much less useful: https://epsg.org/home.html
  • https://epsg.io/
