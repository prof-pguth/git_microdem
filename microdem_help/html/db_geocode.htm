Geocode Addresses From Database

If there is an ADDRESS field in the database, the program can compute the LAT/LONG.  This ADDRESS can be a complete street address, or just the city and state.  There must be a map open from which the database was opened for this option to be available. This uses a web service from a server, with a paid API key.  It has not been tested recently.

  • Open map.
  • Database.  This will probably not plot yet since you do not have locations for the records.
