Optimal Viewshed Location

Chosen from:

  • Calculate Menu, Derivative Grid
  • Raster GIS menu, Create derivative grid

Computes a new grid with the coverage in percentage from each point using selected viewshed parameters.

Options:

  • "Resume computations" option you will be asked about refers to reopening a grid that you previously computed and either stopped partway through, computed only part of the DEM, or used a large thinning factor.
  • Select only a portion of the DEM.
  • Select a thinning factor to make it run faster (and later go back to fill in the missing points).
  • Resave the grid.

This can take a lot of time, so you should save it often, and then resume computations.
