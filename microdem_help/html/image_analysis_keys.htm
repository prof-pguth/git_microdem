Image Analysis Keys

We use a number of cues to interpret satellite imagery or air photos:

  • Shape (Pentagon)
  • Size (Barn versus storage shed; need to get scale first--running tracks an ideal feature)
  • Pattern (Orchard versus forest)
  • Tone/hue (color, depends on soil drainage, tree type)
  • Textures (smooth grass, rough tree tops)
  • Shadows (side views and lost detail)
  • Site (topographic or geographic location)
  • Association (help from what is around it, for instance on a pier you might expect loading cranes)
