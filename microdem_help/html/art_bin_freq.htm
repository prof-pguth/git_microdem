Arterfact Bin Frequency Table

Example BIN_NAME field:

  • Lower gundeck/Left/12-13
  • For this bin, the program must find the following points in the Key points Database:
    • Lower gundeck/Left/12
    • Lower gundeck/Left-mid/12
    • Lower gundeck/Left-mid/13
    • Lower gundeck/Left/13
  • Names can differ, but must match exactly between the artefacts and the key points.
  • Use the String Concatenate fields  operation on the database to create the field from the separate fields.
  • Use the option to remove all values with a "'99" for missing values in one of the fields.
  • Create a Frequency table for the bins, save it as a CSV, and convert to a DBF.
