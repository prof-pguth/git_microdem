MICRODEM.INI

MICRODEM.INI will have the program settings.  It is a text file, which you can inspect and carefully edit (although the program should handle this).  The program will create this file the first time you run it.

It is in:    "C:\Users\your_user_name\AppData\Roaming\microdem".  You can get the excact name in the debug file.
