Data Base Defaults File

If you set the option on the Database tab  of the options form to "Save data base status" a file with the same name as the database and a "DEF" extension will be created.  

This will store: 

  • Fields used for some statistics and graphical operations

  • Database linking information

  • Fields used for labels.

  • The hidden fields for display

  • Symbols used for display
