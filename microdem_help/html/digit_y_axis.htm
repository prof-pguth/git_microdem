Digitize on Y axis

 

Digitize y-axis
Pick button
Pick database (prompt will be for "x axis lines") with horizontal lines along the x-axis. The z values should be on the floor of the deck.  It must have the fields shown (X,Y, Z, and BEAM).  It should be for a single z level or deck.
Pick database (prompt will be for "names of y axis points to digitize") with the names of the points you want to digitize along the horizontal lines.  It must have the field SIDE.
  Adjust the slice so you have an x-y view, with the z value at the level you want to digitize, since the outline might change with z value.  You can adjust the z value, or the thickness of the slice.
Digitize.  The horizontal line being digitized will be in red.  The name of the location will be in the lower left corner of the program status bar.  When done, you will be asked to give the file a name.
The resulting data base.  The x and z values are from the horizontal line database, and the y values from your digitizing. Add a field showing the level being digitized, in this case the DECK.
  Final database with the key points, after filling the "DECK" field with the name of the deck.
