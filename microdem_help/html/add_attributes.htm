Adding Attributes to a Database

Use Excel. 

  • If the lat/long fields are in the DBF file, this is easy and probably the best way.
  • Must be done with extreme caution with line are area shapefiles.  You cannot add, delete, or rearrange records.
  • Excel will read a DBF file, but you must save it as CSV, and then use MICRODEM to import as a DBF.  Once you have the DBF, all edits in MICRODEM will be to that, and you must avoid reopening ghe CVS and over writing your edits (the program will warn you).

Use MICRODEM.

  • Put the database into edit mode.
  • Insert new fields.  You must get the data types correct for what you want,  the field length, and for real numbers the number of decimals.
  • You can fill a field for all records in the database, or all those that meet a filter, by Double clicking on field name and "Fill field".
  • You can edit a record by Double clicking on a record in data base and picking "Record edit".  You must "Post edits" when you close that window.
