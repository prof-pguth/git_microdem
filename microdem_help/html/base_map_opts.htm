Base Map Options

These are base maps for use in  Local map databases, with  Integrated database.

Vector Base Map.  Coastlines, boundaries, and major bodies of water show up. DEM Base Map.  This is from SRTM-30, with coastlines and rivers and major boundaries shown. Image Base Map.  This is from NASA's Blue Marble, with coastlines and rivers and major boundaries shown.

Set the base map type and properties on the Indexes tab of the options form.
