Lidar Resampling

Data Manipulation,  Resample  submenu.  MICRODEM will only deal with lidar tiles in either lat/long or UTM coordinates, on the WGS84/NAD83/ETRS89 datum.  The projection information must be in the LAS or LAZ file.

You can encounter two kinds of problems:

  1. Projection is not UTM or geographic (most likely SPCS).
  2. Projection is not included in the LAS file.

You can look at the metatata in the LAS file, using both MICRODEM and lasinfo, to see if metadata is present. File Metadata on the Analyze menu.

There should be separate metatadata files with the LAS or LAZ files, or as a separate download from the web site where you got the data.

Three options, each handles a different set of files.  Do not delete the original files until you know the conversion has been correctly handled.

  • LAS files (MD)  This will be the best choice if it does what you need.  Use it for SPCS, UK OS, and Irish grids.
  • LAStools.  This will be the best choice when you have to define the projection because it is not in the file.
  • libLAS
