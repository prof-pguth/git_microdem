Chirp Depths (currently probably disabled; note date of last revision)

Chirp display includes a dBase table.

One field contains depth computed from the chirp record.

Plot these depths on the map, using the Color code by DB field option from the Plot button on the Table view.

Ping versus depth graph, 2D graph from the Stats button on the Table view.

We have noticed anomalies:

  • Some depths are too small
  • Some depths are much too large

To get a good display, filter the database:

  • Use a Filter to restrict display by removing depths that are too extreme. This works when there are a large number of adjacent bad records.
  • Median filter to replace isolated bad values. This is on the Edit button menu.
