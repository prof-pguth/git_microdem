Resample Data

Data Manipulation  Menu choice.  Many of these require GDAL
  • Shapefile: reproject shapefile or add fields to its DBF file.
  • DataBase--includes shapefile DBFs.
  • GDAL rasters:
  • GDAL/OGR vectors:
  • Vertical datum shift, overwrite DEM/grid.  You must have set up the geoid grids needed, and need to save the original DEM if you do not want it transformed and then can work on a copy.
  • DEM formats--can go to Geotiff or MICRODEM format, copying all files in selected directory
    • ADF directory requires GDAL
    • BIL directory
    • IMG directory requires GDAL
    • DEM  directory : overwrites the original DEM files; the DEMs will be smaller and will load much faster for later use.
    • ASC files
    • TXT files
    • Convert to Geotiff (can use GEOTIFFS that have to be imported via GDAL, in which case the original files will be overwritten)
  • LAS lidar
  • Prep OSM/Natural Earth/GADM shapefile: insures the bounding box is present, and trims the length of excessively long string fields.  You select a folder, and all shapefiles under that directory will be processed.  The overall progress will be shown in the lower left corner of the screen.
