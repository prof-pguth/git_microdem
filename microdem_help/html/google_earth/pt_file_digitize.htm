Digitization in Google Earth

Digitize locations in Google Earth, and save as a KML file.  Only the "NAME" data and the locations will be imported.

You can import:

  • Point, Lines, or Areas in either KML or KMZ format with DNR GPS to convert to shapfiles, and then Open database in MICRODEM

  • Open database with MICRODEM and pick a KML point file with will create a shape file.  If you want a line or area, digitize as as sequential points, and then convert them to a line or area shapefile in MICRODEM.

If you need to add additional fields for attributes, which Google Earth does not allow.

  • Check the edit checkbox on the database table form once you have the file in MICRODEM
  • Edit database
  • Pick Insert Field, and define the type and size of the field.
  • If all records in the table have the same value for the field, "Edit, Fill field for all records" is a fast way to fill them in.
  • KML dates must be in the format YYYY-MM-DD, but MICRODEM can convert many formats.
