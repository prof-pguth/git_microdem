ArcGIS viewshed sensors

Option on Reports Menu of database table form. 

Renames the fields of the sensor table for the non-intuitive names uses in ArcGIS, and exports as a point shapefile format .

MICRODEM ArcGIS
SENSOR_UP OFFSETA
TARGET_UP OFFSETB
SENSOR_RNG RADIUS2
MIN_RNG RADIUS1
LEFT_AZ AZIMUTH1
RIGHT_AZ AZIMUTH2
MAX_VERT VERT1
MIN_VERT VERT2

Note the intuitive names (at least for MICRODEM), which you will remember as soon as you see the definitions once.

ArcGIS creates a composite viewshed, from all the sensor locations, with the number of sensors that see each pixel.  If you want to see individual fans, you must filter the database either before export, or in ArcGIS.
