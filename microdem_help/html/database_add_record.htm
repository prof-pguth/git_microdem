Data Base Add Record

Map showing line digitizing, and entry of data base fields.

You can enter values for the fields for the record just digitized.  This will be the top line in the table, with the * in the leftmost column.

If required, the ID field (a numeric record number) will be entered automatically; you must enter the others from the keyboard. If you have a numeric field, you will be unable to enter text.

The two buttons do the following:

  • + Rec & resume digitizing: post this record and returns to digitizing.
  • + Rec & close shapefile: post this record and close the file

The buttons will not be enabled until you have entered attributes for at least one field, and then either hit the enter key or clicked on another record.
