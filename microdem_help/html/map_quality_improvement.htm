Map Quality Improvment

  • Scientific figures.  General guidance which also applies to maps.
  • Grid/graticule.  Labelled graticule probably on at least one map, the first small scale one which shows the area.  Copy to clipboard with detailed option.  To save space for getting more visible detail on the remaining maps, they do not have to label the graticule.
  • Legends/marginalia, including scalebar
    • Scale bar should be on all large scale maps.  Because map scale changes on small scale maps, you cannot have a single scalebar there.
    • Move these to a corner where they do not cover important parts of the map
  • Legends
  • Large maps: maps should be large to show as much detail as possible
    • Use a large monitor, and have the map fill the screen.
    • Blow up so the map is bigger than the small laptop screen, and has scrollbars; when you Copy to clipboard you get a map that is bigger than the screen size
    • Zoom in (blow up; same coverage area on a larger scale)
    • Run the figures as close to the margins of your page as possible, and put pages in landscape mode if it helps.
  • Don't try to use color for two different things on a single map.  If you have color overlays, consider grayscale for the base map.
    • Grayscale reflectance map for a DEM
    • Single band satellite image
    • Grayscale and subdue any map
  • Match other maps
    • Show the location of one map on a smaller scale map.
    • Make all maps have the same size, so you can put them side by side on a figure for easy comparison
    • Have all maps use the same color scale and range for better comparison
  • Colors not using full spectrum and how to fix.  
    • DEM/grid: Z range
    • DB: Pick color palette and range of values to use for assigning colors
    • If you are trying to compare multiple maps, Match other maps so they all have the same color scale
  • Fonts: the bigger the map, the bigger the fonts need to be.  This is probably the most frequent criticism of figures for publication, so look carefully at all your fonts.
