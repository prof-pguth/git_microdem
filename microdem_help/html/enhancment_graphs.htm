Enhancement Graphs

Band and  contrast:  Set when changing the contrast enhancement on the imagery, which creates two graphs.

 

Histograms for the three bands in the display
Cutting off the tails.  Note that very few points had a DN over about 40,000, so a lot of colors were not used and the scene was dark  This setting will brighten the scene.
Histogram equalization.  Especially with multiband displays, this can lead to jarring displays but can reveal subtle difference by using a vast range of colors.  Note that most of the colors follow in a narrow  range of DNs (about 10000 to 20000), so those are really stretched out.
