Koppen Grid

  Use map toolbar icon and "Global monthly winds" option from the popup.  This requires c:\mapdata\database\koppen_grid.dbf which will download automatically if you do not have it.

This is actually a point database, but the values are on a ½º by ½º, for the time period 1976 to 2000. The data comes from  Rubel, F., and M. Kottek, 2010. 

File, legend, Koppen legend  will get you a legend with either just the categories or the categories and a descriptive name.
