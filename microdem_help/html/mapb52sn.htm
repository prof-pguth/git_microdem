Logical Contouring

Have: isolated data points

Want: continuous map or surface

logical contouring:

  • need all changes in slope as input data points (and only those points are necessary)
  • assume uniform slope between control points
  • linearly interpolate between known points

assumptions for logical contouring:

  • surface is single valued
  • surface is continuous (problem with faults for geologists if they contour the position of an underground surface)
  • surface autocorrelated over greater distance than control points
