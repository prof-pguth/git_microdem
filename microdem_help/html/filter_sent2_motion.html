Filter Sentinel-2 to Track Motion

  • Image analysis, Open band for raster analysis, and pick a single band
  • Subset the image, save, and reopen the saved image.  This can be very slow on the full image.
  • Raster GIS, Filter DEM/grid, Filter grid with .fil file.  EDGE.FIL is a good starting point
  • Make sure you are at 100% zoom (so you see all the points, since you might wind up with a single pixel wide crest)
  • Elevation/grid colors, and change to a color option (your pick)
  • z Range
    • Check clipped colors
    • Pick a color for the Lows
    • Go to specifed for the z Range
    • Repeat these steps until the peaks are all that remains above the low color
      • Move the min up
    • Remember the minimum
  • Edit, Mark as missing, Everything below the cutoff (the min from the last step)
  • Edit, Reclassify, All valid to single value.  If you pick 1, 2, 4, 8, 16, 32, 64, and 128 you can play some neat bit games
  • Save the grid
  • If you have several dates, say coded 1 and 2, you can add them, and then:
    • 1 just just the one date
    • 2 is the other date
    • 3 is both dates
    • 0 or missing data is neither data
  • If you have a reasonable number of points, you can File, Save DEM/grid, DBF current map area
