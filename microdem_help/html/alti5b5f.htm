TOPEX File Names

The files names on the computer for the TOPEX data: ALE-023.106:

  • The first three characters denote the location code, keyed to the  bathymetric data set selected and used to pick the map and other relevant data
  • The fourth character must be a “-“ or an “_”. The “-“ indicates the file is a compressed geophysical record, with certain fields removed. This cuts the size of the files to 1/3 the original size.
  • The three digits after the underscore indicate the altimeter cycle. TOPEX/Poseidon has a ten day repeat cycle. This gives the date of the pass.
  • The three digits after the period (the file extension) indicate the orbit or pass within the cycle.
