Koppen Color Coding

 

 

You can get a legend for the Koppen climate classification from the file menu of the map.  You can get either the symbols or the descriptive names.

 

 (Red is for A tropical climates, Yellow for B dry climates, Green for mesothermal C climates, Blue for microthermal D climates, and Purple for E polar climates).
