PLSS DBF Files: Required Fields

The PLSS DBF files require the following fields:

  • All 3 require: LAT_HI, LAT_LOW, LONG_HI, AND LONG_LOW
  • quarter_84 requires: LNDKEY, QSECTN (QQSECTION for the new files)
  • section_84.shp requires: LNDKEY, SECTN
  • twnshp_84.shp requires: LNDKEY, TOWN, TWNFRT, TWNDIR, RANGE, RNGFRT, RNGDIR, PRIMER, STATE

You may remove the other fields using a database program like MS Access or Excel. This will make the files smaller and save hard disk space.
