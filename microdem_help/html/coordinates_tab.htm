Coordinates (Coords) Tab

This tab is on the options form, brought up from the Options menu selection.

  • Dual elevations: elevations are always displayed in meters; you can also display them in feet.
  • Both datums when roaming--show the location on both the primary and secondary datums, set on the maps tab.
  • MGRS and Lat/Long when roaming--show both sets of coordinates.
  • Keyboard selection for target run
  • Satellite image coordinates: display the image coordinates
  • Verify graphical selection: allows precise selection of locations.
  • Verify, either graphically or by keyboard
  • Show roam on all maps--when you roam, you location can appear as a cross on all open maps.
