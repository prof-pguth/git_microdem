Mark Land Cover Categories as Missing

This can be slow.  A faster option is described under NLCD manipulation.

Menu choice on the Edit menu  available with a DEM available for the active map window. 

Pick NLCD-2001 for the area.  

  • Select terrain categories to use for the exclusion.  These will be deleted.
  • If you have a very large DEM, pick a zoom factor.  100%, for a 1:1 map view, will give the best results, but could lead to insufficient memory problems.
  • Pick a mask size, in pixels.  This is to insure that there is a buffer around features, and all of the features are masked.  This could be required due to rounding errors in the reprojection from the conical projection of NLCD to whatever projection your DEM uses, or if the DEM and NLCD had different resolutions.

The program will mask the selected NLCD categories on the map, and then remove them from the DEM.
