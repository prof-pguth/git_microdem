Flooding Analysis

You can do flooding analysis in two ways:

  1. Using Terrain category (on the Overlay menu), and setting the elevation range to color.  You can use IHS colors to see the terrain through the flooding.  This will flood areas behind levees or otherwise naturally below sea level, like Death Valley, and might have some anomalies if you want to model storm surge or sea level rise.
  2. Using Calculate, Drainage, Flood Basin. This will be the best way to  model storm surge or sea level rise.  Double click on a point, and then pick the elevation to flood up to.  The default will be the ground height at the starting location.
