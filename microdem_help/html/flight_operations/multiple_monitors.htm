Multiple Monitors

The program works well with multiple monitors. We have used  nVidia cards, generally with two monitors of identical resolution but have also worked with 4 monitors (1920 by 1200 each).  If you want to quickly expand MICRODEM to fill all 2 or 4 screens, use the "Span all monitors" command on the Window Menu.

We generally run MICRODEM only on a single monitor (either 4K or 1920x1200), and use a second monitor for other options (Google Earth, web browsing, Microsoft Office, the Delphi compiler for development).

The solution might not work (well)  if you have an unusual position for monitor 1 (not the upper left), or different monitor sizes.  In that case do your own resizing.
