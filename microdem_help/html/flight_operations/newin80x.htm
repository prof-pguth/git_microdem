What's New in MICRODEM Version 8

Version 8.0 first appeared in May 2004 and was retired in September 2005.

General new options or improvements:

3D viewing:

  •   VRML Option to output directly to WalkAbout and Immersaview
  • Crest separators in perspective view (general tab).
  • Draw parallel profiles to a LOS/topo profile

Intervisibility

  • Major overhaul of many of the intervisibility operations, including new options for computing coverage of a specific target region.
  • Experimental Option to set a minimum blocking distance for LOS and viewsheds.
  • Optimal location in box and Optimal location to cover target
  • Experimented with new Intervisibility options.  These are not yet in the program, but will be added as we determine the best way to use them.

DEM Formats

  • New DEM Format.
  • Save DEMs in 3D formats (OBJ and DXF)
  • Save DEMs as PGM files (no cartographic information, incorrect scaling if lat/long DEM).
  • Read CDED from zip file with two halves of the DEM, and index in the new integrated database.

GIS Databases

  • Label TIGER features
  • Database indexes use full path names. They used to use relative paths, to allow the entire data directory structure to be copied to a CD or other portable storage device. The new structure allows data to be spread across multiple drives.
  • Subset shape files.
  • Create a climate data base for Koppen climate analysis.
  • Use a DEM or Image for the index map in the USGS data or NGA data databases.
  • Enhanced edit of line/area shape files.
  • Option to set width, color, and plot or not plot of world outline of coasts, rivers, and political boundaries.
  • Stratcol GIS
  • County outlines can use TIGER or National Atlas files.

Map Display

  • Create DEM map covering the exact same area as an image. It will be interpolated with the same area and spacing as the image.  (May not work at the moment)
  • Redraw all open maps to cover the same area and at the same scale as a map (on the map's File menu).
  • Redraw all open elevation maps using the same color scale (on the map's File menu).
  • Option to used defined color palettes.
  • Open a shape file and display it in a blank vector map.  On the File menu.

Others

  • Improved Terraserver image preview.
  • Drainage basin operations using TARDEM.
  • Some options are now threaded to use a multi-processor computer. (May not work at the moment)
  • Some options can now run over a grid network, with computations distributed to client computers. (May not work at the moment)
  • Sunlight maps and computations of sunlight above the horizon/blocked by topography.
  • Option to open a text window to display the results of analyses.
  • Option to zip file on the Uncompress menu of the Data Manipulation form
  • Improved the use of color schemes for elevation maps in the palette dBase table.
  • Masking grids for statistical operations
  • Thin DEM option moved to the file menu when a map window is active.
  • Option to fill holes in NED extending into Mexico with SRTM.
  • Program will open a Zipped TIFF image, as long as it is a single file (all bands in the same file, with the registration included [i.e. no world files]), and the name of the TIFF file is the same as the ZIP archive.

Disabled some options.
