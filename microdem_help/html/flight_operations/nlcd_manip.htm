NLCD Manipulation

Many operations with NLCD will work best if you have NLCD and a DEM that match exactly.

  • Download the 1" NED and NLCD (1992, 2001, or both) from the USGS Seamless server.  If you have the 1/3" NED, you can Thin it by a factor of 3 on the Raster GIS menu.  The 1" spacing is most appropriate for the 30 m NLCD.
  • New DEM to open the NED.
  • Create derivative grid on the Raster GIS menu.  This will create a grid exactly matching the NED.  Save it with a file name and location where you can find it.
    • NLCD-2006 grid
    • NLCD-2001 grid
    • NLCD-1992 grid

Mask Grid with NLCD:

  • New DEM to open the NED and the NLCD.
  • On the NLCD, Edit, Mark as Missing, Single Value to remove a category.  You can remove additional categories as well.
  • On the NED, Raster GIS, Mask Grid, Remove missing points in mask, and select the NLCD as the mask.
