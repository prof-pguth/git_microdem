CLIWOC data base obtained from the web was modified for use in MICRODEM as follows:

  • Created DEC_YEAR field.
  • Removed records without LAT, LONG
  • Create SHIP field, by concatenating the country and ID fields.
  • Fixed LONG to be -180 to 180
  • Computed new file
    • with speed and heading of the ship,
    • at the center of  the time interval between two readings
    • at the midpoint of the line connecting two readings
    • removed all speed over 25 knots (this is generally over a day)
