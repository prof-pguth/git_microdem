Version 5.1

TBII Version 5.1 officially appeared Dec 2001 with:

  1. Weapons fans enhancements (IHS colors, complete coloring, and improved edits).
  2. Maximum map size has been greatly increased
  3. Load from local NIMA data base by clicking on any map.
  4. Option to speed up shape file GIS operations by saving the bounding box for each record in the DBF file.
  5. Improved shape file conversion from VPF.
  6. Associate a DEM with imagery after it has been loaded.
  7. Import e00 gridded DEMs (Arc/Info format)
  8. ID of data base records by clicking on them.
  9. Gazetteer location of locations, and data loading by location name.
  10. Local USGS data base.
  11. Adjust position of map marginalia.
  12. Printing to large format HP plotters using HP RTL is much faster and no longer memory limited.
  13. Improvements to the ambush movie.
  14. 3D merge of DEM reflectance with scanned maps.
  15. Display of DTSSOP products.

Versions 5.11 and 5.12 made minor changes (primarily sanitizing help file after 9/11).
