MICRODEM DataBase Access

Microdem requires access to database files, specifically dBase DBF files which are the de facto standard of the GIS world because of their use in ESRI Shapefiles.  In the past this has been with the Borland Database Engine (BDE), which is ancient and will not work with Windows 64 or the handheld devices. 

The debug log will list the version of database access used, which might be:

  • BDE (only for Windows 32 version of the program, which is no longer being distributed). 
  • tDBF, a free package   (http://tdbf.sourceforge.net/,  http://sourceforge.net/p/tdbf/discussion/)
  • sqlite
