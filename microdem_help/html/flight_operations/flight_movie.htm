Flight Path Movie and Clearance Computation

New DEM: Open a DEM covering the area.

New image to display or drape the path on a satellite image.

Database should be closed for this operation 

Pick "File, Replay Flight Route" menu choice.  If you want to drape the satellite image, insure that the image is the active map window.  

If you thin the dataset for faster operation (Use Thin option on Edit button of database table form) it can display faster for preliminary assessment of the results.

Set movie parameters:

  • Viewport depth on Viewport tab may require increasing the value, since the default may be too small.

  • Insure cross track and along track profile settings are correct on Flight options tab.

Movie can have up to 4 images:
  1. View ahead from the cockpit.
  2. Map showing the route and the area covered in the view.  You should set the map area, and position or turn off the scale bar, elevation color scale, and map grid before making the movie.
  3. Cross track view at current location.  You set the horizontal and vertical scale for this on the Flight options tab before running the movie.
  4. Along track profile.  Set the parameters on the Flight options tab before running movie.
  • The green line is ground track below the aircraft
  • The dark gray line is the flight path.
  • The light gray points are elevations to either side of the aircraft.  In this case, there are 10 profiles, five on each side, 250 meters apart.  This shows the elevations within a band 2.5 km wide centered on aircraft.  The highest point in this band will be placed in the CLEAR_R and CLEAR_L fields of the database.
