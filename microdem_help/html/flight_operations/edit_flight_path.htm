Edit Flight Path File

Create a flight path file.

New DEM: Open a DEM covering the area.

Database Open the CSV file on the Map window toolbar.  MICRODEM will convert the CSV file to a DBF file, and from this point on you want to use the DBF file.

Adjust Flight Path Database File
If you do not have Z field (elevation of the ground directly below the aircraft) populated:
  • Check the EDIT box on the database table display window.  The EDIT button will be enabled.
  • Hit the Edit button, and then pick the "Add elevation from DEM" menu choice.  The ground elevations (in meters) below the points on the flight path will be filled in the Z field.

If you do not have the DISTANCE field (this is the distance from the airfield, used to plot the glide path) populated:

  • Insure that you have the LAT, LONG of the airfield.
  • Insure the EDIT checkbox is checked.
  • Hit the Edit button, and then pick the "Distance from point" menu choice.  Enter the coordinates of the airfield.  The DISTANCE field (in kilometers) will be populated with the distance to the airfield.  If your flight path makes a circle, this is not the best solution since you will be at the same distance from the airfield repeatedly.
  • Alternatively, you can also add the distance with the "Add azimuth to travel path" option discussed below.  Since the azimuths from this option will be less accurate than using the World Magnetic model,   you can set the azimuth option just to get the distance, and then get the azimuth from the magnetic model.

If you do not have the AZIMUTH field (must be in terms of true north) populated.   There will also be problems if the input file repeats the same LAT, LONG pair and MICRODEM attempts to compute the AZIMUTH.

  • Insure the EDIT checkbox is checked.
  • Then you have two choices:
    • If you have compass headings in another field, you can use the World Magnetic model to turn them into true headings.  Hit the Edit  button, and then pick the "Get azimuth from magnetic heading" menu choice.  This would be the preferred option.
    • Hit the Edit  button, and then pick the "Add azimuth to travel path" menu choice.   The AZIMUTH field (in degrees) will be populated, based on the azimuth between adjacent pairs of points in the data base.  Due to rounding errors which can cause a wobble, it would be better to get the azimuth with the other data.  This option can also put in the cumulative distance along the path.
