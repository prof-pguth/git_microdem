Multiple Grid Creation

Create with Raster GIS, Create derivative grid, Multiple parameters

Many of the groups compute multiple parameters together, so creating several grids will not significantly increase the time (but will increase the memory requirements):

  • Relief1
  • Relief 2
  • Terrain Fabric
  • Slope/aspect
  • Curvature--use these with caution (maybe even extreme caution).  External tools has other options for creating curvature maps.
  • Openness

The moments computation will increase the time required depending on the number of parameters included.  It will create up to our grids for mean, standard deviation, skewness, and kurtosis for each parameter.  Note the caveats for Curvature.

Several grids require a region size, and the ability to thin the resulting grids.  Thinning speeds up the processing, and is a good option for parameters like relief which change slowly.

Create all maps on the lower left creates all the checked maps.
