Grid Parameters for New DEMs

  • Creates blank new grid covering map area
  • On the point cloud control form  for DEM creation from point clouds
  • Subsample DEM by averaging
  • Reinterpolate DEM on File, Save DEM/grid, where it is immediately create

The grid can be created with:

  • UTM: pick the grid size in m.  You can force the UTM zone if you do not want the default, which should only be done when you are at the edge of a zone and want to merge with data from the next UTM zone.
  • Lat/long: you can pick different values the lat (y) and long (x) spacing
  • Other WKT: you pick the WKT projection file, and the grid size in projected meters.  WKT grids can currently only be saved in MICRODEM format.

Auto fill holes, and and the max gap to fill in grid cells when the grid is created when a grid is created from lidar or resampling an existing grid. Lidar will commonly have small data voids, and hole filling can greatly improve the visual quality.

PixelIsPoint/PixelIsArea

Close create the grid; this is not needed when you will use the 1 pass or multi pass options
