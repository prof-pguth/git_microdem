Modify FFT

The fft button on the power spectrum graph lets you modify the fft.

 

Set options for the fft. Most of the defaults should be correct; you might want to change:
  • Period or frequency
  • The size of the bins (by default they are 1)
  • The units for the time bins.

 
The same graph, with the period set to 6 sec, and shown with both frequency and period. You can adjust the range of the axes by right clicking on the graph.
