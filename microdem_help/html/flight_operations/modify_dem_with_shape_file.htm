Modify DEM with Shape File

  • Open DEM
  • 1:1 view only those points on the screen will be modified, so the pixel size must be less than the data spacing.
  • Open shapefile database:
  • Filter database so the points you want are colored, and displayed in a color not otherwise used on the map.  If you pick a reflectance map, bright red is a good choice.

You now have two choices:

  • "Edit, Mark as missing, Selected color" will make all points in the specified color missing data.  This option will work with multiple records simultaneously.
  • "Edit, Flatten lake" will ask you for an elevation, and then put all points in that color at that elevation.  This option should probably only be used for a single record in the shape file at a time.
