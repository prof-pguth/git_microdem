What Was New in Version 5?

The program underwent a number of significant changes in the summer and fall of 2000 with Release 5.0 of TerrbaBase II (version 12.60 of MICRODEM).

  • Display and query VPF files
  • Military icons and overlay on maps
  • Create shapefiles
  • Local  data base to handle CIB, CADRG, and DTED on the hard disk
  • Improved speed in data base operations, using the Borland Database Engine (BDE)
  • Save and restore of projects, with multiple data sets and overlays

Version 5.01 fixed minor problems with Geosym display of VPF, and display of PLGR GPS waypoints.

Version 5.02 improved OpenGL saving; allows printing OpenGL; fixed problems with PowerPoint export; fixed problem with three letter datum codes like BOO; improved finding image in ERDAS Imagine IMG files.

Version 5.03 made additional minor bug fixes.
