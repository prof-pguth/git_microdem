Nearest Point on Line/Perimeter

Accessed from the Edit button Field geometry on the database table display window.  Two options:

  • Distance to line/polygon, just gets the distance to the nearest point on the line/polygon
  • Nearest point on line/perimeter, also records the LAT_2_LINE and LON_2_LINE for the coordinates of the closest point on the line.

Input:

  1. Start this option on a point database, or a polygon shapefile with a single record with an associated point (like the centroid).  You might want to restrict to the points that are inside or outside the polygon.  For this you can Edit button Field geometry and Point in polygon/area.  This will add a field, with the name from the area DB, and you can filter based on that.
  2. Then pick Line/area shapefile to which the distance on the line or perimeter will be computed for every record in the other shapfile
    • Dissolve lines can make the operation run faster
    • Filter the database so it just covers the area of the map reduces the number of computations
    • Subset the database so it just covers the area of interest reduces the number of computations
    • The optimal answer is a database with a single record that just covers the area of interest

Output

  • New field with the distance in meters to the line or polygon
  • The new field will be DIST_LINE which you can rename
