Varieties of North

The program uses three types of north:

  • Geographic or true north.  This runs along lines of longitude, and will be the vertical on maps aligned along geographic north.  This includes all data sets from NGA, and USGS data sets like the SRTM, NED, or the 250K DEM.  These data sets will have crooked UTM grid lines.  These lines should converge towards the poles.
  • Grid or UTM north.  This runs along the UTM grid, and rotates away from true north as you get away from the central meridian in a 6° UTM zone.  This will be vertical on all maps using a UTM projection, which includes most of the native USGS data sets, especially the 24K DEMs, DOQQs, and DRGs.
  • Magnetic north.  The direction a compass will point. This can be converted using the World Magnetic Model.

When preparing a map for printing, you can include a declination diagram in the map legend.

The "Info" main menu choice, or the Information button on the toolbar button on a map window, will calculate the declinations at the center of the map.
