New in MICRODEM Version 10

Version 10.0 first appeared in June 2006, compiled under Delphi 2006.  It was replaced with Version 12 starting in June 2009.  Because of the difficulty in converting to Delphi 2009, the posted releases remained version 10 for some time.

Version number has now been de-emphasized on the splash screen, in favor of the build number, which is easier to track because it changes daily instead of annually.

Program History 

Program installation:

  • (12/31/2007) Program installs into c:\microdem
  • (6/24/2008) Program defaults changed.  Programs and directories needed to run are stored in a new database.
  • (12/30/2008)  Improved program update

Map appearance and options:

  • Revised selection of legend elements, and added option to do that when right clicking on a map.
  • Option to subsample map to a specific geographic quadrangle.
  • Improved options for Map Marginalia on maps; right click on the map, and pick the legends/marginalia choice from the pop up menu.
  • Option to display a native grid for DEMs such as UK OS grid or SPCS DEMs.
  • Use percentiles to set extreme values for elevation map coloring.
  • (11/30/2007) Option to display elevations from a second grid while roaming (on the Calculate menu), and to overlay contours from a second grid (on the Overlay menu, say to put elevation contours on a slope map).
  • (12/28/2007) Further changes and enhancements  to manage overlays. and removal of older options.
  • (2/18/2008) Map image saving as a Geotiff can save as a grayscale or color image, with a savings of 1/3 the file size.
  • (2/28/2008) Should be major speed up of most map redraws with saving of map layers as temporary bitmap files.   This is done by saving each layer as a bitmap, and only redrawing when necessary.  If the program is fooled, there is an option on the bottom of the modify menu to force a complete redraw of all layers.
  • (3/3/2008) Scrollbars for map and satellite image windows expanded so you can now drag the map and scroll through them.
  • (3/10/2008) Scrolling over the DEM will show the elevations in all open DEMs in the status bar.
  • (5/12/2008) Grid options changed--if you want to label the grid or graticule, you must use the Prepare printer image option.
  • (8/11/2008) Improved options to display Tissot indicatrix
  • (4/26/2009) Option to put simple grid on printed map.

Data download:

  • (3/26/2008) LOS has always been done at screen resolution.  For very long ranges, or very high resolution DEMs, this could cause problems, which would be most severe if you were on the back side of a hill.  Hills could be missed (especially very close to the observer), and you could get a very skewed interpretation.  This has been fixed.   In hindsight this was a major oopsie, but with the computer hardware available when the program started this would never have been a problem, and it just never got considered as gigabytes of RAM became cheap.
  • (3/27/2008) LOS option, when right clicking on the LOS profile, to get a blowup map of the sensor/observer location.  This will be a slope map, color coded by trafficability categories, with contours overlaid.  The observer will be a small red square in the center of the view.
