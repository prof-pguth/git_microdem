3D Viewer

There is a new 3D viewer being introduced.

  • Uses FMXutils from Eric Grange.
  • Option to choose it on the Views tab of the Options form.
  • Currently lacking the save options, so if you need those, use the old version

Used for

  • LIDAR Point clouds, then OpenGL button.
  • OpenGL 3D view button on map toolbar to view single entire map
    • If it is an image, it must have an associated DEM, set on the modify menu.  If you open the DEM first, and it covers the image, it will automatically be associated
    • If it is a DEM/grid, it will use the elevations from the grid/DEM
    • If you want to use one DEM for the topgraphy, and another for the colors ing (say a change grid on a DSM), use OpenGL, drape on another DEM. from the Overlay menu
  • 3D viewer with multiple maps
  • From point cloud slices
  • Chirp subbottom display, Fence display.
  • Quality notes:
    • If you only want part of the map (or have a large volume of data), use Subset & zoom first.
    • To see the 3D, you need a relatively small area, and would like it to have steep topography; over a large area, the 3D effect will not be as impressive.  Cranking up the vertical exaggeration will only be partly successful.
    • For the best detail, you want the map to be as large as possible.  Zoom in.  This is particularly important on a small screen.
    • Consider if you want to turn off grids, scalebars, and legends before picking this option

Key display options:

  • Depressing left mouse button allows you to drag the view.  If this is too hard, use the large X, Y, Z rotation buttons.
  • Zoom in/out of the view with mouse scrollbar.
  • Mouse viewports buttons let you move the viewport up/down (z axis), left/right, and in/ out.
  • Change Vertical exagerration. If you have a small area with significant relief, you should not have to use vertical exaggeration.  If the default view at vertical exagerration 1 is insufficient, use the VE buttons in the upper right.  Alternatively close and pick a smaller area.

If the window is covered by other windows, you can bring it to the front using  a choice on the Window menu.


Round controls let you rotate about the x, y, and z axes.  You should be careful with the x and y axes not to overturn the view  You can also drag the view with the mouse, and zoom in/out with the mouse scrollbar.

Reset returns to the default orientation, if you turn the view upside down and cannot get it back.

Move viewport lets you move the view should it become uncentered.

  • You are most likely need to use the Z++ or Z-- to move along the vertical axis.

VE+ and VE- change vertical exaggeration.

Control multiple clouds

  • If you have multiple views of the same data with different coloring, you can select the one you want to show.  Because they are all at the same locations, you cannot display more than one at the same time.
  • If you export by category, or have multiple point clouds,  and the points in the different layers are not at the same point, you can turn layers on and off using checkboxes, adn see several layers at the same time.

Capture options

  • Z rot movie: sequence moving around the z (vertical) axis
  • Clipboard: copies the view to the clipboard.
  • Mult view movie: sequence shows each of the images in sequence.
