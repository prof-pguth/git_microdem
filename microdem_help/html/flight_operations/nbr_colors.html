Fires: NBR and dNBR

The program has two defined palette display for dNBR, from the Sentinel hub.

dNBR Steps in MICRODEM

  • Image analysis Create new band NBR.  Create pre and post fire NBR
  • Raster GIS  Multiple grid arithmetic Difference between two grids   PreFire - PostFire
  • Display Parameter, Elevation map, Fixed color palette, and pick one of the dNBR palettes
  • Right click popup menu to get legend, with percentages in each category
