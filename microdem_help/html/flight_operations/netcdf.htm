netCDF Bathymetry and Other DEMs

The program will  read netCDF bathymetry (and other files) using FWTools (GDAL).

Tested multibeam seafloor bathymetry is from

  • http://dusk.geo.orst.edu/tonga/mgr/
  • http://ocean-ridge.ldeo.columbia.edu/general/html/home.html
    • "masked" data shows the ship tracks and includes voids.
    • "unmasked" data fill in the holes with very coarse data.

Depths may need to be multiplied by -1 to work correctly.
