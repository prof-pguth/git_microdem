New in MICRODEM Version 7.0x

Version 7.0 was first fielded as an alpha version in April 2003. 

Version 7.01 appeared in July 2003, and interim minor incremental builds were made regularly. It was always regarded as an alpha version.

General new options or improvements:

  • TIGER button on main program toolbar loads vector map of a single county (must be enabled on the options menu form, Menu choices tab).
  • New format for TIGER database.
  • Option to download Garmin GPS waypoints, and import Magellan waypoints.
  • Changed where some of the standard US databases need to go (Counties and USGS 24K quads), and automatic drawing of the counties, states, and Fenneman provinces.
  • Previews of map color schemes (elevation, slope, reflectance) when picking
  • Interactive selection of reflectance parameters.
  • Should be able to load imagery first, and then the DEM, and the DEM will be associated with the imagery when you select an option that requires a DEM.
  • OpenGL remembers some settings in program defaults, has slider bars for interactive light adjustment, and fixed some lighting problems that correct sun orientation was not set. Improved OpenGL 3D controls.
  • GMT used to grid irregular XYZ points.
  • Improved entry of slope categories.
  • Downhill drainage direction overlays and other improvements to drainage basin delineation.
  • Improvements to Trend surface analysis.
  • Volume computations with difference grids.
  • Floating point DEMs
  •   PolarView extension, to use MICRODEM with Geowall.
  • Improved results working with multiple monitors.

New data import options:

  • Native read of NED BIL and gridfloat files; Globe, GTOPO30, and SRTM-30.
  • Display Global Land Coverage 2000 files.
  • Display IBCAO (International Bathymetric Chart Arctic Ocean).
  • Read netCDF/GMT bathymetry.
  • ASCII XYZ import can be comma, tab, or space delimited, and uses a dd-mm-ss.sssN type of format.
  • Read Mars DEMs.
  • Break SRTM-30 into 1 degree tiles, and index them.
  • Fill holes in SRTM-3 with SRTM-30.
  • Use the new (May 2004) PLSS data files.

Changes for large DEMs:

  • Predicted bathymetry can now load the entire data set.
  • No longer asks about thinning or subsetting DTED2 but loads automatically.

Changes for Imagery:

  • Previews of satellite contrast and band selection.
  • Support for SPCS imagery (state plane coordinate system)
