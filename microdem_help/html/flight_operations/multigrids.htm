Multigrids

This involves a series of grids stored in a single directory, selected by picking the directory.

It is used for:

  • Landsat TM imagery, so that the same code used for operations on DEMs can be used for numerical analysis.
  • Hyperpectral imagery
  • Multi-resolution DEMs, where as series of DEMs covers the save region at different spatial resolutions.
