NASA/USGS DEM Downloads

  • Open MICRODEM program
  • Open any map
  • Subset & zoom the map so it covers the area you want for the DEMs.  Do not be greedy; this is not a good option for a large area
  • File, import
  • This will open a new tab on your browser for each file
  • Files will be downloaded to your downloads directory
  • You will have to unzip before using
  • DEMs available
    • ASTER GDEM to match this map
    • NASA DEM to match this map
    • SRTM: ought to work but we cannot get it to work, and it would be complicated because there are 6 folders with data. https://e4ftl01.cr.usgs.gov/MEASURES/SRTMGL1.003/2000.02.11/
