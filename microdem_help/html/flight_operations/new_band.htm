Create difference index, band ratio, new band

Submenu from Image Analysis  on map window menu.

With Landsat, you can choose whether to use the DN or the radiance or reflectances.

 
  • Filter
  • Add (sum) bands
  • Contrast stretch
  • RGB Grayscale
  • Normalized satellite indexes with Landsat TM or similar imagery.
    • NDVI (vegetation)
    • GDVI (Vegetation)
    • NDSI (soil)
    • NDSI (snow)
    • NDWI (water)
    • NBR Normalized burn index
    • Normalized difference (pick bands)
  • VARI--Visible Atmospherically Resistant Index
  • Band ratio
  • Landsat VIS, NIR, SWIR
    • TOA radiance
    • TOA reflectance
    • TOA reflectance with sun position correction
  • Landsat TIR
    • Brightness temperature
    • Land surface temperatue--you must assume an emissivity, and must assume it is valid for the entire scene, or mask out parts of the scene where it is not applicable (say if you want to get sea surface temperature)
    • TOA radiance

An option on the Imagery  tab of the  options form determines if the new band is stored as bytes or 4-byte floats.

These will create a grid in the temp directory; you can save them elsewhere if you want to access the grids after closing the program.

 

Several other options (Principle components, Unsupervised classification) also create a new band.
