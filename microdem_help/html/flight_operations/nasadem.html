NASADEM

This is a revision of SRTM, filling in holes with the best available other data. 

NASA registration and password required

Download Integer version in the HGT format, and the ZIP file contains two additional files which MICRODEM does not use.

  • Hole filled
  • https://search.earthdata.nasa.gov/search?q=C1546314043-LPDAAC_ECS
  • For small areas to cover an open map, File, Import, NASA DEM to match this map

LP DAAC - NASADEM_SHHP (usgs.gov)

  • Not hole filled
  • NASADEM SHHP in  HGTS format (same as HGT but with floating point elevations). 
  • Very large directory with the elevation and error estimate files--have patience while it loads
