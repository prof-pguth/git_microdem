NBCD Data

Vegetation Inventory available at National Biomass and Carbon Dataset 2000 (NBCD 2000) download

Grid with average basal area vegetation weighted height. 

First uncompress (gzip, then tar).

These are 16 bit Geotiffs, in decimeters, and can be opened either as a DEM (preferred) or an image.

New DEM  NBCD as a DEM

  • Remove 0 values (Mark as missing)
  • Multiply by 0.1 so the values are in meters instead of decimeters (Single grid arithmetic).
  • Display with elevation colors (change display parameter, to Elevation map).

New image: NBCD as an image

  • White will be the highest vegetation, and black the lowest.
