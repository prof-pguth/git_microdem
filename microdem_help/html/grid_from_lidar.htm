Gridding Lidar Point Cloud Data

Principles

  • Open point cloud
  • If you have multiple clouds, you can
    • Merge them into a single grid (check all of them as active point clouds)
    • Make seperate grids for each  (check only one at a time as active point clouds)
  • Select the settings on the Create grids tab that will control the generated grid
    • Insure you have a UTM grid unless you have specific reasons to select geographic
    • Pick the grid size.  With most modern lidar 1 m is a good starting point for the grid resolution
    • Auto fill holes.  If there are holes in the data, you can interpolate across them. 
    • Algorithm for DTM point selection
  • Create a point density grid to see the distribution of points.  The program reports the average density on the File list tab, but it includes the entire rectangular region that encompasses the data, and that might have significant voids that will drive down the average point density.
    • The best option is probably first returns, which will give you only one point per pulse.  The other options make sense in certain specific circumstances.
    • To see the voids, consider Marking as missing all points with 0 density
    • Consider changing the color scale to ignore the few points with very high densities (it might be interesting to see where the high densities occur, which might say something about the survey)

 

  • Create grids  Create a new grid that matches the DEM covering the map.
    • 1 m is a good starting point for the grid size.  Older lidar might not support 1 m resolution, and some newer lidar might support denser grids which will be larger.
    • Create a DSM, DTM, and NVS (can be done in one step).  You can only get the DTM if the point cloud was classified; if it was not, you can classify it but must download other software to do this.
    • Check the hole density.  If too large, fill across the holes.   This may not appear to fill a large fraction of the holes if you have coastal lidar which does not cover the entire map area, but will fill a lot of the holes in the portion of the DEM with data.
      • You can change the settings on the Create grids form, and create the grids again
      • Interpolate across holes (smooth)
      • Filtering the DEM
  • Save DEM for the grid or grids you want.  Save in MICRODEM or Geotiff format.
