Heat Map

The heat map shows the density of points using a color scheme

Heat map showing locations visited; green the fewest, red the most Heat map showing the number of buildings closest to each location along the coastline.

Two cases:

  • Start with a DEM.  If your original map is not a DEM, create a UTM DEM to match the region you want, on the File menu.
    • Heat map Option on Raster GIS menu.
    • Select multiple DBs which will be opened, processed, and closed.  They must have LAT,LONG fields and must be point databases.
    • No ability to Filter the databases.
    • This option should be very fast
    • Example would be
      • a series of logs showing locations of something over time, and the desire to to find where something spends the most time
      • crime locations, and find the most dangerous area
  • Start with a single DB, which can be point, line, or polygon.  Must be fields with LAT,LONG, but they can have any name as you will pick them.
    • Filter if desired
    • Pick the Lat and Long fields to use for the heat map.  For lines or polygons you can use the centroid, or Nearest point on line/perimeter of another database
    • Examples
      • Building density along the coast line

Creates a UTM projection map with a specified grid size.
