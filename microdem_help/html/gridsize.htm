The number of points to cover a one degree square at the latitude of Annapolis, MD (39ºN). This assumes 2 byte integers for the elevations; most newer DEMs, especially those at higher resolution, use 4 byte reals, which would double the size of the DEM. These values also do not include any headers or metadata, which are typically a tiny fraction of the file size.

The values for spacing in geographic coordinates would not vary with latitude, while DEMs with spacing in meters will get smaller with latitude.

Actual file sizes depend on the size of the region covered by the DEM and the resolution.
