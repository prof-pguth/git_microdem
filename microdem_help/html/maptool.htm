Map Window Toolbar

These options affect only the map window on which they appear.  Items on the main toolbar are not restricted to a particular map window. 

The icons shown will depend on:

  • The type of data in the map window
  • The size of the window (less important icons will not appear in small windows)
  • The purpose of the map

All choices here can also be accessed by menu choices, and some can be accessed with a right click on the map.
