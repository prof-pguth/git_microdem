Plotting Highways

The DB groups file, stored in c:\mapdata\database\groups, which will plot roads in the US.  Each category uses a different filter, and appears only when the pixel size on the map is less than that in the PIXEL_SIZE field.  This file grouping can be created on the Database menu  and then modified to add the FILTER and  PIXEL_SIZE fields.

Plot the grouping. 

 

Map of the entire US, showing interstates.  Since the pixel size is greater than 5000 meters, only the Limited Access Highways appears.

Regional map showing the interstates and other principal highways.

Local map showing all highways  .Since the pixel size is less than 1000 meters, all Highways appear.
