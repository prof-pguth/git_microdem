MICRODEM Code Model

Class/Object Function Form/Window Notes
DEMDatum Geodetic and coordinate conversion routines    
tDEMDataSet Model of a Digital Elevation Model   Global variable
tSatImage Model of satellite image, air photo, or scanned map   Global variable
MapProjection Model of an arbitrary map projection    
GISDataSet Model for a shapefile    
tDBTableF Window to display a data base Yes  
tMapDraw Model for a map   Must have a tDEMDataSet, tSatImage, or a MapProjection
tMapF Window for a map Yes  

This is provided as an aid to understanding what MICRODEM does, and the interfaces it uses.

Important notes:

  • No promise that the interface will remain constant. 
  • The underlying source code will not generally be supplied.  Source Code, last DOS version of the program, circa 1995, is available.
  • There are probably about 200 code modules in the program, and 350,000 lines of code.
