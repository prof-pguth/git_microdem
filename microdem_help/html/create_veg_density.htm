Create Vegetation Density Voxels

Create on tools tab of  Point cloud options. 

  • You need a DTM of the area to use. 
  • This will create grids with the number of lidar point returns in bins above the ground surface.  Vertical bin size is currently set at 1 m.
  • You should have a horizontal bin size that includes a reasonable number of returns in each grid cell, since this works on the statistics of the returns. 
  • The horizontal bin size of the DEM used to open the point cloud will be the same as that used for the derived grids.  If necessary, Thin the starting DEM first.  You will have to use this DTM when using the data.
  • Filter the export by LAS class for instance to only include vegetation.
  • Pick the number of bins, which determines the highest vegetation.
  • Ignore or include points above the highest class.  If you do not ignore them, they will be placed in the highest bin.
