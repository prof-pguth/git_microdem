KML Beach Balls

 

KML Beachball export:

  • Open database file with Geology options popup menu, Earthquake focal mechanisms choice.
  • Select Plotting options.
  • Plot, Earthquake mechanisms, KML Export

Export should only be done for a small number of earthquakes, after Filtering the database.

Requires Global (Harvard) CMT catalog or similar format.
