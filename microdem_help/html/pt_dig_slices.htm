Point Digitize on Slices

Backup the database frequently (do one deck at at time, and only test the operation first to insure the program is running, and you understand its logic).

To digitize a deck.  Do each deck individually.

  • Open a second view.  You probably want the x-z and y-z planes.
  • Adjust scaling on the views so you can precisely locate the points. (Rescale on menu, or right click on  view.)
  • Pick the more buttons to see expanded options
  • +XYZ: digitize points on slices graph
  • Pick a file name appropriate to what you want to digitize. Include XYZ in the file name.  If the file exists, you can overwrite it or append new points
  • Digitize points. Double click on slice graph or registered deck plan.  The XYZ triple will be added to the key points database.
  • When done with a section, move either into out of the plane using the controls for slice placement.

 

Prepare text file with list of locations  you want to digitize.

  • Open table with data.
  • In necessary, Concatenate a new field that will be the name of the locations with artefacts.
  • Get list of unique values in the field, and save it to a text file.
