Resample/Reproject Shapefiles

Reproject shapefiles on Data Manipulation Form, under the Resample menu choice. The following are currently implemented:

  • Datum shift: change datum. Identify existing datum, and select new datum.
  • Specified shift
  • Add line endpoints to DBF file: (UTM coordinates).
  • Repair SHP and SHX bounding boxes for the entire shapefile.
  • Create DBF: if you have a shapefile missing the DBF file, this can create one with a record field for each record.  It will obviously not have any attributes.
  • Rename shape file

To subset a shapefile, you must open it with a base map.
