MICRODEM versions

The Program tab on the options form lets you select a number of subsets, with simpler menus, and access to labs and data downloads:

  • Regular (or Expert)--the default
  • Geology--geological oceanography
  • Physical geography
  • Remote sensing
  • DragonPlot--fire towers with Osborne fire finder

 Greater customization is on the on the Menu settings button of the Program tab on the options form.
