DB Plotting Colors

  Set the range of values to be used for coloring. By default, these are the min and max values in the database for this field (perhaps filtered), but you can override them. You might want to do this if there are a few extreme values which will cause a large part of the color range to not be used.  You can also verify the range of values in the database.

You can also pick the color scheme. 

Chloropleth colors gives you a number of predefined palettes.  These will tell you the number of color steps, which are relatively small.

You can reverse the color schemes 

 Color schemes with 256 colors:
  • Gray scale (top)
  • Rainbow colors, which have discrete breaks
  • Terrain colors
  • Spectrum colors (bottom)
