Geomorphometry Analysis

New DEM must be open and associated with the map. (requires geomorphometry menu choices be set)

Options; the parameters available for each vary slightly.

  • Create derivative grid.  This includes point transformations, computations for the point and its eight neighbors (slope, curvature, and so on), and computations that require a region about it point (moments, openness).  This will be done for every point in the DEM.
  • Create database on a grid.  You will pick the size of the region for the computations (as this goes up, the values do not change much over small distance), and the spacing for the center points at which the computations are done.  This could be turned into a grid, with the resolution center point spacing.  This could be faster than creating a derivative grid.
  • Create database for area shapefiles.  The sampling size will be the size of each record, and thus vary from record to record.
  • Create database for point shapefiles.  You pick a region sampling size

Database operations

  • Stats: perform statistical operations (not map based) on the database.
