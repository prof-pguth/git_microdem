Data Base Table Stats Options

Options on popup menu from Stats button on table display window.

  • Some options will be available only for certain types of database files. 
  • Some options will also be available by clicking on field name .
  • All statistical operations will use the current filter.
  • Count unique values: for a string field, the number of unique entries. 
  • List unique values: for a string field, a memo box with the unique entries in alphabetical order.
