Linear Feature Migration

Choice on Raster GIS menu, migration over time submenu.  This is designed for sequential time surveys of features like dunes.

Pick the direction:

  • N-S
  • NE-SW
  • E-W
  • NW-SE

Pick two grids, which must have linear features.  These could be created with  Find edges directional filter. 

Pick a maximum distance to check.

The program will create a table for each point in the first grid for which it was able to locate match in the second grid.  Positive values will be to the N, NE, E, and SE, while negative values will be to the S, SW, W, and NW.

Because this can only provide an answer in 4 directions, you need to decide which directions to check.
