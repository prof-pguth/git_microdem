Edit Registration Points

  • You can edit points in the grid.
  • Double click on a registration point in the data table to:
    • Delete it completely.
    • Move it to the bottom of the list. Since only the first three points will be used for a Rotate Reg, this lets you change which points are used.  You can also edit only the last point in the list, so if you want to move a point, you must move it to the bottom of the list first.
