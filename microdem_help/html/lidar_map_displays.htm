Lidar Map Displays

 

This is undergoing rapid changes.

 

Feature maps on another map

  • Open a map (DSM, image, lidar intensity)
  • Make it grayscale
  • Open on File Menu
    • Feature Grid: lidar classfication grid as a feature grid on the file menu, which will be in color
    • CHM grid for the vegetation heights, in shades of green
    • Change grid: difference over time.  Green for higher (deposition/growth), red for lower (erosion/destruction)

3D option

  • Open two grids
  • From the lidar classification, pick 3D map options
  • Pick choose reflectance or intensity from the second grid
