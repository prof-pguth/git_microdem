Geology options, Magnetic anomaly picks.  This shows points where someone has identified a particular anomaly.

Magnetic picks, shapefile or KML:

To get a new version of the file and prep it for MICRODEM:

  • Download global shapefile from http://www.soest.hawaii.edu/PT/GSFML/ML/index.html  (Seton and others, 2014) No updates since Oct 2018, and the site is not secure)
  • Open
  • Edit
  • Edit, Coordinates, add lat/long from SHP file
  • In the GEEK2007 field, limit the decimals to 3
  • Edit, String field operations, Trim blanks, All fields
  • Report, Text (CSV):
  • Open the TXT/CSV file, which removes a hugely excessive number of blanks in the string fields
