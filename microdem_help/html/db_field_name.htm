Data Base Reserved Field Names

The following field names have special significance:

  • NAME
  • LAT
  • LONG
  • LAT_HI
  • LAT_LOW
  • LONG_HI
  • LONG_LOW
  • COLOR
  • FONT_NAME
  • FONT_SIZE
  • FONT_COLOR
  • SYM_SIZE
  • SYM_COLOR
  • SYM_TYPE
  • PLOT
  • INDEX
  • LEGEND
  • DISTRICT--for post-census redistricting.
  • IMAGE
  • WWW, URL, or URL1, or WWW_xxx: link to web pages from the database record display.
  • QUAD
  • DIP
  • STRIKE
  • CGM_AREA
  • CGM_POINT
  • CGM_LINE
  • GEOSYM_ID
  • USE
  • DEM: open a DEM

These may not be used any more:

  • FILENAME
  • FENNCODE
  • DEM_NAME
