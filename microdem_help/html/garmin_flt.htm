Garmin FLT Files

FIT, Flexible and Interoperable Data Transfer is a file format used for GPS tracks and routes. It is used by newer Garmin fitness GPS devices, including the Edge and Forerunner.

MICRODEM converts it to a DBF when you open the file as a database.  This requires:

  • Install GPSBabel in the default location: https://www.gpsbabel.org/
  • Connect the fitness tracker to the PC with its USB cable.
  • The files will be in E:\GARMIN\ACTIVITY, with names like "97KB2003.FIT".  Use the Date Modified to find the desired file.
  • MICRODEM will execute: "C:\Program Files (x86)\GPSBabel\gpsbabel.exe" -i garmin_fit -o gpx -f c:\temp\97J72123.FIT -F c:\temp\97J72123.gpx  which will extract the latitudes, longitudes, times, and speed.
  • MICRODEM will natively convert the GPX to a DBF with the latitudes, longitudes, times.  This could be done with GDAL, but that creates a large number of empty fields, and creates a number of shapefiles.  The DBF files will be in c:\mapdata\tracks\.  They will be named with the date/time of the first reading.
  • Speed, distance, and azimuth can be added on the Navigation menu.  They will automatically added during import using your current settings, if you have the option checked on the Import/Export  tab of the options form,
  • The FIT files on the device will be deleted by default.  If you want to retain them, turn the option off on the Options form, Database tab.

The FIT documentation is included in the FIT SDK archive http://www.thisisant.com/resources/fit/  which also has some tools for dealing with FIT files.  These can extract all data from the file, including things like heart rate, but you would have to do some custom programming to convert the times and coordinates into something useful in GPS.

 

Testing has been with files from a Garmin Forerunner 35.
