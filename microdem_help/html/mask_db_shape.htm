Mask Database with Shapefile

Option from Edit button on database table form.

  • Pick  open area database to mask areas
  • Each record will be plotted on its own map, and that map used for the determination
  • Compute the point for each record in the database: for area shape files, the centroid will be used, and for point databases files, the coordinates of the point.
  • If the point is within a record in the database used as the mask, the MASK field will be marked with "1"; otherwise it will be "0".  If there is more than one MASK field, you will be asked which to use.

You can filter the database using the Mask field, and then save the filtered data using the Report button.
