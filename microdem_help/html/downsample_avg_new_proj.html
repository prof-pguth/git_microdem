Downsample, Average to new projection

Option on Raster GIS menu, Downsample

  • Pick the new projection, which should have pixel spacing significantly larger than the original grid so there will be a sufficient number of points
  • Each point in the grid will be assigned to the correct point in the downsampled grid, and they will be averaged (mean)
  • In addition to the new grid, a grid will be created with the number of points averaged in each pixel of the new grid
