Drainage Directions

  • Plot drainage directions from "Calculate, Drainage, Downhill Vectors" menu option.
  • Set parameters controlling display.
Map overlays:
  • Contours
  • Drainage directions (blue arrows).  These point in the downhill (aspect) direction
