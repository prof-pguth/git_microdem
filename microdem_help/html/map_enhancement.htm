Map Enhancement

Map Overlays for boundaries

  • US smart outlines--can put counties, states, or TIGER data automatically on your maps.
  • World boundaries, rivers, coastlines
  • GADM: higher resolution, with lower level boundaries.

Zoom in to region of interest

  • Subset data set--picks a smaller area, but keeps the map window the same size;
  •   Zoom in/out (same coverage area, but changes the size of the map, and can rapidly lead to a map that is too large for easy use)

Scalebar

  • Legends/marginalia after right clicking on the map, lets you put a scalebar on the map, and grid legends, and place them in a location that minimizes disruption to your map content.

Grids/Graticule

  • Grid adds or removes a grid, either UTM or Lat/Long. Set a grid and neatlines with  Prepare printer image
DEM Map Coloring
  • Map coloring can be changed with the Display parameter by right clicking on the map. 
  • You will probably get best results with Reflectance maps  or Elevation maps.  The shading on a reflectance map greatly enhances topography, and you can include elevation coloring as well.
  • If you need color for overlays, the grayscale reflectance if probably best.

Margins

  • Prepare printer image to get an option to set up the map for printing (or saving) with marginalia and map neatlines.

Legends can be added in several ways

Grayscale and subdue map can allow emphasis on map overlays instead of the base map.
