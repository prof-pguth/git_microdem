Map Legends

Legends explain the color or symbols on a map.  They work differently depending on the type of data (Raster DEM/Grid, or vector grid/shapefile).  The program also treats legends where strings are used, when the categories are discrete values rather than continuous numberic values, differently than continuous scales.

Raster Grids/DEMs

  • Legends/marginalia after right clicking on the map, which will embed the legend in the map area for the grid on the map.
  • Prepare printer image which can put the elevation  legend outside the map area.

Vector Databases

  • Pick legend on the Bottom panel controls   of the Symbol and color selection form. 
  • Put it in a different corner from the Grid legend and scalebar.

Legends with string fields

  • These can be very large, if there are a lot of categories, or the labels are very long.  They might cover a large fraction of the map, espcially if you have a small laptop screen.
  • The program will create a legend in a separate window if it thinks this will be the case
  • To place the legend on the map
    • Right click on the legend's graphic, and copy the image to the clipboard from the popup menu
    • Right click on the map, and pick paste image from clipboard.  Move it to the location where you want it, and double click to lock it and make it opaque.
  • Alternatively use Paint to paste in both the map and legend into a single graphic, with the legend above or below the map.
