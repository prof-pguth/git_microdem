RVT Relief Visualization Toolbox

Produces

  • hillshading,
  • hillshading from multiple directions,
  • PCA of hillshading,
  • slope gradient,
  • simple local relief model,
  • sky-view factor,
  • anisotropic sky-view factor,
  • positive and negative openness,
  • sky illumination, and
  • local dominance.

This is currently disabled.  If you want these options, run from the program's GUI.

Directions

  • Download from https://iaps.zrc-sazu.si/en/rvt#v and place in x:\gis_software\rvt\
  • Open the program, and select the options you want.
  • Close the program
  • In MICRODEM, open a DEM
  • On Raster GIS, External tools, select RVT
  • The DEM will be written to a RVT subdirectory in Geotiff format, RVT will be called, and the selected output files will be added to the RVT subdirectory
  • To see problems, run the batch file (rtv.bat), which will be in x:\mapdata\temp, in a cmd window.
  • To open the RVT program again for manual operation in its GUI, first delete the "process_files.txt" in the setting directory.
  • This probably will not work with geographic DEMs
