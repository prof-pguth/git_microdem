Test Fan Sensitivity

Pick on Calculate, Intervisibility menu from a map.

  • Pick location for the fan.
  • Pick viewshed parameters which will be the same for all the algorithms.
  • Pick radial spacing and point spacing along the radials to test.  The algorithm will be a radial lines discrete with the radial and point spacings you select; the horizontal curvature and point interpolation will be those in your current defaults.
  • The program will cycle through a series of fan, with different radial spacing and point spacing along the radials.
  • The program will prepare a table showing the time and coverage percentage for each algorithm.
