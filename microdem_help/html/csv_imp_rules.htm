CSV Import Rules

This table (currently CSV_IMPORT_RULES_v2.dbf) has the GAZ_TYPE (USGS, NGA,CCOG) and  which automates following the rules for field names.  Fields whose names are too long for the DBF rules must be shortened.  SKIP for the OUT_NAME means the field will not be imported because it is not useful.
