Gazetteer Toolbar

Gazetteers add a second toolbar to standard database table display  when loaded as a gazetteer.

  • Find:  as you type in a feature name, the gazetteer is filtered.
  • Plot & Label:  all of the current features are plotted on the map and labeled.  You are responsible for any filtering.
  • Plot:  all of the current features are plotted on the map.  You are responsible for any filtering.
  • Features:  standard gazetteer plotting and labeling is done.  This filters, and plots and labels the categories you want.
