Photogrammetry from Video

Image prep

  • Get screen captures.  They must all be the same size with the same extent. 
    • You can use PrintScreen, paste into Paint, and save.
    • You can use MICRODEM to automate the process
      • File, tools, load image
      • Make MICRODEM smaller, with just the PrintScreen button on the image editor visible.  We can capture the entire monitor the program is running on, but not all the monitors.
      • Arrange the screen so the movie covers as much of the screen as possible
      • Play the movie
      • Hit Print Screen on the image editor to capture an image.  You do not have to stop the movie, provided the motion is not super fast.
      • The images will be in "c:\mapdata\images", sequentially numbered.
  • Open one of the images
    • File, tools, load image
    • Currently you must be at 100% zoom
    • File, subset pictures
    • Pick the box desired.  Cut out any text, gauges, or other extraneous infomration that will confuse the image matching
    • Pick this subset on multiple images from the popup menu
    • Pick the images to subset, which must all be in the same directory.
    • Subsets will be saved in the same directory, with a "sub_" added to the start of each file name

SfM

  • Get Meshroom (requires nVidia graphics card; if you have an nVidia that is not working, try to update the graphics driver)
    • https://alicevision.org/
    • https://alicevision.org/#meshroom
    • https://github.com/alicevision/meshroom
    • https://sketchfab.com/blogs/community/tutorial-meshroom-for-beginners/
  • Load the images
  • Run
  • The results will be in the same directory as the images, in something like "C:\Users\pguth_2\Pictures\MeshroomCache\Texturing\22e5a490033b8337227007400e7ad40de3cae771"
  • Within the directory, you need 3 files, and they must stay together
    • texture_1001.png
    • texturedMesh.mtl
    • texturedMesh.obj
  • You can view in MeshRoom; use the load button below the 3D viewer window in the upper left corner of the screen

View in Cloud Compare

  • Image will likely appear "upside down", so use that knowledge to overcome any initial misorientation on your part
  • Clean up extraneous objects in the point cloud: https://www.cloudcompare.org/doc/wiki/index.php?title=Interactive_Segmentation_Tool
