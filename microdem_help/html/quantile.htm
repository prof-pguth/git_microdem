Quantile Scaling

There are two ways to do automatic coloring, without having to set the break points manually:

  • Linear scale.  From the maximum and minimum values, the breaks are distributed evenly.
  • Quantile scale: attempts to have the same number of records in each color category; the boundaries and sized of each color band will vary to enable this.

Annapolis Population Density
Linear scale.  Values range from 0 to 47,000.  There are very few densely populated blocks, so most of the color scale is unused. Quantile scaling with a five color scale.  The breaks are chosen to put the same of blocks (or as close to it as possible) in each category. In this case there are 645 blocks, and the boundaries are very unevenly distributed.  Quantile scaling with a five color scale, with the unpopulated blocks not included (only 436 blocks actually have people living in them).
