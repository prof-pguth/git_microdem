Create an index file for the TM scenes.

To get all the NDVI,

  • Open database without map
  • Filter the index if desired.
  • Stats, TM Process, NDVI

To clip all the NDVI grids to a common area

  • Edit a DEM/grid to have the area desired
  • Data Manipulation, Subset, Mask DEM

Working with a series of NDVI grids:

  • File, Open submenu, Open NDVI grids
  • Pick the directory for the grids;  They will be opened, and a single one will be displayed.
  • Image analysis, NDVI/NSDI.  These will create a database and a map; you can do some displays with the database to color by the satellite or the year.
