Map Coordinates

Submenu on "EDIT" button on database table form.  The button must be enabled.

  • Add UTM--UTM x and y, and the UTM zone.  You can force the conversion to a particular zone, or use the correct zone based on the longitude.
  • Add lat/long: adds but does not populate the fields.
  • Add centroids
  • Add UTM from offsets--
  • Add MGRS
  • Add MGRS  bounding box for each record
  • Add Mercator--Mercator x and y, and h/k distortion factor.
  • Add projected coordinates--using the map projection of the map
  • Add decimal seconds string--adds text/string field of the form N54°23' 4.29" E0°26' 5.89".
  • Add decimal minutes string--adds text/string field of the form N54°23.072' E0°26.098'.
  • Add decimal degrees string--adds text/string field of the form N54.3845250° E0.43497000°.
  • Fix 360 longs--convert longitudes from 180 to 360 to negative values.
  • Add lat/long from shp file  
  • Add lat/long from joined database
  • Add Bounding box
  • Add XYZ from shp file  
  • Add XYZ from joined table: XYZ fields will be added if they do not exist.  For the field used to join, the program will find all the unique values in the primary table.  It will filter for each unique value in turn, and apply a corresponding filter to the join table.  If the filtered join table has at least as many records and the filtered primary table, XYZ values will be placed in each record in the primary table.  The first record will be matched to the first record, the second to the second, and so on. 
  • Convert DDMM.MH/DDDMM.MH/ string to Lat/Long: e.g. 2037.0S/02913.0W
  • Convert UTM to Lat/Long
  • Datum shift (reproject lat/long coordinates): pick the datum used by the database, and a new datum, and the coordinates will be transformed
  • Vertical datum shift: creates a new field with the  shift between the WGS84 ellipsoid, and the EGM2008 geoid.  The shift file from NGA must be available in  C:\mapdata\geology\tgeology_grids
  • Randomize duplicate points: if there are points in the database with the same coordinates they will be given new coordinates within a specified region about the point.
