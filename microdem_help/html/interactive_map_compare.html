Mulitple map interactive comparisons

On Overlay menu, Mulitple map interactive comparisons

  • Variable opaque merge (interactive) (two maps)
  • Merge another DEM hillshade (quick)
  • Remove quick DEM hillshade merge
  • Vertical swipe compare (two maps)
  • DEM map flicker movies (all open maps)
  • OpenGL, drape on another DEM. When the map is a grid with some computed value, you can drape it on an actual DEM with elevations.  Pick from the menu on the grid to be draped, and then pick the DEM.
