Max/mean/min grids from point data

Option on Calculate menu

Creates UTM grids with your choice of point spacing, covering the current map area. 

Open any map, and zoom to the desired area.  Since this creates 4 new grids, you want to minimize the area of each.

Pick the grid size to use, in meter.

Select the files to put into the grids.

  • 3D point shape files (currently the best option)
  • Databases: pick field for the z values to put in the grids.
  • ASCII XYZ data.  Order currently must be lat/long/z; for other choices, convert the file to a database.

Creates grids for the min, max, mean, and point count within each grid cell.
