Database Auxiliary Files

MICRODEM can compute the following files to go with a database stored in a "db_aux" subdirectory of the directory where the DBF file is located.

  • Same base name as database, .DEF extension: display settings for the database
  • NEIGH-DB.DBF
  • RANGE_ + name of the database: contains the max/min in each field, which can be used to greatly speed up color coding of the map with database fields.  This option can be toggled on  Database tab of the options form
