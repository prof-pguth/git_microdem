Add NLCD classification to database

Option on  Edit tab of the database table form.  The database must be an area shape file.

You must have opened an NLCD data file, which must have been saved so that MICRODEM can recognize it.  

You must also have the current LAND_COVER.DBF.

If your NLCD is not a single file, you cannot merge them in MICRODEM.  

New image: open all the NLCD files that correspond with your database.

  Database:  open the database on any one of the maps.

Pick this option to add NLCD do the database.

You can select a single NLCD category, or you can do all at the same time.  Doing all will take effectively the same time as doing a single one, and is a slow process

A series of fields will be added to the database, with the fraction of the record belonging to each NLCD categories.

You can use map algebra to merge categories.

 

As an alternative:

Mask the shape you want on the land cover data set, and then do a popup legend.
