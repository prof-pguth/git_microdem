Bin Corners

Must open three DBF files:

  1. Side limits--must have fields "SIDE", "LEFT", and "RIGHT"
  2. Key points file (created with Digitize y-axis)
  3. Artefact Bin frequency table, created from the Artefacts database with a Frequency table operation on the BIN_NAME.  Must have fields "BIN_NAME" and "N".  You must insure that the BIN_NAME field follows the conventions from the image below.

Creates a CSV file which you can turn into a database. CSV will be in the same directory as the Project file.  This requires no input from you after entering the three files.

Note any error messages about problems in creating bins; you must then fix the input files or determine if the anomaly requires other operations.

  • Files needed
  • Point Digitize on Slices
  • Move points up/down
  • Digitize y-axis
  • +Box--add more bins interactively
  • Add bin corners to data base
    • Create a bin frequency table
    • Open it with the Box option
    • Turn on edits
    • Add the corner coordinates  to the DB table: Edit, Insert New Fields, XYZ, and pick 4 sets of coordinates.
    • Double click on a record in the DB table display, and pick Edit, Add XY box
    • Outline the box, and pick the Z value
    • If there are already coordinates, this will overwrite them
