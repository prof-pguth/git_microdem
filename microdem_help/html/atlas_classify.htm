Atlas Classify

Option under active development; contact us if you want to try to use it and are having problems. 

Open on Raster GIS Menu, Geomorph atlas. You must first load a base map to use (ETOPO is a good choice).

  • Grids.  Set directory for grids on Directories tab of options form.  The grids must exist in that directory, and you must have a correctly defined data base to control operations.
  • Database.  Currently hard wired with required files.
