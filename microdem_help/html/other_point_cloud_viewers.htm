Must have

  • mcc-lidar
  • LAStools (read license carefully)
  • Cloud Compare
  • Fusion

OPALS - Orientation and Processing of Airborne Laser Scanning data--does classification, among other things.  Must license for more than 1 million points.

Commercial, no demo version: http://lidarwidgets.com/

PDAL

OpenLSEF: https://beta.openlsef.org/digital-elevation-models/ Extracting information for 3D point clouds

Web viewers:

  • http://plas.io/    LAS and LAZ input, locally and some web samples
  • http://potree.org/
  • https://usgs.entwine.io/ USGS online viewer for lidar
  • http://speck.ly demonstrates Greyhound+Entwine in action. Both software components are open source.  Variety of data sets online.
  • Handheld: LiMo: https://play.google.com/store/apps/details?id=com.ogs.limo
    • File extensions must be lower case
    • 3D motion hard to understand
    • Color for 3D geotiff all the same
      • Help file does not work
  • http://ahn2.pointclouds.nl/ Netherlands viewer, with ability to download points in LAZ format; source code available
