Fabric By Region

On

  • Calculate menu, Topographic grain submenu.
  • Right click on map, Geomorphometry
 

Set parameters for computation of the fabric parameters by region size.

You can plot the following:

  • Graph of organization parameter by region size
  • Graph of flatness parameter by region size
  • Graph of fabric direction by region size
  • SSO diagram for each region size (Nets with grain fabric)

 

Set region sizes for the computation. They are the size in meters of a box centered on the point you select. Because of the data spacing, the actual values will vary somewhat from your selections.

This a a point parameter for a single point.
