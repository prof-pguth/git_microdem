DB Insert Field

Edit button on Table view, which provides popup from which you pick Insert new fields to get the following choices.  Only enabled if you check the "EDIT" box on the right of the row of buttons, since all changes are permanent (save a copy of the database if you are concerned about editing mistakes).

  • Insert new fields
    • Insert field--lets you pick the name, type, and field length.
    • Concatenate fields: lets you combine two fields with optional text before, in the middle, or after.
    • Split field with separator: if you have a field with records like "Annapolis/MD", this option would let you name one or two new fields, identify the "/" as the separator and then:
      • Keep before: make a single new field with just "Annapolis"
      • Keep after: make a single new field with just "MD"
      • Divide two fields: make two new fields with "ANNAPOLIS" and  "MD"
    • Copy string field
    • Insert viewshed fields: for a point database, this will add the fields for Weapons fan/viewshed , and populate them with default settings.

    • Insert WWW field:
    • Add integer code field: finds the unique values in a string field, creates an IDX_n field, and assigns each entry a numerical value starting with 1 for the unique entires in the string field.
