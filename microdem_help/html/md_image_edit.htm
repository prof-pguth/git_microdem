MICRODEM Image Editor

Tools submenu of  file menu:

Load image for display or various manipulation routines. This is not meant to be a complete edit program, but is a collection of routines I have found useful.

If you edit a JPEG, its EXIF data will be preserved.  You are responsible for not changing significant characteristics of the the image.

Options:

  • Rotate image by an arbitrary angle.
  • Subset image to a particular aspect ratio
  • Change colors (Color menu).
  • KML Legend: KML button on toolbar places the image as a legend in the upper or lower left of the map in a KML file for Google Earth.
  • Get  color palette if less than 1024 colors.

      Color palette displayed as a database, with the count of pixels in the image in each color.
  • Erode/dilate images
  • Process all files in a directory
    • Open single image
    • File, Review directory
    • Cycle through images with the < and > buttons
  • Set bottom portion of image to single color
    • Pick the color to use, by
      • Right click, and "Pick this point for solid color"
      • Recolor menu, Pick solid color
    • Right click on image, and all points below will be recolored
  • Repair lost EXIF headers
    • Open an image with intact EXIF header
    • On the file menu, pick the option to clone its EXIF header.
    • Then pick the files to be operated on.
