Map Display Parameter

You can adjust the DEM map display by:

  • Using the Modify, Display parameter menu choice.
  • Right click on the map, and picking the "display parameter option"
  • Elevation/grid value map
  • Slope map
  • Reflectance/Hillshade map
  • Difference map
  • Openness--quick version that can make a good grayscale base map in suitable terrain
  • Contour
  • Aspect
  • Aspect/slope merge DEM color merge,
  • DEM mask
  • VAT
  • Blank--useful to create a map showing only overlays.

The options compute the values on the fly, and do not create a second grid.  They thus do not clutter up the hard disk.  Create derivative grid generates a second grid, which allows more complex display and analysis.
