Structural Geology File Format

These must be in a dBase file, with the fields:

  • Lat
  • Long
  • Orientation, either as
    • Separate DIP & STRIKE fields
    • Single DIPSTRIKE
  • PLOT
  • FEATURE, with a numeric code indicating the type of feature
    • 0= sedimentary bedding
    • 1=joints
    • 2=foliation
    • 3=fault plane

The old option, a file with the extension NET, is no longer supported.
