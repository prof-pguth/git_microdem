Edit/Display DEM Headers

This window allows you to display or edit DEM headers. 

  • To edit,  use  Data Manipulation form,  Edit menu.  You should be extremely careful using this option, as errors may cause severe problems and you could be unable to reopen the DEM.
  • To display, use option on analyze menus.  The display option disables edit operations.
  • While the DEM is open, you can change the Z units on the Edit Menu.
 

For lat/long DEMs, unless you have an even spacing in minutes or seconds, use a spacing in decimal degrees.

Projection requires that you link a projection file, created with the Vector maps option and then saved by right clicking on the map  The program can automatically create some of these files, for instance for SPCS DEMs.  It is probably not working, and will replaced by new options with the EPSG codes.  Contact us if you need this capability.
