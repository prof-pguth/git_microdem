Change Sensor Azimuth for LOS

   Option for Line of sight/Topographic Profile .

You can use the slider bar to change the azimuth.  If you use the right and left arrow keys, the response will be immediate.  If you use the mouse, you must click the redraw button to get the new view.

The map will show the visible and masked terrain.

This example also shows the effect of masked airspace below the blue lines on the right side of the profile.

Available for required antenna height if you select the LOS option.
