MICRODEM Installation

Get program and CHM help file from https://microdem.org

Source code and archived installation available at https://github.com/prof-pguth/git_microdem

  • Complete MICRODEM install (~30 MB). 
    • Installs 64 bit version Windows and required files into c:\microdem\. 
    • Uses 7Z SFX intallation program.
      • Does not require administrator priviledges
      • Will be a small window asking permission to install, which may be hidden, in which case it will probably have its icon on the far right of the task bar.
      • Will not install shortcuts, icons, or entries on the program list.  You must and create them if desired.
        • In Windows File Explorer, navigate to c:\microdem
        • Right click on microdem.exe and pick options desired:
          • Pin to taskbar
          • Pin to start
          • Send to > Desktop (create shortcut)
      • The program does not use any registry entries.
      • This includes support databases
  • Help file which you must place in c:\microdem\:

Executing the installation program will create two directories:

  • c:\microdem.  You can put this elsewhere, but if you encounter problems, you were warned and we will not try to figure out what is wrong unless you can very clearly indicate exactly what steps are required to recreate the issues.  Our limited testing in other locations has been in the root directory, x:\microdem, so you might try that.
  • c:\MapData\ with subdirectories. You can move this entire directory to another location or computer.  If copying from an external hard drive or USB drive, you do not need administrator rights to install MICRODEM into c:\microdem.  This must also go into the root directory of its drive.

Updating the program can now only be done with a full install.  The difference in size does not warrant multiple procedures.

Uninstall

  • Delete c:\microdem
  • Delete c:\mapdata, unless you want to retain any data.
