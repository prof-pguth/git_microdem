Subset Drainage Basin

(may not work properly; use with caution)

This option allows you to subset a DEM into component drainage basins. You access this procedure from the data manipulation option of the File menu; it appears under the Subset main menu choice.

You select the DEM, and then select points one at a time to start the basin delineation. All neighbors of the selected point at either the same or a higher point are included in the basin and flagged for later checking; expansion ends when all flagged points have been added and checked for uphill neighbors not yet in the basin. Points added to the drainage basin will be shown in white.

You can save the DEM with the save option under the file menu. You can save either the masked points in white, or the unmasked points, to a new DEM. Saving will end the basin delineation
