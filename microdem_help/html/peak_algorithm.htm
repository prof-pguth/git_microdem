 This is a 1 m lidar DEM, and the search radius is 2 m.  Based on the radius selected, the program picks the number of pixels to go in the x and y directions.  The number can be different for a geographic DEM.

In this example the program considers a box 5 pixels by 5 pixels in size, centered on c (2 m each direction, with 1 m LIDAR DEM).

Peak Algorithm.  Point c is a peak if:

  • C must be taller than all other points in the box.
  • C must be taller than  than the minimum point in the region by the selected peak height (relief).

Pit Algorithm.  Point c is a pit if:

  • C must be lower than all other points in the box.
  • C must be lower than the 8 points marked by "x" by the selected pit depth.

Spire Algorithm.  Point c is a spire if:

  • C must be taller than all other points in the box.
  • C must be taller than  than the 8 points marked by "x" by the selected spire height.
  • If the looser standard is allowed, it must only be taller than the number picked, from the 8 marked points.  As an example, this might allow selection of a lamppost, which would extend at the top and thus not have all 8 neighbors lower.
