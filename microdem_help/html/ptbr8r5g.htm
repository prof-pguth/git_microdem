Rotating Text

This box lets you add text to the image in the main window with any desired orientation. When you select this option from the main menu (Text, Add option) this box appears.

You must pick a font that is scalable (a True Type font), pick the orientation angle, and enter the text.

The orientation angle is in degrees. An angle of 0 gives horiztonal text; the angle increases in a counterclockwise direction (standard angle notation for mathematics).

The OK button will write the current text. You can also write the text by double clicking on the image where you want the text positioned.

The Cancel button removes the last text written.

To close the window, Uncheck the option on the main menu.
