Point Cloud Mask via Slices

This option may not be fully robust.  Do not try it with more than one graph open.  This option does not appear to be very useful, and we may remove it.

Mask Pts button of the Point Cloud manipulation form
  • Pick field to use for masking.  If it does not exist in the selected  shapefiles, it will be added. This can be slow if you have very large databases, or very many of them.
  • The selected shape files will be opened, and the tables displayed.

Toolbar lets you:

  • Select individual points by double clicking on the slice graph.
  • Outlining a box (drag from upper left to lower right) on the slice graph.
  • Points will have the mask set, and the memo box will indicate the number of points masked in each shape file.
The masking will put "Y" in the mask field for the selected points. You can Filter based on this field, save the filtered database, and use it for further operations.
Procedure to create a new 3D point shapefile:
  • Select points and mask using the procedure above.
  • Filter the database to just those points with a "Y" in the mask field.
  • Save the filtered database as a DBF file. (Report menu on the database table form). 
  • Open the new DBF file.
  • Write the point shapefile SHP and SHX files. (Report menu on the database table form).  This will make many operations significantly faster.
