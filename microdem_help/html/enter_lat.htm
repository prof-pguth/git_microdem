Enter Coordinates via Keyboard

 

Form allows coordinate entry using lat/long, MGRS, or UTM.

The caption bar will tell you the use of the coordinates, and the current datum.

You can save the default hemisphere so that it will appear the next time.

Lat/Long: Enter either an integer or a real value into each box. 

  • To enter decimal degrees, enter 23.50 in the degrees box. 
  • To enter degrees and minutes,  enter 23 in the degrees box and 30 in the minutes box.

Insure that the correct latitude radio buttons are checked for the hemispheres.

If you change the decimal entry form, the current entry will be converted to the new equivalent.
MGRS:
  • Enter using the top row to enter a correct MGRS as one continuous string.
  • Enter using the bottom row to enter the Grid Zone, 100K grid square, the easting, and the northing.

Easting and northing can be entered with as much precision as desired, from 1 digit each (10 km) down to 5 digits each (1 m)
UTM:

All fields must be filled out.  The easting will normally be 6 digits long and the northing 7 digits, and they must be completely filled in.
