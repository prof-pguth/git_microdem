Segmenting a Digital Elevation Model (DEM) using the Orfeo Toolbox (OTB) typically involves applying segmentation algorithms, such as Large-Scale Mean-Shift (LSMS), to identify topographical features or geomorphological units based on elevation values. Because DEMs are single-band images (representing height), the segmentation acts on height gradients rather than spectral color. [1, 2, 3, 4, 5]

Here is how to segment a DEM using OTB, primarily via QGIS:

1. Pre-requisites and Setup
  • Install OTB: Download and extract the Orfeo Toolbox.
  • Configure QGIS: Go to Processing &gt; Options &gt; Providers &gt; Orfeo Toolbox, enable it, and set the application folder to the path in your extracted folder.
  • Input Data: A GeoTIFF formatted DEM (e.g., SRTM or DSM). [6, 7, 8, 9, 10]
2. Segmentation Workflow (via QGIS OTB Provider)

For large DEM datasets, the Large-Scale Mean-Shift (LSMS) workflow is recommended to avoid memory issues.
  1. Step 1: Mean-Shift Smoothing ()
    • This reduces noise in the DEM while preserving edge boundaries, which is crucial for elevation data.
    • Input: Your DEM ().
    • Parameters: Adjust the spatial radius and range radius to control smoothing strength.
  2. Step 2: Segmentation ()
    • This converts the smoothed DEM into segmented regions based on elevation similarity.
    • Input: The smoothed image from Step 1 ().
    • Parameters: (minimum object size in pixels to prevent over-segmentation).
  3. Step 3: Vectorization ()
    • Converts the raster segmentation output into polygons (Shapefile or KML).
    • Input: The segmented image from Step 2.
    • Output: Vector file. [1, 12, 13, 14, 15]

4. Line Segment Detection (Topographic Lines)

If the goal is to segment the DEM into ridges or valleys (edges), use the Line Segment Detection application.
  • Algorithm: Based on the Burns, Hanson, and Riseman method.
  • Usage: Extracts structural linear features from the raster DEM and saves them directly as vector data. [18, 19]

  • Elevation Data: If the OTB application asks for elevation data (e.g., for ortho-rectification), you can point it to a directory containing your DEM files. [11, 18, 20, 21]

Here is how to segment a DEM using OTB, primarily via QGIS:
1. Pre-requisites and Setup
  • Install OTB: Download and extract the Orfeo Toolbox.
  • Configure QGIS: Go to Processing > Options > Providers > Orfeo Toolbox, enable it, and set the application folder to the lib/OTB/applications path in your extracted folder.
  • Input Data: A GeoTIFF formatted DEM (e.g., SRTM or DSM). 
2. Segmentation Workflow (via QGIS OTB Provider)
For large DEM datasets, the Large-Scale Mean-Shift (LSMS) workflow is recommended to avoid memory issues. 
  1. Step 1: Mean-Shift Smoothing (otbcli_MeanShiftSmoothing)
    • This reduces noise in the DEM while preserving edge boundaries, which is crucial for elevation data.
    • Input: Your DEM (-in).
    • Parameters: Adjust the spatial radius and range radius to control smoothing strength.
  2. Step 2: Segmentation (otbcli_LSMSSegmentation)
    • This converts the smoothed DEM into segmented regions based on elevation similarity.
    • Input: The smoothed image from Step 1 (-in).
    • Parameters: minsize (minimum object size in pixels to prevent over-segmentation).
  3. Step 3: Vectorization (otbcli_Vectorization)
    • Converts the raster segmentation output into polygons (Shapefile or KML).
    • Input: The segmented image from Step 2.
    • Output: Vector file. 
3. Alternative: Connected Component Segmentation
For specific, smaller-scale segmentation, you can use otbcli_ConnectedComponentSegmentation. 
  • This allows defining a formula to merge neighboring pixels based on whether they meet specific criteria (e.g., elevation thresholds).
  • Parameter expr: A formula for segmentation, such as defining slope ranges or absolute elevation ranges. 
4. Line Segment Detection (Topographic Lines)
If the goal is to segment the DEM into ridges or valleys (edges), use the Line Segment Detection application. 
  • Algorithm: Based on the Burns, Hanson, and Riseman method.
  • Usage: Extracts structural linear features from the raster DEM and saves them directly as vector data. 
Tips for DEM Segmentation in OTB
  • Scale: Ensure your tilesize in LSMS is appropriate to handle large data without crashing, even if the total image is small.
  • Visualization: Use the ColorMapping application to visualize the .tif result in RGB.
  • Elevation Data: If the OTB application asks for elevation data (e.g., for ortho-rectification), you can point it to a directory containing your DEM files.
