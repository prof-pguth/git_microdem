MICRODEM Help File

Currently the help file is only available as a CHM file, downloaded from https://microdem.org

You have the following options that have been tested:

  • Use the CHM viewer in Windows.  This has problems with scripts not working, but has table of contents, index, and full text search.
  • Decompile the CHM file, using 7ZIP (or the Windows help compiler).  This loses the table of contents, index, and global search.  The directory structure follows the old web page.
  • Open with SumatraPDF.

At https://web.archive.org/web/20230922083644/https://www.usna.edu/Users/oceano/pguth/md_help/html/microdem.htm, the Wayback Machine has 23 captures of the help file between 2015 and 2023 (2024 if you count messages that popular web pages do not survive retierment of their creator).
