Database Record Display

Right click on record in table display window and pick Record Display. 

Buttons on the top allow specialized options.
  • <Prev, Next>: cycle through records
  • CSV save:
  • Change image size
  • +Font: change font used
  • HTML: save table as HTML

 

Tabs select the view, if there are graphics

  • Database
  • Image: size set in the edit window
  • Climograph
  • Earthquake focal mechanism, and nodal plane orientations
  • Focal planes
