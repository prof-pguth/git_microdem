Military True Type Fonts

You must have the military TrueType™ fonts from Major Tom Mouat (www.mapsymbs.com) installed. The fonts are used with his permission.  This takes advantage of the zero-width characters in his fonts. You can use any other font, but unless it follows the same conventions as his fonts, the symbol color fill and unit size will not work (so just don't try to use them).

There are two versions availalble:

  • http://www.mapsymbs.com/app-6a.html   Latest and greatest.  Old timers might not like the diamond shaped icons.  The program does not recognize these as military, so the "Military options" do not appear.  This could be fixed if anyone uses them.
  • http://www.mapsymbs.com/mapsymbs.html  Classic.  The examples in the directions used these.  The installation program does not work in the latest version of Windows, but the option for "just the fonts" works.
  • If the download does not work, try a differnt browser.

You can check for the fonts using the Fonts option in Control Panel, where you can see them listed.

If they are present, you have access to the Military icon generator
