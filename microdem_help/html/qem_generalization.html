Program name

  • https://github.com/xiceph/physical-geomorphometry-tools/tree/main/generalization sends you to a https://github.com/xiceph/physical-geomorphometry-tools/releases  page and says to get the qem_generaliazation.exe file, but the only file there is raster_tools.exe
  •  I think the two pages listed above are not consistent and call it generalization and raster-tools
  • Unless you will add more options, Generalization might be better choice

Modify help

  • Unless some of the optional parameters are included, nothing happens.  At least one place in the documentation says it should work.  I think this should be there, with a recommended general purpose set of defaults.  If users have to set multiple parameters, this becomes another black magic tool.
  • Output appears to be scaled 0-255.  Is there any way to scale it to the original DEM range?  Document in any case.
