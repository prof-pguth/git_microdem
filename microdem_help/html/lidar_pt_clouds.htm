LIDAR Point Cloud Sources

Best sources for free LIDAR point clouds (probably LAS or LAZ format):

US Interagency Elevation Inventory:   https://coast.noaa.gov/inventory/

For MICRODEM, the data must be in UTM or geographic coordinates.  There are options to reproject with MICRODEM, liblas, or LAStools.  LAS lidar

General caveats:

  • May need to get LAStools to
    • reproject data if it is not UTM
    • uncompress LAZ data if that is what you get
  • MICRODEM will identify each project area by the name of the directory.  You should put the files for each project in a separate directory, and give it a short, meaningful name

USGS TNM Download National Map directions https://apps.nationalmap.gov/downloader/ Probably best starting location in the US
3DEP Lidar Explorer https://apps.nationalmap.gov/lidar-explorer/#/

Alternate USGS source I have not used much.
1/15/25:

  • US Albers projection at least in the one tile I downloaded
  • Pros: does not get a lot of tiles outside the area, which the National Map does
  •  Cons: no ability to load outlines, no batch downloads; while downloading a single tile at a time, no indication is is working
OpenTopography directions https://opentopography.org/ Mostly US, but now a lot of data from New Zealand.  If you have an edu email, you can get USGS and NOAA lidar.
NOAA Coastal Lidar directions Graphical: http://coast.noaa.gov/dataviewer/?#
Projects list: http://coast.noaa.gov/dataviewer/?dataregistry/#/coastallidar
Getting lots of data		 Some surveys may penetrate water; others only coastal strips of land.  For many areas multiple dates are available, showing temporal change.
Over 800 data sets now available at OpenTopography to academic users.
