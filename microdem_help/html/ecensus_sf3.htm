SF3 Data from the Census Bureau

This data is available for the 2000 Census.  It was based on the long form filled out by 1/6 of the households.  It has been replaced with American community survey

It is demographic data only: data files (no geometry)--which must be linked/joined to Geometry files for Block Groups or Tracts

This site has been redesigned (and it is not easy to find the 2000 data), and these old directions are no longer current.  If you want the 2000 data you will have to figure out how to get it using the American community survey directions.

Some interesting tables:
  • P8, Sex by age
  • P31, Travel time to work
  • P37, Sex by education (Also P148)
  • P52, Household income
  • P84, Sex by earnings
  • P145, Sex by age
  • H47, Plumbing facilities
