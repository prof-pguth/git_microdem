Using EGM Grids in MICRODEM

  • Grids can be used to do a vertical datum shift on the edit menu.
  • Point data base (such as ICESat2): EGM2008 can be used to add a field VDATM_SHFT, Edit, Map coordinates.  You can then
    • add to an elevation field in the database to get the geoid height if you have ellipsoid heights
    • subtract from an elevation field in the database to find the ellipsoid heights if you have geoid heights.
  • Right click, cartography will show the values for 96, 2008, and the difference between them
