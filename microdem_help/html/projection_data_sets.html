Projection Data Sets

MICRODEM uses projection data downloaded from OSGeo4W, which is used by GDAL and is ultimately maintained by PROJ.

With an OSGeo4W default installation, this will be in  OSGeo4W\share\proj but can be copied elsewhere. 

  • The  README.DATA file describes the data files, and their maintenance on github at https://github.com/OSGeo/PROJ
  • There is a text file describing each of the files from a national mapping agency.
