MEM Power Spectrum Options

  The following options control the MEM Power spectrum:
  • Number of profiles. The profiles will be evenly spaced throughout the analysis region.
  • Number of poles for the MEM method. Press and others (1987) suggest 10, 20, or 50, but note that the method can be quirky and that you should experiment.
  • Low and high frequencies used.
  • Log/log plot.
