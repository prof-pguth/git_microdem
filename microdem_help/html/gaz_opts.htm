Gazetteer Label Options

When labeling features from the gazetteer, you can turn on or off categories with the Features button, and pick the color for the symbol and the text.

Pick the gazetteer file to use. These should be in c:\mapdata\gazetteer: USGS Gazetteer for a state, or NGA gazetteer for a country.

You access this window in two ways:

  • Map Overlays
  • Gazetteer toolbar
