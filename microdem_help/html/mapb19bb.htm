Grid Contouring

Gridding

  • regular pattern of data points, usually square
  • handle all possible ways for contours to enter and leave grid
  • draw only one square at at time (need not know where contour line goes; it will eventually be completed)

grid cells: if large, fast operation but coarse appearance with straight line segments obvious

sorting data before gridding speeds finding neighbors (2D structures)

Interpolations:

  1. any interpolation smooths the surface, removing peaks and valleys
  2. interpolation options
  • simple linear or 1/D or 1/D² weighting with distance (grids values always between data points)
  • fit plane to nearest neighbors (trend surface); considers slope, so can be larger or smaller; can cause problems with extrapolation at edge of data set
  • fit curved surface (more complex trend surface)

3. quadrant or octant requirements: could specify that the neighbors used must include representatives in each of the principal directions.

To honor the data points, can have second pass to adjust grid values if necessary to insure contour lines go in the correct places.

appearance of map not guide to accuracy of method

different objectives may not be compatible:

  • to honor data points, need weighting to drop off rapidly (exact, more detail)
  • to predict, want to include as many points for estimates (smooth, generalized)
