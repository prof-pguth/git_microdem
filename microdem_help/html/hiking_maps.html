Maps for Hiking

+

This may not work with the latest USGS GeoPDF. 

  • Try to get the GeoPDF USGS Topos.  You can merge multiple quads, if you are going across several.
  • The historic maps, which go up to the present, in GeoTIFF might be a better choice.

Get the park GeoPDF (works for Maryland state parks, but not US National Park)

  • Open the PDF, and GDAL will convert to Geotiff
  • Convert the Geotiff to UTM if it is state plane
    • Data Manipulation, Resample, GDAL warp Geotiff to UTM/NAD83
Get the lidar point cloud, create a DSM.
This works best if there is a mixture of terrain types; in this case the route was entirely in the trees, and the DSM does not show much.
 

If the map does not have trails (USGS is phasing these in; there were on Maryland maps, but not Arizona or California, in spring 2018).

  • Get the Open Street Map for the area.  On the roads shapefile, filter OSM for TYPE='path' and pick a color and line width for display
  • HikingProject: free from REI.   App for cell phones, downoload GPX, and open as database on the base map

 

Creating the map

  • Open map in MICRODEM
  • File, Save map as image,   As Geotiff, screen scale (color)
  • Move Geotiff to the mobile
