Geocoding Web Service

In addition to being done locally for a database, geocoding can be done as a web service from a server.  The server might limit you to a specified number of requests per day, tracked by your IP address, and entice you to pay for more requests. This tracking can be fooled, and think you have exceeded your quota far earlier than it should.  It may only complain every few records.  For example http://www.gpsvisualizer.com/geocoder/ can use either the Yahoo or the Google service, but your browser will be the one making the request and running into the limit.

Google geocoding now requires an account and an API key.  You must have a credit card on file, but do get some free usage each month.
  • https://cloud.google.com/maps-platform/user-guide/
  • https://cloud.google.com/maps-platform/user-guide/account-changes/
  • Enter your key on the Web tab of the options form so it can be included in your request.

Google geocoding API uses requests like:

  • Old requests, before your API key had to be included in the request:
    • Address to lat/long: http://maps.googleapis.com/maps/api/geocode/xml?address=107+Claude+Street,+Annapolis,+MD&sensor=true
    • Lat/long to address: http://maps.googleapis.com/maps/api/geocode/xml?latlng=38.9933070,-76.4992460&sensor=true
  • Google documentation:
    • http://code.google.com/apis/maps/documentation/geocoding/#GeocodingRequests 
    • Geocoding API policies: https://developers.google.com/maps/documentation/geocoding/policies
  • If the geocoding fails, you will be informed, along with the address that could not be resolved.  You can look at the web results to determine if you want to continue.

Options

  • Right clicking on the map, Geocode, Address to Lat/Long.
  • Right clicking on the map ,Geocode, Lat/Long to address
  • On a database with ADDRESS field, Edit, Geocode addresses to fill in LAT/LONG fields for every record.  Address can be a full street address, or just the names of the CITY and STATE.  You might have to edit the database to get the field in the correct format.
