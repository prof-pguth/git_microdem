CSV Export

Accessed via

  • Cartography menu, Export computations options.  If you use this option, the approximate map limits will automatically be entered into the form.
  Set limits in decimal degrees for grid.  By default this will be the map extent on the active map when you picked the option.
  Lat High  
Long Low   Long High
  Lat Low  

You can put in a record number with each line if the checkbox is checked.

You can set a constant z value for all records

Set spacing in lat and long for the grid, in dlat and dlong, in decimal degrees.  By default this will be 1, which will not be appropriate for a large scale map.

Parameters:

  • Datum shift, geographic: distance in meters between the two points having same lat/long coordinates in the two datums.
  • Datum shift, UTM: distance in meters between the two points having same UTM coordinates in the two datums. 
  • Map distortion, h and k
  • Magnetic declination and intensity: you can plot this using the Vector data plot option.  You will need a multiple of about 0.002, because the intensity  of the magnetic field is so large.

You have the option to include a header line or not.

Set the file name for the output CSV file.

Write the file.  This will close the form, and open output as  Database overlay.  The file will be saved in a temporary directory, and you must save it elsewhere if you want to keep it.

Close the form without writing.
