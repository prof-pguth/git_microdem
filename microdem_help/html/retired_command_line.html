Retired/Disabled Command Line Options

 

This work was done early in the DEMIX process.  It has not been widely used (maybe not at all?) and has not been tested.

Option Act(ion) Command line, mandatory parameters Optional parameters (not case sensitive) Notes
DEMIX criteria evaluation   ?act=demix-criteria&filelist=c:\microdem\demix_criteria_files.txt  
  • filelist is the full path to an ASCII file, with one full path DBF control file name per test DEM
  • requires that the test and 1" reference DEMs be on the local machine, and indexed in the Map library
DEMIX resample by aggregation   ?act=DEMIX-CREATE-REF&filelist=c:\microdem\demix_resample_files.txt  
  • filelist is the full path to an ASCII file, with one full path DEM file name per test DEM.  This creates 1" pixel-is-area and pixel-is-point reference DEMS, and additional resolutions for high latitude COP and ALOS
  • output will be in the same directory, with standard names

 

Saving and used of GeoJSON was a long ago experiment, and has not been tested recently.  It is now commented out in the code, but could be restored by setting a conditional compiler direction

Option Act(ion) Command line, mandatory parameters Optional parameters (not case sensitive) Notes
DEM to GeoJSON-G dem2json ?act=dem2json&in=c:\mapdata\glacier.dem   Output in same directory, with new extension .JSON
LAS to GeoJSON-P las2json ?act=las2json&in=c:\mapdata\little.las   Output in same directory, with two new files with new extensions .JSON and .JSON.BZ2
7zip used for compression
