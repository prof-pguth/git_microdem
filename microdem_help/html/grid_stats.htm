Grid Statistics for Training Points

Select on Stats option from the table for the database with the training points.

  • Pick an open grid.
  • Program will create two graphics:
Histogram shows the min and max with the colored boxes, and either one standard deviation on either side, or quantiles (25% and 75%).  Pick which on the Analysis tab of the options form.  The caption to the graph will indicate which is plotted.
Table shows basic statistics for each class.
