Add Flag from Substring

Edit button on Table view, which provides popup from which you pick Flag, Add from substring.  Only enabled if you check the "EDIT" box on the right of the row of buttons.

Pick the name of a new field (will be a one character string), the field to search, and the substring to look for.  All records that match will be filled with a "Y".

This is useful because you cannot filter the database with an initial wild card.

The database at left was digitized in Google Earth, which allows only a single text field to transfer (NAME).  To get the ICON field populated with a minimum of effort, you could make flags for "GERMAN" and "INF_DIV" by searching for substrings "Ger" and "Inf" in the NAME field, and then filter for 'GERMAN='Y' AND INF_DIV='Y', and then fill all those records with the correct icon in one step.
