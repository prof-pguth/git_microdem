Live Flying

Select this icon on the main toolbar.  Then

  • Double click on the map to mark the starting location.
  • Drag out a triangle to define the first viewport.
  • Set the flight perspective view parameters.
  • Control the flight with this form, the cursor keys, or a joystick.
  • Flying initially comes with Auto NAP (nap of the earth).  This keeps you from crashing.  If you turn NAP off, you must monitor your elevation relative to the ground.
  • Speed:
  • Heading:
  • Altitude
  • Frame delay: increase this if you are moving too fast.
  • Pitch: tilt the view up or down.
  • Colors lets you adjust the reflectance or elevation colors used.
  • Controls adjusts sensitivity for the arrow keys and spin controls. (see window below)
  • Done ends the live flying
  • Pause stops the flying momentarily.
  • Slider lets you set the the viewing angle relative to the aircraft.  Negative angles run counterclockwise (to the left).  This lets you get the view experienced by a door gunner.
  • Center: recenters viewing, so you see the cockpit view ahead.

 

Speed: how much the speed (meters/frame) changes with each change of the cursor keys or the spin controls.

Heading: how many degrees the heading changes with each change of the cursor keys or the spin controls.

Altitude: how many meters the altitude changes with each change of the cursor keys or the spin controls.
