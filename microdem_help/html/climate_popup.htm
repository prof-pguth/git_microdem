Climate Options on Map Toolbar

Climate options button on map toolbar, (or an option on the Overlay menu) which opens a popup menu.  Most of these require a database, which will be downloaded automatically if it is not present.

  • Koppen grid data everywhere on earth
  • Koppen climate stations data only at stations, and some countries did not contribute data.  This is only enabled if you specifically request on Geology/Geography tab.
  • Global monthly temperatures at Koppen climate stations
  • Global monthly rain at Koppen climate stations
  • Global monthly wind vectors
  • Hurricanes: https://www.ncdc.noaa.gov/ibtracs/index.php?name=wmo-data new format coming fall 2018
  • US tornadoes: https://www.spc.noaa.gov/gis/svrgis/
  • CLIWOC ship logs
    • Weather
    • Ship motion
  • Recent world piracy: NGA ASAM database of recent piracy.  This shows current piracy world wide, starting in 1978.
  • Tide gauges and sea level rise
  • Sun above horizon
  • Night lights 2016
