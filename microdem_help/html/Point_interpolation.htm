Point Interpolation

t

Option, accessed on the "Calculate, Intervisibility" menu from a map, shows effectd of different interpolation algorithms.

  • The contour lines follow different rules, using only elevations at the grid points and linear interpolation between them.
  • Circular artefacts in the 1/R and 1/R² algorithms.
  • Extreme blockiness of the Nearest Grid and SW grid algorithms.
  • Only bilinear interpolation or triangle algorithms should be used.
  • Effects of the interpolation method will decrease as the data spacing decreases, and the four surrounding values become closer and closer in value, although if the elevations are in integer meters there will be stairsteps in certain grids where the values change.  The differences will have most impact along ridges and hilltops, because these features control intervisibility.

Some authors have advocated using more than four neighboring elevations; this is not implemented in MICRODEM and is not felt to be worth the additional effort.
