Identify Directory

File, Tools, Identify directory.  This will write a recognition file in the directory, so MICRODEM can identify the data and act intelligently.

  • UTM
  • SPCS
  • Mercator
  • South polar stereographic
  • UK-OS grid
  • XYZ shapefiles/LAS
  • Full world imagery: create a world file for images of the entire earth.  The upper left corner must be -180, 90 (North Pole, IDL), and the image must have a width twice its height. This is appropriate for images like Blue Marble or Night Lights.

For MICRODEM to recognize NLCD, you must do one of the following.  

  1. Insure that the words NLCD and the year (1991, 2001, 2006, or 2011) appear in the file name or the path.

Do not put NLCD canopy or impervious cover files in the same directories as the NLCD land cover.
