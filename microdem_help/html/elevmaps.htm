All maps of the Hanging Rock, Canyon, California, 1:24,000 DEM.

To change the type of map displayed you have several options:

  1. Right click on map and change display parameter.  Then select the type of display desired.

  2. Use "Modify, Display Parameter" menu choice.  Then select the type of display desired.

To change the characteristics of the map displayed you have several options:

  1. Right click on map, and select choice corresponding to the type of map currently displayed.

  2. Use "Modify" menu choice, and select the choice corresponding to the type of map currently displayed.
