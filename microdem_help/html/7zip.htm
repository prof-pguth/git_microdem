7-Zip

We use the freeware 7-Zip for compression and decompression.

  • Uncompression of Unix TAR and GZ files
  • Extract ESRI Layer Packages, which is probably the safest and easiest way to transfer data from ArcGIS.
  • Compress and uncompress BZIP2 files.
  • It will also deal with RAR files, if you have students who think that is a good format.

The MICRODEM full install will place the 64 bit versions of the 7z.exe and 7z.dll files in the MICRODEM program directory.  You can use the EXE file and its GUI to compress and decompress files.
