GEOCODING

Geocoding is going from an address to lat/long, or from a lat/long to a street address.

MICRODEM provides multiple ways to do geocoding:

  1. Geocoding can be done with the local TIGER database,  using the properties of  TIGER databases.
  2. In addition to being done locally for a database, geocoding can be done as a web service from a server.
