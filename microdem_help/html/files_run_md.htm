Files Required to Run MICRODEM

Setting up this program requires the following files to be placed in a program directory.  The installation program will accomplish this with two separate downloads.

  • MICRODEM.EXE, executable program.
  • MICRODEM.CHM, help file which you must correctly locate in the same directory as the program executable.

MICRODEM.INI with default settings.  You can delete this file and the program will recreate it.

Required EXE files, automatically installed:

  • 7z.exe and 7z.dll

Required downloads if you want to perform specific operations, which you must install on your own:

  • GDAL--it will be very hard to proceed without GDAL
  • Other key external programs

Support databases: should update automatically when they are changed and a new version of the program requires them.
