Area Geomorphometry Stats

There must be a DEM associated with the map on which the DB is plotted.

Stats button  popup menu for an area/polygon shapefile.

Set parameters that you want computed.  You have the same choices as used a regular array of blocks.

  • Advanced elevation option has the highest and lowest elevations, the locations where they occur, and the direction, distance, and slope in a staight line between them (think for landslides).  You can extract these into a new point shape file

Set masking options. 

  • Pick the buffer size.  You want some buffer if you have thin polygons.  If you are computing variables like slope which require that the nine point neighborhood be full, you probably want one at least a one pixel buffer if the polygons are relatively small.
  • Mask out points not in the polygon.

Point shapefile will be created, with the lat/long of the center of each polygon's bounding box, and the requested geomorphomertic parameters.  You can join with the polygon shapefile using the REC_ID record in each.
