LOS Spacing Analysis

To perform, right click on the LOS window and select Spacing Analysis.

The computer will attempt to use profile point spacings from 1- 60 meters.  This will be the difference between points along the radial used to determine intervisibility.

The top profile with the first algorithm will be drawn with the correct vertical scaling; the other profiles will be dropped down to clearly show the effects.

The computer will draw the LOS profiles in a single graph window, which can be rescaled as desired to see the individual points.  The largest point spacing will be on the top, and the smallest on the bottom.  Note that in the case below, the 60 m spacing misses the masked terrain at about 8.7 km along the profile.
