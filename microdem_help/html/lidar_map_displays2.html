Lidar Map Overlays

 

Look at the different display options for the data on the base map, which do not provide any 3D capability.  Insure you understand what each shows, and how that information was acquired.
  • Elevation.  If there are bird strikes or below ground returns, you may have to adjust the scaling range.
  • Return intensity.  You might have to adjust the color range, as the standards for how to code intensity have be interpreted inconsistently.
  • Classification
  • Scan angle--this can show the direction of aircraft motion.
  • Returns in pulse.  Multiple pulse returns highlight vegetation, or power lines.
