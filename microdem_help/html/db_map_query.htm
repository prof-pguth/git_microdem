DataBase Map Query

Accessed with Map Query button on database table display.

Geographically filters the data, and the restriction will remain in effect for all database operations until you remove it by changing the filter or selecting "All Recs" button.
  •   Rectangular region: Outline region on the map, and show all records in the box.
  • Irregular region: select a region using current digitizing options.  For area shape files this uses the centroid, and for line shape files the coordinates of the central point.  Puts a "MASK" field in the database.
  • Radius about point: select a point and a radius.  For area shape files this uses the centroid, and for line shape files the coordinates of the central point.  Puts a "MASK" field in the database.
  • Current map area: Restrict database table to just those records currently on screen.
  • Pick points for time/sequential series:
  • Near TIGER roads: will fill a USE column in the data base with "Y" or "N" depending on whether a point is within a specified buffer distance of a road in the TIGER data base.
  • Away from TIGER roads: will fill a USE column in the data base with "Y" or "N" depending on whether a point is at least a specified buffer distance away from a road in the TIGER data base.
  • Find address

After the query, if it does not automatically fill in a MASK field, you can manually set it.
