DEM Coloring Vector Map Projection

  • New Vector Map: Open a vector map.
  • Blow up or subset as desired. Some vector projections cannot be blown up.
  • If you have the DEM open, associate it with the vector map with the "Modify, Set DEM with map" menu choice.
  • Pick Overlay, DEM elevation colors
  • Select the DEM.
  • The map will be recolored.
