Measure Distance

MICRODEM provides a number of ways to compute distance:

  • Buttons on map toolbar
    • Measure distance--straight line segments.  Double click to select points.
    • Measure distance--curved trajectories picked with the mouse and stream mode digitizing.  Move to the first point, depress the left mouse button, and move to the end location and the move the mouse button up. 

Other ways to measure distance.

  • Keyboard entry on Computations form, from Analyze menu
  • Using a database, Distance from point option on from the Edit button on the database table form

Distance algorithms

  •  Vicenty formula: ellipsoidl earth, more accurate, and the only one recommended
  • Haversine formula assumes spherical earth, and has errors over very small distances
  • Cosine formula assumes spherical earth, and has errors over very long distances (opposite side of the earth)
