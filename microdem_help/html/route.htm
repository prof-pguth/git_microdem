Route Observation

 A DEM must be associated with the map.  This shows all terrain visible from a designated route, either collectively or at specific locations along the route.

This will probably work best on:

  • Reflectance map, with grayscale
  • Grayscale satellite imagery or aerial photograph
  • Scanned map, turned to grayscale (use Modify menu, Map shading options).
  • Pick on Calculate, Intervisibility menu from a map.
  • Pick Route observation
  • Pick the new route using point or stream for digitizing. After each point you can verify its location, if you have point verification set, and determine whether to continue the route.
  • Set options.
  • You set a distance apart for fans. 
  • Program will draw the points that can see or be seen from the route.
  • Program will open the database with the locations, which you can use to view/edit the results with a special toolbar.
  • Speed up ambush movie.

The data for the route is stored in a database, called AMBUSH.DBF. 

The viewsheds are stored, with a world file, a PRJ file for use with ArcMap, and a MET file with metadata, in the Viewshed Directory (set on Directories tab of options form).  By default this is the Windows Temp directory, which will be cleared out when you close the program.
