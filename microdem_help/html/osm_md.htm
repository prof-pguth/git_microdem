OpenStreetMap in MICRODEM

MICRODEM recognizes OSM files with

  • the string "openstreetmap" or "osm" in the file name or the path.  You might put the OSM files in c:\mapdata\openstreetmap.
  • Prep OSM/Natural Earth/GADM shapefile  should be done.

There are several ways to use this data:

  • Database to open one or more single files. 
    • The program will automatically use smart display to show the roads by category, and with an appropriate density based on the scale.
    • You can also pick the buildings
    • This will allow access to the DB table display options, and to ID records on the map.
  • Select a directory to get all the layers, on the Database, Load OSM overlay menu.
Database with rules for OSM road files. It will be in the main program directory.

Open it and edit it like any database. 
Database with rules for full OSM symbolization.  Roads use the rules above.

Open and edit it like any database.
