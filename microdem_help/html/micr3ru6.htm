Relief vs Region

On Geomorphometry Analysis form on the Analyze menu.  This can be used to pick region sizes for some of the operations.

Right click on map, Geomorphometry

Pick parameter.

Pick region to be analyzed.

Max range (m), or size of the analysis box.

Max point sampling factor: for the current map or Entire DEM, this will determine the number of points used.  Processing speed will go up as the square of this.
Inflection points on the graph show the spacing between ridges and valleys, perhaps at several scales.
