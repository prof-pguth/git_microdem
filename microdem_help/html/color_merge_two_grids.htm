Color Merge of Two Grids

Select from the menu or toolbar for the grid to be on the bottom, which in this case is Global topography near the Mid Atlantic Ridge.  This grid should probably be a grayscale Reflectance map.

Manage Overlays

  • Pick the second grid, in this case the rock types.  Sediment thickness is also a good choice,  The grid must already be open.
  • Adjust opacity. 
    • High values will let none of the lower grid show through.
    • Right click on map, and pick overlays
    • Used sliders on the 2d DEM/grid panel

OpenGL 3D view can show this map in 3D
