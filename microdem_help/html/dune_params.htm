Dune Detection Parameters

  • Wavelength/height database: many options previously available on this form can now be created from the database.

 
Parameter Usage Calculate Thread crests Crests in region  
Wavelength computation (m): Distance along the crest used to find the next point.  Setting this larger will run faster, but risks jumping crests if the dune curves.        
Region points: Point will be a peak if it is the highest in a region that extends this many points on either side of the central point.  If you use the default profile spacing of half the DEM spacing, this will be a total distance along the profile of about the Region Points times the DEM spacing.        
Min height (m): Peak/pit must have this minimum amplitude to be recognized.        
Number peaks: Must be at least this number of peaks along the profile to compute statistics.
