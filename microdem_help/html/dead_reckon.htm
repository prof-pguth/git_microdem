Dead Reckoning

Choice on Navigation submenu of "EDIT" button of database table form.

Requires fields LAT, LONG, KNOTS, HEADING, with time readings in  HOUR (e.g. from a log book).  You can filter the database to only operate on part of the record.  If there is a field DT_HOURS, it will contain the intervals for computation in hours; otherwise the program will assume one hour per reading.

Insure that the LAT and LONG fields have enough precision for the computations; you might want to make them 12.2500001 rather than 12.25.

 

  • Forward: Using the first position, will compute a position for every reading until the end of the file.
  • Backward: Using the last position, will compute a position for every reading until the beginning of the file.
