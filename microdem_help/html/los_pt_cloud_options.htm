LOS Point Cloud

  • Show slice: determines whether you can see a slice through the point cloud, with the points shown on the LOS View.  This can show  classification, return number, and differentiate air points.
  • Show floor/ceiling points: shows the points categorized as the floor and ceiling.
  • Show floor/ceiling lines: connect with a line the points categorized as the floor and ceiling.
  • Lateral  buffer (m): thickness, perpendicular to the profile line, of the slice.  Total thickness will be twice this value. This should be larger than the box size if you are interested in perpendicular profiles.
  • Box size (m): size of the search box, centered on the computation point.  It will go out half this size in each direction.
  • Floor (m): size of the search box, centered on the computation point.  It will go out half this size in each direction.  Because there will often be fewer ground returns, a larger value here than that used for Box size may be appropriate.
  • Color cloud masking:
  • Cloud density along LOS
  • Full cloud density
  • Load button: opens point cloud if not already open.
  • LAS button: change display options.
