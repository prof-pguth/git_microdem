Required Directory Structure

 

The program requires the following data directories:

  • x:\MapData\DEMS--DEMs written by the program will go to this location.
  • x:\MapData\SATS--satellite imagery or ancillary files written by the program will go to this location.
  • x::\MapData\DataBase --the ODBC alias will point to a dummy data base in this location; other data bases will be copied here. This is a scratch directory and you cannot store your data bases here. (perhaps not true; this aspect is rapidly evolving)
  • x::\MapData\MOVIES --movies will be written to this location.

You set the x drive designation in the Options form, Directories Tab.  

The drive on which you place them must be one on which you have write permission.  Picking this drive will create the required files if they do not already exist.

The MapData Directory must be at the root level of whatever drive you use.
