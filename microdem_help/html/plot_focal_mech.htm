Plot Single Focal Mechanism

 
  • Load the focal mechanisms database. 
  • Subset & zoom
  • Map query, current map area, so the search is fast
  • Plot a single focal mechanism by clicking the button on the map toolbar.
  • Verify options for how to plot the beach balls.
  • Double click on the earthquake location on the map. 
  • The focal mechanism will draw; it is not sticky and will disappear the next time the map redraws
