OSM Download

OpenStreetMap Download sites:

  • Best choice: http://download.bbbike.org/osm/ --get city already created, or define polygon
    • Insure that you change the format to shapefile
    • When you save and unzip, insure that "OSM" in the file or path name so MICRODEM will recognize the files and use smart display symbolization
    • Prep OSM/Natural Earth/GADM shapefile  should be done.
  • Other choices:
    • http://www.geofabrik.de/data/download.html --get country or state, but limited shapefiles
    • https://mapzen.com/data/metro-extracts/
      • Datasets split by geometry type: lines, points, or polygons
      • Datasets grouped into individual layers by OpenStreetMap tags: some addresses are present, but not usable in those tested
