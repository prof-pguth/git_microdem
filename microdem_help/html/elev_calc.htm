Elevation Calculations

On Calculate menu with the following choices:

  • Elevation extremes--brings up the terrain category form, and lets you highlight all elevations with a specified range, either by value or percentile.
  • Data voids--shows all the missing data

This insures that every point that matches the criteria will appear on the map with an exaggerated size.  This is particularly valuable if the points of interest are rare, isolated points which might otherwise with difficult or impossible to see, the effect of which will be magnified if you cannot display a 1:1 map view .
