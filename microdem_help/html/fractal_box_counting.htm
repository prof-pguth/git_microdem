Fractal Box Counting

From Geostatistical analysis option of Analyze option of Main Program Menu.

Graph.

Fractal dimension on the title bar.  It is 2 - slope of the best fit line, which has a negative slope so the fractal dimension is greater than 2.

Follows Clarke, 1986, with the following adjustments:

  • Adjust the areas for holes (such as SRTM).
  • Adjust for different areas at different resolutions--use as much of the DEM as possible at each resolution.
