Rhumb Lines

Invoked from "Calculate" menu of a map window

Enter coordinates by:
  • Start and End buttons, and type them in
  • Use "+" buttons to copy in from clipboard (enter by right clicking on the map, and adding the screen position to the clipboard, or by double clicking on a record in a point database.
  • You can use constant point spacing, or pick a specified number of points.
  • "Rhumb line" will create a shapefile with points, and plot it on the map.  If you want to save it, use Report button on database table form. 

A rhumb line is a line of constant bearing.  It will be a straight line on a Mercator projection.  It will not be the shortest distance, which will be a great circle, but which requires changing the heading as you travel.
