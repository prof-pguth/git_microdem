Lidar multi-dimensional displays

Open the grids with File, Open submenu

Create with All option on Other grids with the lidar point cloud.  Can add two other grids, which must start with:

  • CHAN

 

Base layer

  • NVS and DTM only appear if selected, since the DSM is generally a much better choice for the base DEM layer.
  • Classification only appear if all base options is checked, since the best use of the lidar classification is as a color layer overlay

Elevation coloring

Color layer overlay

  • Colorize in place uses the IHS color model from the color layer and the intensity from the base layer
  • Opacity overlays the color layer on the base layer, with the desired opacity.

Colorize in place

Terrain shadows: uses IHS color mode from the DSM on a color image like RGB, NIR, or the lidar classification.
