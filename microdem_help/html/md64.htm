64 bit MICRODEM

 The 64 bit version of the program is now the default installation, and we will  update the 32 bit program only in exceptional cases like teaching students with no access to modern Windows machines. 

In addition to those changes required for 64 bit, a number of changes were also made to get the program to run on Android, Mac OS and iOS.  We have noticed significant speed up using the 64 bit version of the program, and also with Windows 8.1 and 10. 

As a 32 bit program, the most memory the program could address was 2 GB. 

MICRODEM Version BDE tDBF SQLITE Changes required
32 bit (will run on 64 bit Windows) Yes--requires BDE installed Yes Yes  
64 bit No Yes Yes New database engine
Android, iOS, Mac OS No ? Yes New database, case sensitive file names, \ to / in file names, no BMP files

You should be aware of the following limitations:

  • For the SQLlite versions,  if you try to load a DBF file (including a shapefile), the program will automatically convert it to a .db file.  This could be slow.
  • Automatic saving of databases might not be perfect, and you might lose edits to the databases.  You can manually save if needed.
  • Syncing of shapefiles is not happening, so any edits to the database will not be reflected in the shapefile.
  • If you want to share your database (including shapefile) with another program, you must save the .db file to a .dbf.

If you have problems or suggestions, do not hesitate to contact us.
