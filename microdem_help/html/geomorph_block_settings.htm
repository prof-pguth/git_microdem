Geomorph Block Settings

There are several ways to select the points:

  • Use a "Points location" database.  The computations will be done at each point in the database.  You might or might not get complete coverage, and you  might be overlapping regions.  If you select this, the region could be defined either by the number of grid postings, or the size in meters.  The box will be the size selected, centered on each point.
  • Boxes in either grid postings or meters.  A box will be defined in one corner, and the next boxes will be adjacent.
