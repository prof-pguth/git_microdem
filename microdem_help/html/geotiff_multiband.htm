GEOTIFF Multiband, Multifile Satellite Imagery

Some satellite imagery in GEOTIFF format, with each band in a separate file

  • Landsat data from  USGS
  • IKONOS 1 and 4 m imagery 
    1. Open a single band.  It must have 'BLU' or 'GRN' or 'RED' or 'NIR' in characters 10-12 of the file name.
    2. Confirm that you want to open the 4 band, 4 m IKONOS data.
    3. Use like any other image.
    4. This uses the file naming scheme on the data sets I have seen.  If this is not standard, you will be prompted to enter the data set names.
  • QuickBird 0.61 and 1 m imagery
