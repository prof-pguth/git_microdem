Convert PLSS to Lat/Long

On the Calculate: menu.

You must have a valid PLSS data base defined.

 

Build up a PLSS location string using the controls.

When you have the section, township, and range entered, the program will compute the lat/long at the center of the element.

You can optionally add the 1/4 section, and the 1/4-1/4 section.
