Color Based on DB Field

From Color popup menu when  "EDIT" button is pressed on the database table form.  You must have already put in a COLOR field for this option to appear.

Pick the field to use.  You should insure that there are a reasonable number of unique entries in this field, as you will have to pick a color for each.

The program will then cycle through all the categories in the field you selected.  The status bar for the main program window will show:
  • Progress (in the screen shot, it is on category 1 of 21)
  • Name of the category (you can also get the name form the database table, which will be filtered)
  • The existing color, if any, will be the default choice.

The status bar for the DB will show the number of records in the category.

If one of the fields in the DB is NAME (you can rename a field to this), it will have the color shown.
