GPX Files

Options:

  • Database (shapefile).   This creates a single DBF with no extraneous fields.
  • Other options, not recommended
    • Data Manipulation,  Resample, GDAL/OGR, OGR GPX to shapefile.  This creates a number of files in a new directory, including a line with the track, and a points DB with the recorded points.   You might also want to:
      • Edit, Map coordinates, Add lat/long from shp file.
      • Hide all the empty fields produced in the GDAL import, with Report, Hide unused fields.
      • Edit, Delete unused fields
    • Import with DNR GPS and convert to shapefile
