Print Preview & Print to Scale

The print button on the map toolbar offers "print preview & print to scale" and "Prepare printer image" options. Both can eventually bring up this form, which shows a reduced view of the current map, with the locations of page breaks indicated.

  • You can adjust the scale to get the map onto a specified number of pages.
  • If you want to get the map onto a specified number of pages without changing the scale, you must subset the map area.
  • The preview map shown may have the grid with many missing lines due to reduction to fit the small window. 

You will get the best quality on the final printout with the largest size image possible.  Windows will limit the size of the map window you can open, but it is at least 2000 x 2000 pixels and is likely much larger (we have not tested this option in a long time), and a function of your hardware and the version of Windows you are running.  Using a large map size is most critical when you print to a very large printer.
