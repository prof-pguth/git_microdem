Get reasonable number of points, by either
  • Open database, and Edit, Thin database

  • Max/mean/min grids from point data  Thin the soundings which are probably much denser than the 30 m Landsat spacing,  and to get a grid to view them.  You must see if the Max, Min, and Mean grids are different (perhaps by doing a difference operation with Multiple grid arithmetic  Difference between two grids) and then decide which to use.  You can then File, Save DEM, Save DBF table.

New image to open satellite image.

  • Database Open a DB with depths (perhaps thinned), and add the band reflectances with Edit Satellite, add DN.  If one of the bands (e.g. ETM pan) has different spacing, the values for it will not be correct. You should remove that band.
  • New DEM:  Open the Pan band as a DEM, and then on the database toolbar,  Edit DEM, Add nearest elevation from DEM. You can then rename the field to the correct BAND.
