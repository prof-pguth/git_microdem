Keyboard Creation of a Point Database

Open a map.

Select map annotation.

Select Point symbols, and pick a DBase file for your entries.

Insure that "Reasonable" or "All" graphical point verification is turned on (Coordinates tab of Options form).

If you want to enter labels, insure the check box is checked.

  • Double click to selection points.  You need not be precise.
  • Verify (or completely re-enter) the coordinates you want.
  • Select a label and point symbol.
  • Select OK.  The point will be plotted on the map.

Repeat for additional points.

The "Calculate", "Where is" function is now used only for quick location of a point and temporarily marking it on the map.
