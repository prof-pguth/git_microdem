Relief Slope Inflection

Select from Calculate menu, Topographic grain submenu with the Fabric at point option.

Relief calculated in successively larger regions about selected point.

Characteristic ridge spacing from the change in slope, in this case about 2 km.

 

Uses

  • Statistical sample interval, for the number of grid points to use in increasing the spacing.  Set on Analysis tab of  Options  form .
  • Max region size:   how far to go out.  Set on inflection tab of Topographic fabric vector overlay form.

Reference:

  • Goldberg, G.M., 1962, The derivation of quantitative surface data from gross surveys: Surveying and Mapping, vol.22, p.537-548.
