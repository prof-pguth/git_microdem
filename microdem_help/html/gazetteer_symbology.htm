Gazetteer Symbology

File "GAZ_SYMBOLS_Vnnn.DBF" in program root directory.  This contains categories to use, and symbols.

Gazetteer labeling uses definitions database as displayed above.

Fields used:

  • NAME--Each category has a name.
  • GAZ--whether the category applies to USGS, NGA or other gazetteers.
  • USE--Y/N, to determine if the category will be plotted.  This is set on gazetteer options form.
  • Radius (m) used for buffering. Use gazetteer to mask a DEM
  • Filter--filter used find the members of the category in the database.
  • SYM_COLOR, SYM_SIZE, and SYM_TYPE--determine the point symbol. You can edit the database and graphically modify these by right clicking on a record.
  • ICON
  • FONT_NAME, FONT_SIZE, FONT_COLOR

To change symbology and font:

  •  Open Database
  • Check edit checkbox.
  • Double click on the record, and pick "Edit point symbol" or "Edit font"
  • Select the color, size, and symbol type.
  • Changes should take effect immediately.

To add another category: 

  •  Open Database
  • Check the edit checkbox.
  • Double click on the record, and pick "Insert record"
  • Fill in the following fields:
    • Name--can be anything; it is what will be displayed
    • GAZ--must be "USGS" or "NGA" or another such as "CCOG", which will be defined in this file.
    • FILTER--fill in a filter that identifies the type of data from the gazetteer.
    • SYMBOL and FONT options can be most easily added later with change symbology option.
    • Use--"Y" to display, "N" to skip
  • Select Post edits
