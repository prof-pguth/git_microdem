Layer Opacity

You can set layer opacity for many layers, which can allow seeing several things on the same map.  A grayscale reflectance map may be the best choice for this, and will show how the other layers relate to topography.  For imagery you can grayscale or subdue the base map.

You might set opacity with:

  • Slider bar: 100% on the right of the slider bar will not let anything show through, and 0% on the left will show nothing from the database.
  • Entry of the value in an edit box.

 

  • Variable opacity merges
  • Alpha Blending DEM generated Perspective and Oblique Photograph
