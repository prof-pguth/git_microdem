All Band Histograms

Option invoked:
  • Image analysis menu
  • Supervised classification

If opened on the supervised classification, you have the option to show or not all the points not in a training set.  This can be slow; the speed is set by the maximum number of points to consider on the Imagery tab of the options form.  A sampling factor is picked that uses no more points than set here.

This shows all the band histograms in a matrix.  This allows easy visualization of which band pairs discriminate your training areas.  You can compare this with a band correlation matrix.

You can show the training sets, with the points and selected window in which points must lie; this option is set on the  Supervised classification form.

The graphs on the principal diagonal how well single bands discriminate the training areas.
