Copy Records Between Databases

  • Double click on a record in the data base
  • Pick "Copy point coordinates to clipboard"
  • Insure Edit is enabled in the second data base
  • Pick Edit button
  • Pick "Insert record from clipboard"
  • This will copy  NAME field, and for points the Lat/Long fields, and for lines/areas the coordinates and the bounding box .
  • Edit any other fields in the
  • Pick "Post"
