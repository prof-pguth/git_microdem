TIGER Geocoding in MICRODEM

Access from the Map query button's popup menu on the database table display.
 
  Address geocoding form appears if the data base contains TIGER data, in ESRI shapefile format.

Type in the address, and hit FIND.  Most addresses do not use FEDIRP (prefix direction) and FEDIRS (suffix direction), and you should leave them blank.

Number FEDIRP  FENAME FETYPE FEDIRS
1 N Main St NE

You will get the lat/long of the address and the ZIP code, and it will be highlighted on the map.  The street segment will also be highlighted.

How TIGER address geocoding works.
