QGIS

QGIS can read (and then convert to common formats) ESRI Geodatabases (.gdb).

Open QGIS and drag the GDB file (probably actually a folder) into the Layers pane of QGIS.  Then select the layers you want.

 

Get QGIS with OSGeo4W

 If you want a more recent QGIS than comes with OSGeo4W, go to https://qgis.org/en/site/forusers/download.html#
