Report, Legend

Option for a database Report, Legend.  A number of options create a database with the information for a legend.  This is particularly useful for landcover.

 

Table display. 

  • You can do a screen capture to use this directly
  • You can Filter
  • You can adjust the fonts with Report, Grid fonts
  • You can reorder fields by drag and drop
  Create a bitmap legend with the Report, Legend option. 
Color bar chart


Export the legend database with Report, Export HTML, Single table.  You can take the table from the web page, paste it into Excel, and do computations.

 


Columns A-E were created with a count of the number of pixels in 4 categories.

Count has the number of pixels in each category.

This is from a Sentinel-2 scene, where each pixel is 10 m on a side, with an area of 100 square meters

Column F is added in Excel to get the area in square meters.

Column G converts the square meters to square kilometers, which makes better sense.
