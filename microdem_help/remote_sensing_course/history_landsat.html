Landsat Directions

  • Download data: Earth Explorer directions and EarthExplorer download.  Search for a small area, so that all the scenes cover the area you want.  Some of the early Landsats had different orbits and the scenes do not cover the same area as the later missions.  Insure that all the dates show the area you want.
  • Manage Landsat data
  • Subset & zoom to an interesting part of the scene, and at a scale that shows the detail captured (100%).  You need only do this on a single satellite scene.
  • Legends/marginalia  Pick the map name option, which will put the satellite and date on the maps
  • On the same scene, make movie with all open maps.

Directions for a group

  • To assemble you animation on multiple computers
    • Make sure all computers have the same map area
      • Options, Maps, Default size.  These might be the same, unless you have changed them
    • Share a subset area,
      • Annotate Map, Create shapefile, Oultine map area.  This creates a shapefile with the outline of your current map.
      • Database table controls, Report, ZIP shapefile
      • Send the small zip file (with SHZ extension) to team mates
    • Save the maps as a PNG
    • Move all the PNGs to the same computer
    • File, Tools , Movie replay, Create new and pick all the PNGS
  • Create animation on a single computer
    • Have all the maps open, subset, and the same size
    • File, Graphics with all maps
      • , Movie with all maps
