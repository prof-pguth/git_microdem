Landsat File Names

LXSS_LLLL_PPPRRR_YYYYMMDD_yyyymmdd_CC_TX

LC08_L1TP_015033_20220108_20220108_02_RT

LM01_L1TP_015033_19770526_20200907_02_T2

Name elements

  • Key groups
    • First group
      • L for landsat
      • Sensor: “C”=OLI/TIRS combined, “O”=OLI-only, “T”=TIRS-only, “E”=ETM+, “T”=“TM, “M”=MSS
      • Two digits for landsat number
    • Fourth group, date image acquired
  • Secondary group
    • Third group, path/row which shows where the image was located.  In particular you can quickly see if scenes cover the same area
  • Other groups
  • Second group, Landset processing correction levels
    • Level 1, L1TP/L1GT/L1GS 
      • Precision and Terrain Correction (L1TP)--attempted for all scences
      • Systematic Terrain Correction (L1GT)--if L1TP fails
      • Systematic Correction (L1GS)--if L1GT fails
      • Level-1 data are delivered as Digital Numbers (DN) in an unsigned 16-bit integer format and can be converted to Top of Atmosphere (TOA) reflectance or radiance using the radiometric scaling factors provided in each scene metadata file. 
      • https://www.usgs.gov/core-science-systems/nli/landsat/landsat-collection-2-level-1-data?qt-science_support_page_related_con=1#qt-science_support_page_related_con
    • Level 2
      • surface reflectance and surface temperature.
      •  https://www.usgs.gov/core-science-systems/nli/landsat/landsat-collection-2-level-2-science-products
    • Two characters:
      • SP, science pagkage, both surface reflectance (SR) and surface termperature (ST)
  • Fifth group--date image processed or reprocessed
  • Sixth group, collection number
    • 1: older format
    • 2: reprocesing, will be the only option starting in 2022
      • https://prd-wret.s3.us-west-2.amazonaws.com/assets/palladium/production/atoms/files/Landsat-C1vsC2-2021-0430-LMWS.pdf
      • No more Collection 1 processing after Dec 2021, and no more availability after Dec 2022
      • Should not mix Collection 1 and Collection 2 in same workflow
      • The Landsat Collection 2 U.S. Analysis Ready Data (ARD) product contains Top of Atmosphere (TOA) reflectance, TOA Brightness Temperature (BT), Surface Reflectance (SR), Surface Temperature (ST), and Quality Assessment (QA) data. It uses 150 km tiles, with only the points collected that day.  Albers projection
  • Seventh group, Landsat Tiers or collection category--designated by the last two digits in the name
    • RT--newly acquired
    • T1--highest quality data after processing and analysis
    • T2--lower quality data after processing and analysis

 

MICRODEM relies on the file naming convention from USGS.  It may still deal an older variant, but not the  LANDSAT 8 SURFACE REFLECTANCE CODE (LASRC) PRODUCT.
