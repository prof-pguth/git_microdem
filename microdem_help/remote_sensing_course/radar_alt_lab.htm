 Jet Propulsion Laboratory, 2002, PO.DAAC merged geophysical data record from the TOPEX/Poseidon Mission Generation B Cycles 355, 356, 357: CD-ROM, POMGB_119_1. Converted by Prof Guth to a DBF file

This contains the altimeter data to compute the geoid anomaly.

The altimeter has a 10 day repeat cycle.

Point spacing along track is 6.2 km

The ground sampling distance is a few km

Height good to about 5 cm

 

 Database (which could also overlay the data on another base map; the file will be in c:\mapdata\etopo1 after you have downloaded it with File, Geology labs).  The most important fields in this database:
  • ORBIT: These are time sequential during the cycle, and each jumps in location with the gaps filled in on later days.
  • GEOID_ANOM: the height of the geoid above (+) or below(-) the ellipsoid.
  • ELEV_M: height in meters
  • DYN_TOPO: dynamic topography
  • SWH_K: significant wave height with the K band altimeter; in meters.
  • SWH_C: significant wave height with the C band altimeter; in meters.
  • WIND_SP: wind speed (m/s).
  • ASC_EQ_X: sequential ascending equatorial crossing; there are in east-west order and allow rapid cycling through the data.  These run from SW to NE.
  • DES_EQ_X: sequential descending equatorial crossing; there are in east-west order and allow rapid cycling through the data.  These run from SE to NW.
  • DIS_KM: distance along the pass in kilometers, which allows a better graph with uniform scaling compared to latitude or longitude.
