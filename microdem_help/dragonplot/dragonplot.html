DragonPlot

DragonPlot was conceived by the late Tom Craven of Sisters, Oregon as a tool to help lookouts in fire towers.  It added a digitial telescope to the Osborne Firefinder, and allows a single observation of a fire to locate the position using a DEM.  It was originally a stand-alone program, but eventually became a module in MICRODEM.  It was marketed by a company called Geodesy Base, primarily to USFS in Oregon.

Information on obtaining the data and setting up systems is no longer readiliy available; that was the responsiblity of Tom Craven, I don't have any electronic help files. Help might be available from the lookouts in Oregon still using the program;  they were given the laptops with the program when Geodesy Base dissolved.

As part of MICRODEM, DragonPlot is now open source.  The last code changes to the DragonPlot module's came in 2022, but it has been recompiled with all subsequent changes to MICRODEM.

Dragon Plot runs in two modes:

  • Start solo program, which looks like the old DragonPlot, and which will persist until you change the program mode:
    • If you rename the program EXE to include the phrases "DP" or "DragonPlot" it will start in this mode
    • If you pass the program the command line parameter "-DP"
    • If you set the program mode to DragonPlot on the Options form, program tab, MICRODEM
  • Option within MICRODEM
    • Start with the option on the file menu.  This will currently close all windows, but bring back MICRODEM when you close the program.

To set the defaults for DragonPlot

  • Options tab on DragonPlot control form, then other options, then reset DragonPlot Options.
