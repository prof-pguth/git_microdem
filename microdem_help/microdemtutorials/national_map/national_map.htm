USGS National Map

  1. Open by:
    • Point browser to https://apps.nationalmap.gov/downloader/
  To download data
  • Zoom to your area of interest;
  • Easiest to just download the entire screen.
  • You can upload a zipped shape file (insure you get all the files) that will zoom the map to your area of interest.
 
 
  • Select available data products.  The most useful will be:
    • Elevation Products (3DEP): this is 3DEP/NED in 1", 1/3" available nationwide;  and 1/9", and the 1 m DEM available in some locations.  Different spacing in Alaska.
    • Elevation Source Data (3DEP): LAS files.  These are large
    • Names--Geographic Names Information System (GNIS, or gazetteer)
    • US Topo
    • Historical Topographic Maps
  • Generally less useful
    • Hydrography NHD: you probably want the shapefiles for Dynamic Extract and Dynamic extract with flow table.
  • Select format.  If possible you want Geotiff or Shapefile as the best formats for use in MICRODEM.  In particular avoid the Geodatabase, which is a proprietary ESRI format for Arc.  Some other formats will require that that you have GDAL
 

Elevation Products (3DEP): For the elevation data there are several formats, all of which work with varying challenges.  All will be zipped initially.

  • Geotiff
  • IMG select the IMG file and GDAL converts to a Geotiff which will open.
 

Elevation Source Data (3DEP): LAS files. 

  • Pick lidar point cloud under data, and select the region
  • Download options
  High resolution imagery. 
  • Imagery--1 foot (HRO) .This imagery may have the choice of multiple dates, some of which will include 4 bands with IR for vegetation analysis.
  • Imagery--1 meter (NAIP).  Newer NAIP might have higher spatial resolution.
 

Zoom map to show your area.

Pick the option for "Current extent" on the top of the mpa.

 

You can upload a KML file or shapefile.  For the KML option, you get a hug buffer around the area (3-5 times the number of tiles actually in the area).  If you then manually draw a polygon around the KML file, you get the right number.

Download options

Options here have been changing significantly

Should be able to

  • Download each tile individually. Download link LAZ in blue on the right lets you choose the directory to download a single file. Easiest choice if you want just a few file.
  • Batch download
    • From the results of the search or the cart, export URLs to text (TXT icon)
    • Use Batch Download on Data Manipulation form if you get the TEXT file.
  • You must pick the data you want and add to your cart (you use +All instead of checking each individually). Then view cart.  You can download all files together.

Recently (but not now) the uget option was the only one that worked; it is available from your cart where the individual download option is not present.  Use uGet only if you need a lot of files.
