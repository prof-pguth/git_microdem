Processing steps for LVIS Level 2 data

  • Get the Level 2 ASCII text files at https://lvis.gsfc.nasa.gov/Gabon_2016.html
  • In a text editor
    • Remove the first twelve lines with comments
    • Remove the "#" character from the 13th line
    • Rename the incident angle field name (it is too long for dbASE).
  • Import the text file in MICRODEM It will be converted to a DBF file, which you should use going forward.

References 

  • Metadata: https://lvis.gsfc.nasa.gov/data_sets/2016_gabon/LVIS_GabonAfriSAR_data_release.pdf
  • Tutorial to use data in QGIS: https://lvis.gsfc.nasa.gov/workshop/LVIS_QGIS_Tutorial_Gabon.pdf
