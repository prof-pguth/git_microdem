Lidar Tutorial in MICRODEM

File, Introductory tutorials, Annapolis lidar.  This will download a very large dataset (~ 8 GB download, ~25GB when uncompressed).

This will download data if it is not present  (you should review this page to see what data is present and something of its characteristics)

Lidar puts a laser on an aircraft, and shoots a laser pulse thousands a times a second.  It records the TWTT (two way travel time) and the intensity of the returned energy.  Knowing the location of the sensor from GPS and an INU and the orientation of the energy pulse, it can compute a 3D location.  The laser is typically NIR, but if the survey wants water penetration, it will be green light, which can cause interest or concern when people notice the beams.  When there are trees or power lines, there can be multiple returns from a single pulse because the energy spreads out.

For many areas along the US Coast, there are lidar surveys almost every year, which can monitor coastal erosion and hurricane damage (for example, see Duck, NC).

Open  point cloud data. 

  • Open Grid/DEM  to use as a base map.
  • LIDAR Point clouds.  This can be done either by selecting the folder, or picking the individual files.
  • If you have problems with the "large" data sets,
    •   Subset & zoom to just the area you want to look at.
    • Load only a few of the tiles.  You will have to switch to opening by files rather than by directory to do this, and then will have to figure out the tiles you need.  The map subset and zoom option handles the details of getting the correct file

Use the control window for the point cloud operations.

  • Look at the metadata, on files list tab.  for just one file.  Some of this is in the LAS header, and some is computed.  There may also be Metadata in the directory with the files; this does not have a standard format or file naming convention, so it is not easy for the program to process it, but you can use file explorer.  The metadata may also be online (this data's metadata locations)
  • Look at the different display options for the lidar overlay on the base map.  Insure you understand what each shows, and how that information was acquired.
    • On the intenity maps, what reflects the most energy (do not be confused by the missing data which is white in all the displays)
    • Which display shows your the direction of the flight lines?
    • Where do you see multiple returns?

 

  • Look at Lidar 3D displays  This will work best with a small area, so Subset & zoom before picking

Advanced options (don't do):

  • Look at Histograms/Counts.  (this can be slow)
