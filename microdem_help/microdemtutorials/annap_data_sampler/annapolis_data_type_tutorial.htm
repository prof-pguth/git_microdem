Data Types Tutorial in MICRODEM

File, Introductory tutorials, Annapolis GIS data sampler

 

 This will download data if it is not present  (you should review this page to see what data is present and something of its characteristics)

 

Raster and Vector Data.

  • Open Grid/DEM and open the 3DEP/NED ninth second. 
  • Database (shapefile) and open the TIGER edges file.  There is a database table display, which you can toggle on and off with the checkbox on the top left of the toolbar.
  • The "best" display for the DEM is a shaded reflectance or hillshade.  It can have elevation colors (called IHS, for the color model used), or be grayscale.  To change, right click on the  map, and pick the option desired (there are some more esoteric ones as well).
  • Which of these is raster data, and which is vector?
    • Subset & zoom to be sure you understand what happens to each type of data as you change the scale.

Vector Data Types.

  • Open Grid/DEM and open the 3DEP/NED ninth second
  • Database (shapefile) and open the TIGER edges file.  There is a database table display, which you can toggle on and off with the checkbox on the top left of the toolbar.
  • From the DB table, Report, Database structure
  • What types of data are in this database?  Besides the type, what other key variables are there for each field?
  • Note that the field names cannot be very long, and can only use capital letters, digits, and the underscore.  This is a relict from an ancient PC database, but which was adapted for the shapefile standard and which remains the de facto interchange format even as database technology has improved.

 Vector Data Geometries.

  • Open Grid/DEM and open the 3DEP/NED ninth second
  • Database: Load OSM overlay: pick the directory with the OSM data (there are two years to choose from).
    • This does not load the DB tables, which would allow addtional options if you instead opened the eight files (or just those you wanted) with Database
  • To highlight just the vector data, you could:
    • Hide the DEM with the "blank" option on Display parameter
    • Change the Reflectance map to use grayscale instead of color, since your mind has troubles using color for too many things at once
  • To better see the vector data, you could:
    • Set percentage data shown for raster data
    • Subset & zoom
  • Can you see examples of point, line, and area features?  Looking back to the TIGER data, what geometry was it?

Raster Data Resolution.

  • Open Grid/DEM and open the three 3DEP/NED DEMs, which are all nearly the same file size (10-15 MB).
  • Which has the most detail?
  • Which covers the largest area?
  • What do you have to do to see the pixels?
  • These all have data spacing in arc seconds.  What does this do to the spacing in meters?

Raster Data Types

  • Open Grid/DEM and open the one second 3DEP/NED DEM
    • Edit, Edit DEM grid, and then outline a box.  What is the type for the data in the grid?
  • Open Landcover map and open the 2011 NLCD  (you want the TIF file).  This uses Landsat satellite imagery to classify the ground into about 25 categories. 
    • Roam over the map and note that it tells you the name for feature class at that location.  The grid actually holds an integer, which the program converts to a string using a lookup table.
    • Get a legend by right clicking and Popup legend.
  • Grids cannot hold strings/characters
